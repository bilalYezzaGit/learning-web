## Structure des livres

### Manuel scolaire (CNP)
Chaque chapitre contient 6 rubriques :
1. **Pour commencer** - Consolidation des acquis anterieurs
2. **Cours** - Activites + resultats a retenir
3. **QCM - Vrai ou faux** - Auto-evaluation
4. **Mobiliser ses competences** - Problemes integratifs
5. **Exercices et problemes** - Exercices autonomes + activites avec ordinateur
6. **Math-culture** - Histoire des mathematiques

### Parascolaire (Kounouz)
Chaque chapitre contient 3 sections :
1. **Resume de cours** - Resultats essentiels + reflexes
2. **Enonces** - Exercices par difficulte croissante
3. **Solutions detaillees** - Corriges integraux

### Corrige du manuel (Chams/CMS)
- Corrections des exercices du manuel scolaire (QCM, Vrai/Faux, Mobiliser ses competences, Exercices)

### XY Plus (CAEU Med Ali)
Chaque chapitre contient 3 sections :
1. **Resume de cours** - Rappel des resultats essentiels
2. **Enonces** - Exercices et problemes corriges, par difficulte croissante
3. **Corriges** - Solutions detaillees

**Note** : Le T2 est un PDF image (scanne), pas de texte extractible. Lecture visuelle uniquement.
Le T1 contient du texte extractible.

## Workflow d'extraction

Les pages sont extraites **a la demande** depuis les PDF sources (pas de pre-extraction).
Le dossier `_raw/pages/` est temporaire et ne doit pas etre commite.

### Principe

1. L'utilisateur demande de travailler sur un module (ex: "on attaque la trigonometrie")
2. Claude consulte la cartographie ci-dessus pour identifier les plages de pages
3. Claude extrait les pages en PNG depuis les PDF sources (lecture visuelle, pas textuelle)
4. Claude lit visuellement les images pour transcrire le contenu (formules, figures, tableaux)
5. Les pages extraites sont supprimees apres usage

### Pourquoi une lecture visuelle ?

Le contenu mathematique contient des figures, tableaux de variations, tableaux de signes,
courbes, et des notations spatiales qu'un simple `pdftotext` ne capture pas.
Claude lit les images PNG comme un humain lirait la page imprimee.

### Commandes d'extraction

**Extraire des pages en PNG (300 DPI)** :
```bash
# Pages 21 a 39 du manuel T1 (module Continuite)
pdftoppm -png -r 300 -f 21 -l 39 \
  _raw/Manuel_scolaire_3_math_section_math_t1.pdf \
  _raw/pages/manuel_t1

# Pages 16 a 26 du corrige T1
pdftoppm -png -r 300 -f 16 -l 26 \
  _raw/Corrige_manuel_scolaire_3_math_section_math_t1.pdf \
  _raw/pages/corrige_t1

# Pages 26 a 41 du parascolaire Analyse
pdftoppm -png -r 300 -f 26 -l 41 \
  _raw/Parascolaire_Analyse_3eme_sec_Section_Math_ocr.pdf \
  _raw/pages/parascolaire_analyse
```

**Extraire un sous-PDF** (pour archivage leger) :
```bash
qpdf _raw/Manuel_scolaire_3_math_section_math_t1.pdf \
  --pages . 21-39 -- _raw/pages/manuel_t1_continuite.pdf
```

**Nettoyage** :
```bash
rm -rf _raw/pages/
```

### Gitignore

Le dossier `_raw/pages/` doit etre dans `.gitignore` :
