# Spacing, Layout & Responsive

> Layout structure, spacing tokens, viewport rules, responsive patterns.

- LAY-001 [auto] No 100vh usage — use 100svh for mobile browser compatibility
- LAY-002 [auto] No bare container class — use mx-auto w-full max-w-7xl (Tailwind v4 container doesn't auto-center)
- LAY-003 [auto] Parcours layout: SidebarProvider, AppSidebar, SidebarInset with overflow, id="main-content", @container/main, MobileBottomNav
- LAY-004 [auto] Radius scale: --radius defined in globals.css
- LAY-005 [auto] Header height token: --header-height defined in globals.css
