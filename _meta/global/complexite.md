# Echelle de complexite — Reference globale

Echelle commune a tous les modules pour mesurer la difficulte des exercices et atomes.

---

## Niveaux de difficulte (0 a 3)

### Niveau 0 — Decouverte guidee
- **Description** : Activite introductive, l'eleve est accompagne pas a pas
- **Autonomie** : Tres faible — les etapes sont indiquees
- **Techniques** : Une seule, elementaire
- **Duree type** : 2-5 minutes
- **Exemple** : Calculer f(2) pour f(x) = x^2 + 1

### Niveau 1 — Application directe
- **Description** : Application d'une seule formule ou technique, sans adaptation
- **Autonomie** : L'eleve doit identifier la formule mais le schema est standard
- **Techniques** : Une seule, appliquee directement
- **Duree type** : 3-10 minutes
- **Exemple** : Calculer f'(a) par la definition pour f(x) = x^2

### Niveau 2 — Combinaison de techniques
- **Description** : Necessite de combiner 2+ techniques ou d'adapter une methode a un contexte nouveau
- **Autonomie** : L'eleve doit choisir les outils et planifier les etapes
- **Techniques** : Deux ou plus, enchainées
- **Duree type** : 8-20 minutes
- **Exemple** : Etudier la derivabilite d'une fonction par morceaux en un point de raccordement

### Niveau 3 — Probleme de synthese / approfondissement
- **Description** : Probleme ouvert, modelisation, ou demonstration non triviale
- **Autonomie** : Elevee — l'eleve doit structurer sa demarche
- **Techniques** : Multiples, avec choix strategique
- **Duree type** : 15-30 minutes
- **Exemple** : Probleme d'optimisation geometrique (modeliser + deriver + conclure)

---

## Criteres transversaux

| Critere | Niveau 0-1 | Niveau 2 | Niveau 3 |
|---------|-----------|----------|----------|
| Nombre de techniques | 1 | 2-3 | 3+ |
| Transfert de contexte | Non | Partiel | Oui |
| Modelisation requise | Non | Non | Oui |
| Justification | Calcul | Argument | Demonstration |
| Parametres | Numeriques fixes | Mixtes | Generaux / parametriques |

## Correspondance avec les types d'atomes

- **Lesson** : difficulte 0-1 (les exemples dans les lecons sont de niveau 0-1)
- **Exercise application** : difficulte 1
- **Exercise approfondissement** : difficulte 2
- **Exercise synthese** : difficulte 2-3
- **Exercise probleme** : difficulte 3
- **QCM** : difficulte 0-2 (typiquement 1)
