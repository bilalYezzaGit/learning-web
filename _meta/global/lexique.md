# Lexique mathematique — Conventions globales

Conventions de vocabulaire et de notation partagees par tous les modules du programme tunisien.

---

## Notations generales

- Virgule decimale : $0{,}1$ et non $0.1$ (convention francaise tunisienne)
- Intervalles ouverts : $]a, b[$ et non $(a, b)$ (convention francaise)
- Partie entiere : $E(x)$
- Valeur absolue : $|x|$
- Ensembles : $\mathbb{N}$, $\mathbb{Z}$, $\mathbb{Q}$, $\mathbb{R}$
- Vecteurs : $\vec{u}$, $\vec{i}$, $\vec{j}$
- Repere : $(O, \vec{i}, \vec{j})$
- Courbe representative : $C_f$, $\zeta_f$ ou $C$ selon la source

## Formules de redaction consacrees

Les expressions suivantes sont standard dans le programme tunisien :

- "D'ou..." — transition vers une conclusion
- "Par suite..." — enchainement logique
- "On en deduit que..." — deduction
- "Il en resulte que..." — consequence
- "Soit f une fonction definie sur..." — ouverture standard
- "Montrons que..." — debut de demonstration
- "Or..." — introduction d'un argument complementaire

## Conventions de demonstration

- Une implication ($\Rightarrow$) n'est pas une equivalence ($\Leftrightarrow$) — toujours preciser le sens
- Reciproque : doit etre demontree ou refutee separement
- Contre-exemple : un seul suffit pour refuter une assertion universelle
- Recurrence : base + heredite + conclusion obligatoires

## Verbes d'action et niveaux de rigueur

| Verbe | Niveau attendu |
|-------|---------------|
| Montrer que / Demontrer que | Demonstration complete |
| Justifier | Argument precis, pas forcement une demonstration formelle |
| Verifier que | Calcul direct qui confirme |
| Determiner | Trouver la valeur par le calcul |
| En deduire | Utiliser le resultat precedent |
| Calculer | Application directe d'une formule |
