<Enonce>
Resoudre dans $\mathbb{R}$ les equations suivantes :

1. $x^2 - 5x + 6 = 0$
2. $2x^2 + 3x - 2 = 0$
3. $x^2 - 4x + 4 = 0$
4. $x^2 + x + 1 = 0$
</Enonce>

<Solution>
**1.** Resolution de $x^2 - 5x + 6 = 0$ :

$a = 1$, $b = -5$, $c = 6$

$$\Delta = b^2 - 4ac = (-5)^2 - 4 \times 1 \times 6 = 25 - 24 = 1 > 0$$

Deux solutions distinctes :

$$x_1 = \dfrac{-b - \sqrt{\Delta}}{2a} = \dfrac{5 - 1}{2} = 2$$

$$x_2 = \dfrac{-b + \sqrt{\Delta}}{2a} = \dfrac{5 + 1}{2} = 3$$

$S = \{2 ; 3\}$

**2.** Resolution de $2x^2 + 3x - 2 = 0$ :

$a = 2$, $b = 3$, $c = -2$

$$\Delta = 3^2 - 4 \times 2 \times (-2) = 9 + 16 = 25 > 0$$

Deux solutions distinctes :

$$x_1 = \dfrac{-3 - 5}{4} = \dfrac{-8}{4} = -2$$

$$x_2 = \dfrac{-3 + 5}{4} = \dfrac{2}{4} = \dfrac{1}{2}$$

$S = \left\{-2 ; \dfrac{1}{2}\right\}$

**3.** Resolution de $x^2 - 4x + 4 = 0$ :

$a = 1$, $b = -4$, $c = 4$

$$\Delta = (-4)^2 - 4 \times 1 \times 4 = 16 - 16 = 0$$

Une solution double :

$$x_0 = \dfrac{-b}{2a} = \dfrac{4}{2} = 2$$

$S = \{2\}$

**4.** Resolution de $x^2 + x + 1 = 0$ :

$a = 1$, $b = 1$, $c = 1$

$$\Delta = 1^2 - 4 \times 1 \times 1 = 1 - 4 = -3 < 0$$

Pas de solution reelle.

$S = \emptyset$
</Solution>

<Methode>
**Methode du discriminant** pour $ax^2 + bx + c = 0$ :

1. Calculer $\Delta = b^2 - 4ac$

2. Analyser le signe de $\Delta$ :
   - Si $\Delta > 0$ : deux solutions $x_1 = \dfrac{-b - \sqrt{\Delta}}{2a}$ et $x_2 = \dfrac{-b + \sqrt{\Delta}}{2a}$
   - Si $\Delta = 0$ : une solution double $x_0 = \dfrac{-b}{2a}$
   - Si $\Delta < 0$ : pas de solution reelle
</Methode>

<Hint>
Pour chaque equation, identifiez d'abord les coefficients $a$, $b$ et $c$, puis calculez le discriminant.
</Hint>
