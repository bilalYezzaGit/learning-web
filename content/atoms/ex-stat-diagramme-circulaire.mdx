<Enonce>
200 eleves d'un lycee ont indique leur mode de transport pour se rendre au lycee :

- Bus : 80 eleves
- A pied : 50 eleves
- Voiture : 40 eleves
- Velo : 20 eleves
- Autre : 10 eleves

1. Calculer la **frequence** de chaque mode de transport.
2. Calculer l'**angle** correspondant a chaque secteur dans un diagramme circulaire.
3. Quel **pourcentage** d'eleves utilise un mode de transport ecologique (marche a pied + velo) ?
</Enonce>

<Solution>
**1. Frequences**

La frequence se calcule par :

$$f = \frac{\text{effectif}}{\text{effectif total}}$$

| Mode | Effectif | Frequence |
|------|----------|-----------|
| Bus | 80 | $\frac{80}{200} = 0,4$ |
| A pied | 50 | $\frac{50}{200} = 0,25$ |
| Voiture | 40 | $\frac{40}{200} = 0,2$ |
| Velo | 20 | $\frac{20}{200} = 0,1$ |
| Autre | 10 | $\frac{10}{200} = 0,05$ |
| **Total** | **200** | **1** |

**2. Angles du diagramme circulaire**

Un cercle complet fait $360°$. L'angle d'un secteur est proportionnel a la frequence :

$$\text{Angle} = \text{frequence} \times 360°$$

| Mode | Frequence | Angle |
|------|-----------|-------|
| Bus | 0,4 | $0,4 \times 360° = 144°$ |
| A pied | 0,25 | $0,25 \times 360° = 90°$ |
| Voiture | 0,2 | $0,2 \times 360° = 72°$ |
| Velo | 0,1 | $0,1 \times 360° = 36°$ |
| Autre | 0,05 | $0,05 \times 360° = 18°$ |
| **Total** | **1** | **360°** |

**3. Pourcentage de transport ecologique**

Transport ecologique = A pied + Velo

$$\text{Effectif} = 50 + 20 = 70 \text{ eleves}$$

$$\text{Pourcentage} = \frac{70}{200} \times 100 = 35\%$$

**35% des eleves** utilisent un mode de transport ecologique.
</Solution>

<Methode>
**Diagramme circulaire (camembert)** :

Pour construire un diagramme circulaire :

1. **Calculer les frequences** : $f_i = \frac{n_i}{N}$
2. **Calculer les angles** : $\alpha_i = f_i \times 360°$
3. **Verifier** que $\sum \alpha_i = 360°$
4. **Tracer** le cercle avec les secteurs angulaires

**Formule directe pour l'angle** :

$$\text{Angle} = \frac{\text{effectif}}{\text{effectif total}} \times 360°$$

**Remarque** : Le diagramme circulaire est particulierement adapte pour visualiser des **proportions** ou des **repartitions**.
</Methode>
