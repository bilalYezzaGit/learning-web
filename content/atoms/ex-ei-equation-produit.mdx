<Enonce>
Resoudre les equations suivantes :

a) $(x-2)(x+5) = 0$

b) $(3x+1)(2x-7) = 0$

c) $x(4x-3) = 0$

d) $(x-1)^2 = 0$
</Enonce>

<Solution>
**a)** $(x-2)(x+5) = 0$

Un produit est nul si et seulement si l'un de ses facteurs est nul :

$$x - 2 = 0 \quad \text{ou} \quad x + 5 = 0$$

$$x = 2 \quad \text{ou} \quad x = -5$$

L'ensemble solution est $S = \{-5 ; 2\}$.

**b)** $(3x+1)(2x-7) = 0$

$$3x + 1 = 0 \quad \text{ou} \quad 2x - 7 = 0$$

$$3x = -1 \quad \text{ou} \quad 2x = 7$$

$$x = -\frac{1}{3} \quad \text{ou} \quad x = \frac{7}{2}$$

L'ensemble solution est $S = \left\{-\frac{1}{3} ; \frac{7}{2}\right\}$.

**c)** $x(4x-3) = 0$

$$x = 0 \quad \text{ou} \quad 4x - 3 = 0$$

$$x = 0 \quad \text{ou} \quad x = \frac{3}{4}$$

L'ensemble solution est $S = \left\{0 ; \frac{3}{4}\right\}$.

**d)** $(x-1)^2 = 0$

$$(x-1)(x-1) = 0$$

$$x - 1 = 0$$

$$x = 1$$

L'ensemble solution est $S = \{1\}$ (racine double).
</Solution>
