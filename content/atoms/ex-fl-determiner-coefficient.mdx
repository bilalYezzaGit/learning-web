<Enonce>
Determiner le coefficient de la fonction lineaire $f$ dans chacun des cas suivants :

a) $f(3) = 12$

b) $f(-2) = 5$

c) La representation graphique de $f$ passe par le point $A(4, -6)$

d) $f(7) = 0$
</Enonce>

<Solution>
Pour une fonction lineaire $f(x) = ax$, on a $a = \frac{f(x)}{x}$ pour tout $x \neq 0$.

**a)** $f(3) = 12$

$$a = \frac{f(3)}{3} = \frac{12}{3} = 4$$

Le coefficient est $a = 4$.

**b)** $f(-2) = 5$

$$a = \frac{f(-2)}{-2} = \frac{5}{-2} = -\frac{5}{2}$$

Le coefficient est $a = -\frac{5}{2}$.

**c)** Le graphe passe par $A(4, -6)$

Cela signifie que $f(4) = -6$, donc :

$$a = \frac{f(4)}{4} = \frac{-6}{4} = -\frac{3}{2}$$

Le coefficient est $a = -\frac{3}{2}$.

**d)** $f(7) = 0$

$$a = \frac{f(7)}{7} = \frac{0}{7} = 0$$

Le coefficient est $a = 0$. La fonction est la fonction nulle $f(x) = 0$.
</Solution>

<Methode>
Pour determiner le coefficient $a$ d'une fonction lineaire $f(x) = ax$ :

Si on connait l'image $f(x_0)$ d'un nombre $x_0 \neq 0$, alors :

$$a = \frac{f(x_0)}{x_0}$$

**Remarque** : le graphe d'une fonction lineaire passe toujours par l'origine $O(0, 0)$ et par un point $M(x, ax)$.
</Methode>

<Erreurs>
- Ne pas confondre fonction lineaire et fonction affine : une fonction lineaire a toujours $f(0) = 0$
- Oublier de verifier si $x = 0$ avant de diviser (ici, tous les cas donnent $x \neq 0$)
</Erreurs>
