### Representation graphique

La representation graphique d'une fonction lineaire $f(x) = ax$ dans un repere $(O; \vec{i}, \vec{j})$ possede une propriete remarquable.

<Property title="Graphe d'une fonction lineaire">

Le graphe d'une fonction lineaire $f(x) = ax$ est une **droite passant par l'origine** $O(0; 0)$.

</Property>

### Construction du graphe

Pour tracer le graphe d'une fonction lineaire $f(x) = ax$ :

1. Placer l'origine $O(0; 0)$ (car $f(0) = 0$)
2. Calculer $f(1) = a$ et placer le point $A(1; a)$
3. Tracer la droite $(OA)$

**Il suffit de deux points pour tracer une droite**, et l'un d'eux est toujours l'origine.

<Example title="Tracer les graphes de f(x) = 2x et g(x) = -x">

**Pour $f(x) = 2x$ :**
- Point $O(0; 0)$ : $f(0) = 0$
- Point $A(1; 2)$ : $f(1) = 2$
- Le graphe est la droite $(OA)$

**Pour $g(x) = -x$ :**
- Point $O(0; 0)$ : $g(0) = 0$
- Point $B(1; -1)$ : $g(1) = -1$
- Le graphe est la droite $(OB)$

</Example>

### Le coefficient directeur

Le coefficient $a$ determine l'**inclinaison** (ou **pente**) de la droite :

- Si **$a > 0$** : la fonction est **croissante**, la droite "monte" de gauche a droite
- Si **$a < 0$** : la fonction est **decroissante**, la droite "descend" de gauche a droite
- Si **$a = 0$** : la fonction est **constante** egale a $0$, la droite est l'axe des abscisses

Plus la valeur absolue $|a|$ est grande, plus la droite est "inclinee" par rapport a l'axe horizontal.

<Remark>

Le coefficient $a$ s'appelle aussi le **coefficient directeur** car il indique la direction de la droite.

Par exemple :
- $f(x) = 3x$ : coefficient $a = 3$, droite "plus inclinee" que $g(x) = 2x$
- $h(x) = 0,5x$ : coefficient $a = 0,5$, droite "moins inclinee" que $g(x) = 2x$

</Remark>

<Attention>

Attention : toutes les droites passant par l'origine ne sont **pas** des graphes de fonctions lineaires.

La **droite verticale** passant par $O$ (l'axe des ordonnees) n'est pas le graphe d'une fonction, car a une valeur de $x$ correspond plusieurs valeurs de $y$.

Seules les droites **non verticales** passant par l'origine sont des graphes de fonctions lineaires.

</Attention>
