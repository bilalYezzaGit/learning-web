<Enonce>
Soit le vecteur $\vec{u} = \vec{AB}$ avec A(1;2) et B(4;5).

1. Calculer les coordonnees du vecteur $\vec{u}$.
2. Construire les images de C(2;0), D(5;1) et E(0;3) par la translation de vecteur $\vec{u}$.
3. Donner les coordonnees de C', D' et E', les images respectives de C, D et E.
</Enonce>

<Solution>
**1.** Coordonnees du vecteur $\vec{u}$ :

$$\vec{u} = \vec{AB} = \begin{pmatrix} 4-1 \\ 5-2 \end{pmatrix} = \begin{pmatrix} 3 \\ 3 \end{pmatrix}$$

**2.** Construction des images :

La translation de vecteur $\vec{u}$ transforme un point M de coordonnees (x;y) en un point M' de coordonnees (x+3;y+3).

**3.** Coordonnees des images :

- **Image de C(2;0)** : $C' = (2+3 ; 0+3) = (5 ; 3)$

  Car $\vec{CC'} = \vec{u}$, donc $C' = C + \vec{u}$

- **Image de D(5;1)** : $D' = (5+3 ; 1+3) = (8 ; 4)$

  Car $\vec{DD'} = \vec{u}$, donc $D' = D + \vec{u}$

- **Image de E(0;3)** : $E' = (0+3 ; 3+3) = (3 ; 6)$

  Car $\vec{EE'} = \vec{u}$, donc $E' = E + \vec{u}$

**Verification** : On verifie que $\vec{CC'} = \vec{DD'} = \vec{EE'} = \vec{u} = \begin{pmatrix} 3 \\ 3 \end{pmatrix}$.
</Solution>

<Methode>
Pour construire l'image M' d'un point M par la translation de vecteur $\vec{u} = \begin{pmatrix} a \\ b \end{pmatrix}$ :

1. Si M a pour coordonnees (x;y), alors M' a pour coordonnees (x+a;y+b)
2. Graphiquement : tracer le vecteur $\vec{u}$ a partir de M, l'extremite est M'
3. Verifier que $\vec{MM'} = \vec{u}$
</Methode>
