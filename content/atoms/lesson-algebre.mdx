### Developpement

**Distributivite** : $k(a + b) = ka + kb$

**Double distributivite** : $(a + b)(c + d) = ac + ad + bc + bd$

### Identites remarquables

- $(a + b)^2 = a^2 + 2ab + b^2$
- $(a - b)^2 = a^2 - 2ab + b^2$
- $(a + b)(a - b) = a^2 - b^2$

### Factorisation

**Facteur commun** : $ka + kb = k(a + b)$

**Difference de carres** : $a^2 - b^2 = (a + b)(a - b)$

**Trinome carre** : $a^2 + 2ab + b^2 = (a + b)^2$

### Simplification de fractions algebriques

Pour simplifier une fraction algebrique :

1. Factoriser le numerateur et le denominateur.
2. Simplifier par les facteurs communs.
