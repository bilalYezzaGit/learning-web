<Question>
Le developpement de $(a + b)^2$ est :
</Question>

<Option>$a^2 + b^2$</Option>
<Option correct>$a^2 + 2ab + b^2$</Option>
<Option>$a^2 - 2ab + b^2$</Option>
<Option>$2a + 2b$</Option>

<Explanation>
$(a + b)^2 = a^2 + 2ab + b^2$ est une identite remarquable.
</Explanation>
