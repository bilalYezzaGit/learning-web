<Enonce>
Soit les vecteurs $\vec{u} = \begin{pmatrix} 2 \\ 1 \end{pmatrix}$ et $\vec{v} = \begin{pmatrix} -1 \\ 3 \end{pmatrix}$.

On considere le point A(3;2).

1. Calculer les coordonnees du point B, image de A par la translation de vecteur $\vec{u}$.
2. Calculer les coordonnees du point C, image de B par la translation de vecteur $\vec{v}$.
3. Calculer le vecteur $\vec{w} = \vec{u} + \vec{v}$.
4. Verifier que C est l'image de A par la translation de vecteur $\vec{w}$.
5. Que peut-on conclure sur la composition de deux translations ?
</Enonce>

<Solution>
**1.** Image de A par la translation de vecteur $\vec{u}$ :

$$B = A + \vec{u} = (3+2 ; 2+1) = (5 ; 3)$$

**2.** Image de B par la translation de vecteur $\vec{v}$ :

$$C = B + \vec{v} = (5+(-1) ; 3+3) = (4 ; 6)$$

**3.** Calcul du vecteur somme $\vec{w}$ :

$$\vec{w} = \vec{u} + \vec{v} = \begin{pmatrix} 2 \\ 1 \end{pmatrix} + \begin{pmatrix} -1 \\ 3 \end{pmatrix} = \begin{pmatrix} 2-1 \\ 1+3 \end{pmatrix} = \begin{pmatrix} 1 \\ 4 \end{pmatrix}$$

**4.** Verification que C est l'image de A par la translation de vecteur $\vec{w}$ :

Si C est l'image de A par la translation de vecteur $\vec{w}$, alors :

$$C = A + \vec{w} = (3+1 ; 2+4) = (4 ; 6)$$

On retrouve bien les coordonnees de C calculees a la question 2.

**Verification alternative** : Calculons $\vec{AC}$ :

$$\vec{AC} = \begin{pmatrix} 4-3 \\ 6-2 \end{pmatrix} = \begin{pmatrix} 1 \\ 4 \end{pmatrix} = \vec{w}$$

Donc C est bien l'image de A par la translation de vecteur $\vec{w}$.

**5.** Conclusion :

La **composition de deux translations** de vecteurs $\vec{u}$ et $\vec{v}$ est **une translation de vecteur $\vec{w} = \vec{u} + \vec{v}$**.

Autrement dit : appliquer successivement la translation de vecteur $\vec{u}$ puis la translation de vecteur $\vec{v}$ revient a appliquer directement la translation de vecteur $\vec{u} + \vec{v}$.

**Notation** : $t_{\vec{v}} \circ t_{\vec{u}} = t_{\vec{u} + \vec{v}}$
</Solution>

<Methode>
Pour composer deux translations :

1. Appliquer la premiere translation : $B = A + \vec{u}$
2. Appliquer la deuxieme translation : $C = B + \vec{v}$
3. Calculer la somme des vecteurs : $\vec{w} = \vec{u} + \vec{v}$
4. Verifier que $C = A + \vec{w}$, ce qui montre que $t_{\vec{v}} \circ t_{\vec{u}} = t_{\vec{w}}$

**Propriete generale** : La composition de translations est **commutative** : $t_{\vec{v}} \circ t_{\vec{u}} = t_{\vec{u}} \circ t_{\vec{v}} = t_{\vec{u} + \vec{v}}$
</Methode>

<Hint>
Calculez d'abord B, puis C, puis comparez $\vec{AC}$ avec $\vec{u} + \vec{v}$.
</Hint>

<Erreurs>
- Confondre la composition de translations avec la multiplication de vecteurs (qui n'existe pas dans le plan)
- Oublier que l'ordre des translations n'a pas d'importance (commutativite)
- Ne pas verifier le resultat en calculant directement $\vec{AC}$
</Erreurs>
