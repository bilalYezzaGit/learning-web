<Enonce>
Calculer :

a) $|3 - 7|$

b) $|-5| + |2|$

c) $|3 - \sqrt{10}|$

d) $|x - 3| + |x + 1|$ pour $x = 5$, puis pour $x = 0$
</Enonce>

<Solution>
**a)** $|3 - 7| = |-4| = 4$

**b)** $|-5| + |2| = 5 + 2 = 7$

**c)** $|3 - \sqrt{10}|$

On compare $3$ et $\sqrt{10}$. Comme $\sqrt{9} = 3$ et $\sqrt{10} > \sqrt{9}$, on a $\sqrt{10} > 3$.

Donc $3 - \sqrt{10} < 0$, et par consequent :

$$|3 - \sqrt{10}| = -(3 - \sqrt{10}) = \sqrt{10} - 3$$

**d)** Pour $x = 5$ :

$$|5 - 3| + |5 + 1| = |2| + |6| = 2 + 6 = 8$$

Pour $x = 0$ :

$$|0 - 3| + |0 + 1| = |-3| + |1| = 3 + 1 = 4$$
</Solution>
