### Developpement d'expressions

**Developper** une expression, c'est transformer un produit en somme.

<Property title="Distributivite simple">

Pour tous nombres $a$, $b$, et $c$ :

$$a(b + c) = ab + ac$$

</Property>

<Example>

Developper $3(x + 5)$ :

$$3(x + 5) = 3 \times x + 3 \times 5 = 3x + 15$$

</Example>

### Double distributivite

<Property title="Double distributivite">

Pour tous nombres $a$, $b$, $c$, et $d$ :

$$(a + b)(c + d) = ac + ad + bc + bd$$

</Property>

<Example>

Developper $(2x + 3)(x + 4)$ :

$$(2x + 3)(x + 4) = 2x \times x + 2x \times 4 + 3 \times x + 3 \times 4$$

$$= 2x^2 + 8x + 3x + 12 = 2x^2 + 11x + 12$$

</Example>

### Reduction d'expressions

**Reduire** une expression, c'est regrouper les termes semblables pour simplifier l'ecriture.

<Definition title="Termes semblables">

Deux termes sont **semblables** s'ils ont la meme partie litterale. On peut alors les additionner ou les soustraire.

</Definition>

<Example>

Reduire $5x + 3 - 2x + 7$ :

$$5x + 3 - 2x + 7 = 5x - 2x + 3 + 7 = 3x + 10$$

</Example>

<Remark>

Les constantes (nombres sans $x$) se regroupent ensemble, et les termes en $x$ se regroupent ensemble.

</Remark>

<Attention>

Faites attention aux signes lors du developpement et de la reduction. Un signe $-$ devant une parenthese change tous les signes a l'interieur :

$$-(x - 3) = -x + 3$$

</Attention>
