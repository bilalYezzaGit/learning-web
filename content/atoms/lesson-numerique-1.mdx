### Ensembles de nombres

- $\mathbb{N}$ : nombres naturels (0, 1, 2, 3, ...)
- $\mathbb{Z}$ : nombres entiers (..., -2, -1, 0, 1, 2, ...)
- $\mathbb{Q}$ : nombres rationnels (fractions $\frac{a}{b}$)
- $\mathbb{R}$ : nombres reels (tous les nombres)

On a : $\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$

### Operations sur les fractions

**Addition** : $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$

**Multiplication** : $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$

**Division** : $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{ad}{bc}$

### Puissances

Pour $a \neq 0$ et $n, m \in \mathbb{Z}$ :

- $a^n \times a^m = a^{n+m}$
- $\frac{a^n}{a^m} = a^{n-m}$
- $(a^n)^m = a^{nm}$
- $a^{-n} = \frac{1}{a^n}$
- $a^0 = 1$
