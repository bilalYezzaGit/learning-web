<Question>
L'expression $(\vec{u} + \vec{v}) + \vec{w}$ est egale a :
</Question>

<Option correct>$\vec{u} + \vec{v} + \vec{w}$</Option>
<Option>$\vec{u} + (\vec{v} - \vec{w})$</Option>
<Option>$\vec{u} \times \vec{v} + \vec{w}$</Option>
<Option>$2(\vec{u} + \vec{v})$</Option>

<Explanation>
L'addition vectorielle est **associative** : on peut regrouper les vecteurs dans n'importe quel ordre.

$$(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w}) = \vec{u} + \vec{v} + \vec{w}$$

Les parentheses n'ont donc pas d'importance dans une somme de vecteurs.

**Les autres options sont fausses** :
- $\vec{u} + (\vec{v} - \vec{w}) \neq \vec{u} + \vec{v} + \vec{w}$ (signe incorrect)
- $\vec{u} \times \vec{v}$ n'a pas de sens dans ce contexte (produit vectoriel, hors programme)
- $2(\vec{u} + \vec{v}) = 2\vec{u} + 2\vec{v}$ (pas de rapport avec l'expression donnee)
</Explanation>
