<Question>
Si le plan de coupe n'est PAS parallele a la base d'une pyramide a base carree, la section est forcement :
</Question>

<Option>Un carre</Option>
<Option>Un triangle</Option>
<Option>Un rectangle</Option>
<Option correct>Aucune de ces reponses, ca depend</Option>

<Explanation>
Quand le plan n'est **pas parallele a la base**, la forme de la section depend de l'orientation du plan :

- Si le plan coupe 3 faces : **triangle**
- Si le plan coupe 4 faces : **quadrilatere** (pas forcement un carre ni un rectangle)
- Si le plan coupe 5 faces : **pentagone**

La reponse "ca depend de l'orientation" est donc correcte.
</Explanation>
