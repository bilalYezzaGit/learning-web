<Question>
$-(x - 3)^2$ est egal a :
</Question>

<Option correct>$-x^2 + 6x - 9$</Option>
<Option>$-x^2 - 6x - 9$</Option>
<Option>$x^2 - 6x + 9$</Option>
<Option>$-x^2 + 6x + 9$</Option>

<Explanation>
On procede en deux etapes.

**Etape 1** : Developper $(x-3)^2$ avec l'identite $(a-b)^2 = a^2 - 2ab + b^2$ :

$$(x-3)^2 = x^2 - 6x + 9$$

**Etape 2** : Appliquer le signe moins devant, qui change **tous les signes** :

$$-(x^2 - 6x + 9) = -x^2 + 6x - 9$$

**Piege courant** : Ne changer que le signe du premier terme !
</Explanation>
