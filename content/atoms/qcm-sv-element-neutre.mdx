<Question>
Que vaut $\vec{u} + \vec{0}$ ?
</Question>

<Option>$\vec{0}$</Option>
<Option>$2\vec{u}$</Option>
<Option correct>$\vec{u}$</Option>
<Option>$-\vec{u}$</Option>

<Explanation>
Le vecteur nul $\vec{0}$ est l'**element neutre** de l'addition vectorielle : ajouter $\vec{0}$ a n'importe quel vecteur ne change pas ce vecteur.

$$\vec{u} + \vec{0} = \vec{u}$$

Cette propriete est vraie pour tout vecteur $\vec{u}$.

**Rappel** : Le vecteur nul est le vecteur dont l'extremite coincide avec l'origine. On peut l'ecrire $\vec{0} = \vec{AA}$ pour tout point $A$.
</Explanation>
