Quel ensemble contient tous les autres ?

- [ ] $\mathbb{N}$
- [ ] $\mathbb{Z}$
- [ ] $\mathbb{Q}$
- [x] $\mathbb{R}$

> $\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$, donc $\mathbb{R}$ contient tous les autres.
