Une rotation d'angle $180°$ est equivalente a :

- [ ] Une translation
- [ ] Une symetrie axiale
- [x] Une symetrie centrale
- [ ] Un quart de tour

> Une rotation de $180°$ autour d'un point $O$ est la symetrie centrale de centre $O$.
