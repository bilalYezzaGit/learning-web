<Enonce>
Soit ABCD un parallelogramme.

On note I le milieu du segment [AB] et J le milieu du segment [CD].

Demontrer que le quadrilatere IJDC est un parallelogramme en utilisant les vecteurs.
</Enonce>

<Solution>
**Hypotheses** :
- ABCD est un parallelogramme, donc $\vec{AB} = \vec{DC}$
- I est le milieu de [AB], donc $\vec{AI} = \frac{1}{2}\vec{AB}$
- J est le milieu de [CD], donc $\vec{DJ} = \frac{1}{2}\vec{DC}$

**Demonstration** :

Pour montrer que IJDC est un parallelogramme, il suffit de prouver que $\vec{IJ} = \vec{DC}$.

**Etape 1** : Exprimons $\vec{IJ}$ en utilisant la relation de Chasles.

$$\vec{IJ} = \vec{IA} + \vec{AD} + \vec{DJ}$$

**Etape 2** : Remplacons les vecteurs par leurs expressions.

- $\vec{IA} = -\vec{AI} = -\frac{1}{2}\vec{AB}$ (car I est le milieu de [AB])

- $\vec{AD}$ reste inchange

- $\vec{DJ} = \frac{1}{2}\vec{DC}$ (car J est le milieu de [CD])

Donc :

$$\vec{IJ} = -\frac{1}{2}\vec{AB} + \vec{AD} + \frac{1}{2}\vec{DC}$$

**Etape 3** : Utilisons le fait que ABCD est un parallelogramme.

Dans un parallelogramme, $\vec{AB} = \vec{DC}$. Donc :

$$\vec{IJ} = -\frac{1}{2}\vec{AB} + \vec{AD} + \frac{1}{2}\vec{AB}$$

$$\vec{IJ} = \vec{AD}$$

**Etape 4** : Utilisons a nouveau le fait que ABCD est un parallelogramme.

Dans un parallelogramme, les cotes opposes sont egaux : $\vec{AD} = \vec{BC}$.

Mais surtout, on peut aussi ecrire : $\vec{DC} = \vec{AB}$ et dans un parallelogramme, on a la relation :

$$\vec{AD} = \vec{BC}$$

Or, en fait, on cherche a montrer que $\vec{IJ} = \vec{DC}$ ou bien que le quadrilatere IJDC est un parallelogramme.

**Reprenons differemment** :

Dans ABCD parallelogramme, on a $\vec{AB} = \vec{DC}$ et $\vec{AD} = \vec{BC}$.

On a trouve que $\vec{IJ} = \vec{AD}$.

Or, dans le quadrilatere IJDC, on cherche a comparer $\vec{IJ}$ avec un autre vecteur.

**Calculons $\vec{JC}$** :

$$\vec{JC} = \frac{1}{2}\vec{DC}$$ (car J est le milieu de [CD])

Cela ne nous donne pas directement l'egalite recherchee.

**Changeons de strategie** : montrons que $\vec{DJ} = \vec{CI}$.

$$\vec{DJ} = \frac{1}{2}\vec{DC} = \frac{1}{2}\vec{AB}$$

$$\vec{CI} = \vec{CB} + \vec{BI} = -\vec{BC} + \frac{1}{2}\vec{BA} = -\vec{BC} - \frac{1}{2}\vec{AB}$$

Dans ABCD parallelogramme : $\vec{BC} = \vec{AD}$

$$\vec{CI} = -\vec{AD} - \frac{1}{2}\vec{AB}$$

Cela ne fonctionne pas non plus simplement.

**Methode correcte â€” Montrons que $\vec{IJ} = \vec{DC}$** :

On recalcule proprement :

$$\vec{IJ} = \vec{IB} + \vec{BC} + \vec{CJ}$$

Avec :
- $\vec{IB} = \frac{1}{2}\vec{AB}$ (car I est le milieu de [AB])
- $\vec{BC} = \vec{BC}$
- $\vec{CJ} = -\frac{1}{2}\vec{DC}$ (car J est le milieu de [DC])

Dans ABCD parallelogramme, $\vec{BC} = \vec{AD}$ et $\vec{AB} = \vec{DC}$.

$$\vec{IJ} = \frac{1}{2}\vec{DC} + \vec{AD} - \frac{1}{2}\vec{DC} = \vec{AD}$$

Or, comme $\vec{AD} = \vec{BC}$, on a bien $\vec{IJ} = \vec{AD} = \vec{BC}$.

Mais pour IJDC, il faut montrer que les cotes opposes sont egaux.

Finalement, **la bonne demonstration** est de montrer que **I, J sont les milieux des cotes opposes, donc IJDC n'est PAS systematiquement un parallelogramme**.

**Correction de l'enonce** : On doit montrer que **IJBA** est un parallelogramme (avec I milieu de [AB], J milieu de [DC]).

**Reelle demonstration** : Montrons que **les segments [IJ] et [AD] ont meme milieu**.

Milieu de [IJ] = Milieu de [AD], ce qui implique que IAJD est un parallelogramme.

**Conclusion correcte** : IJDC n'est pas un parallelogramme en general. L'exercice contenait une erreur. On peut demontrer que IAJD ou IBCJ sont des parallelogrammes.
</Solution>

<Methode>
Pour demontrer qu'un quadrilatere est un parallelogramme avec les vecteurs :

**Methode 1** : Montrer que les vecteurs des cotes opposes sont egaux : $\vec{AB} = \vec{DC}$

**Methode 2** : Utiliser la relation de Chasles pour exprimer un vecteur en fonction d'autres

**Methode 3** : Montrer que les diagonales se coupent en leur milieu

**Methode 4** : Dans un parallelogramme ABCD, si I est le milieu de [AB] et J le milieu de [DC], alors I et J sont symetriques par rapport au centre du parallelogramme
</Methode>

<Hint>
Utilisez la relation de Chasles pour exprimer $\vec{IJ}$ en fonction de $\vec{AB}$, $\vec{AD}$ et $\vec{DC}$. N'oubliez pas que dans un parallelogramme, $\vec{AB} = \vec{DC}$.
</Hint>

<Erreurs>
- Ne pas verifier la coherence de l'enonce avant de commencer
- Oublier d'utiliser le fait que I et J sont des milieux : $\vec{AI} = \frac{1}{2}\vec{AB}$
- Ne pas utiliser la propriete du parallelogramme ABCD : $\vec{AB} = \vec{DC}$
- Confondre les cotes opposes du quadrilatere IJDC
</Erreurs>
