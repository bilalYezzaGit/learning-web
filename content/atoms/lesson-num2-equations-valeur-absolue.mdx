### Equations avec valeur absolue

<Property title="Equation |x| = a">

Soit $a$ un nombre reel.

- Si $a < 0$ : l'equation $|x| = a$ **n'a pas de solution** (car $|x| \geq 0$ toujours)
- Si $a = 0$ : l'equation $|x| = 0$ a une seule solution : $x = 0$
- Si $a > 0$ : l'equation $|x| = a$ a **deux solutions** : $x = a$ ou $x = -a$

</Property>

<Example title="Resoudre |x| = 5">

L'equation $|x| = 5$ a deux solutions :

$$x = 5 \quad \text{ou} \quad x = -5$$

On peut verifier : $|5| = 5$ et $|-5| = 5$.

</Example>

<Attention>

L'equation $|x| = -3$ **n'a pas de solution** car une valeur absolue ne peut jamais etre negative.

</Attention>

<Property title="Equation |x - c| = a">

Pour resoudre $|x - c| = a$ avec $a > 0$ :

$$|x - c| = a \Leftrightarrow x - c = a \text{ ou } x - c = -a$$

$$\Leftrightarrow x = c + a \text{ ou } x = c - a$$

</Property>

<Example title="Resoudre |x - 3| = 2">

$$|x - 3| = 2$$

$$\Leftrightarrow x - 3 = 2 \text{ ou } x - 3 = -2$$

$$\Leftrightarrow x = 5 \text{ ou } x = 1$$

Les solutions sont $x = 1$ et $x = 5$.

</Example>

### Inequations avec valeur absolue

<Property title="Inequation |x| < a">

Soit $a > 0$. L'inequation $|x| < a$ est equivalente a :

$$-a < x < a$$

Autrement dit : $x \in ]-a, a[$.

</Property>

<Example title="Resoudre |x| < 3">

$$|x| < 3 \Leftrightarrow -3 < x < 3$$

Les solutions forment l'intervalle $]-3, 3[$.

</Example>

<Property title="Inequation |x| > a">

Soit $a > 0$. L'inequation $|x| > a$ est equivalente a :

$$x < -a \text{ ou } x > a$$

Autrement dit : $x \in ]-\infty, -a[ \cup ]a, +\infty[$.

</Property>

<Example title="Resoudre |x| > 4">

$$|x| > 4 \Leftrightarrow x < -4 \text{ ou } x > 4$$

Les solutions sont $x \in ]-\infty, -4[ \cup ]4, +\infty[$.

</Example>

### Cas general avec translation

<Property title="Inequation |x - c| < a">

Pour $a > 0$, l'inequation $|x - c| < a$ signifie que $x$ est a une distance inferieure a $a$ du point $c$ :

$$|x - c| < a \Leftrightarrow c - a < x < c + a$$

Autrement dit : $x \in ]c - a, c + a[$.

</Property>

<Example title="Resoudre |x - 3| < 2">

$$|x - 3| < 2 \Leftrightarrow 3 - 2 < x < 3 + 2$$

$$\Leftrightarrow 1 < x < 5$$

Les solutions sont $x \in ]1, 5[$.

**Interpretation geometrique** : on cherche les points $x$ dont la distance au point $3$ est inferieure a $2$.

</Example>

<Property title="Inequation |x - c| > a">

Pour $a > 0$ :

$$|x - c| > a \Leftrightarrow x < c - a \text{ ou } x > c + a$$

Autrement dit : $x \in ]-\infty, c - a[ \cup ]c + a, +\infty[$.

</Property>

<Remark>

Les inequations avec $\leq$ ou $\geq$ se traitent de la meme facon, en remplacant les crochets ouverts par des crochets fermes dans les intervalles.

Par exemple : $|x| \leq a \Leftrightarrow -a \leq x \leq a$, soit $x \in [-a, a]$.

</Remark>
