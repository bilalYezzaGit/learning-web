<Enonce>
Resoudre les inequations suivantes et donner les ensembles solutions sous forme d'intervalles :

a) $2x + 3 > 7$

b) $-3x + 1 \leq 10$

c) $4 - 5x < 2x + 18$

d) $\frac{x-2}{3} \geq 1$
</Enonce>

<Solution>
**a)** $2x + 3 > 7$

$$2x > 7 - 3$$

$$2x > 4$$

$$x > 2$$

L'ensemble solution est $S = ]2, +\infty[$.

**b)** $-3x + 1 \leq 10$

$$-3x \leq 10 - 1$$

$$-3x \leq 9$$

En divisant par $-3$ (nombre negatif), on inverse le sens de l'inegalite :

$$x \geq -3$$

L'ensemble solution est $S = [-3, +\infty[$.

**c)** $4 - 5x < 2x + 18$

$$4 - 18 < 2x + 5x$$

$$-14 < 7x$$

$$\frac{-14}{7} < x$$

$$-2 < x$$

L'ensemble solution est $S = ]-2, +\infty[$.

**d)** $\frac{x-2}{3} \geq 1$

$$x - 2 \geq 3$$

$$x \geq 5$$

L'ensemble solution est $S = [5, +\infty[$.
</Solution>

<Erreurs>
- **Oublier d'inverser le sens de l'inegalite** lorsqu'on multiplie ou divise par un nombre negatif (voir question b).
- Confondre les crochets fermÃ©s $[$ et ouverts $]$ dans les intervalles.
- Oublier que pour $x > a$, on utilise l'intervalle ouvert $]a, +\infty[$ (et non $[a, +\infty[$).
</Erreurs>
