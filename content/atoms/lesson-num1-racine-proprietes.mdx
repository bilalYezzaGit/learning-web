### Regles de calcul

Les racines carrees suivent des regles qui permettent de simplifier les calculs.

<Property title="Produit de racines carrees">

Pour tous nombres $a, b \geq 0$ :

$$\sqrt{ab} = \sqrt{a} \times \sqrt{b}$$

**La racine d'un produit est le produit des racines.**

</Property>

<Example>

- $\sqrt{4 \times 9} = \sqrt{4} \times \sqrt{9} = 2 \times 3 = 6$
- $\sqrt{16 \times 25} = \sqrt{16} \times \sqrt{25} = 4 \times 5 = 20$
- $\sqrt{2 \times 8} = \sqrt{2} \times \sqrt{8} = \sqrt{2} \times 2\sqrt{2} = 2 \times 2 = 4$

</Example>

<Property title="Quotient de racines carrees">

Pour tous nombres $a \geq 0$ et $b > 0$ :

$$\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$$

**La racine d'un quotient est le quotient des racines.**

</Property>

<Example>

- $\sqrt{\frac{9}{4}} = \frac{\sqrt{9}}{\sqrt{4}} = \frac{3}{2}$
- $\sqrt{\frac{25}{16}} = \frac{\sqrt{25}}{\sqrt{16}} = \frac{5}{4}$
- $\sqrt{\frac{1}{9}} = \frac{\sqrt{1}}{\sqrt{9}} = \frac{1}{3}$

</Example>

<Property title="Extraction de facteurs">

Pour $a \geq 0$ et $b \geq 0$ :

$$\sqrt{a^2 \times b} = a\sqrt{b}$$

**On peut extraire les carres parfaits hors de la racine.**

</Property>

<Example title="Simplification de racines">

Pour simplifier $\sqrt{72}$ :

1. On cherche le plus grand carre parfait qui divise 72
2. $72 = 36 \times 2 = 6^2 \times 2$
3. Donc : $\sqrt{72} = \sqrt{36 \times 2} = \sqrt{36} \times \sqrt{2} = 6\sqrt{2}$

**Autres exemples** :

- $\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}$
- $\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$
- $\sqrt{75} = \sqrt{25 \times 3} = 5\sqrt{3}$
- $\sqrt{32} = \sqrt{16 \times 2} = 4\sqrt{2}$

</Example>

### Comparaison de racines

<Property title="Ordre et racine carree">

Pour tous nombres $a, b \geq 0$ :

$$\text{Si } a \leq b \text{, alors } \sqrt{a} \leq \sqrt{b}$$

**La racine carree preserve l'ordre des nombres positifs.**

</Property>

<Example>

- $4 < 9$ donc $\sqrt{4} < \sqrt{9}$, c'est-a-dire $2 < 3$
- $16 < 25$ donc $\sqrt{16} < \sqrt{25}$, c'est-a-dire $4 < 5$

</Example>

### Erreurs courantes

<Attention>

**Erreur tres frequente** : la racine ne se distribue **pas** sur une somme.

$$\boxed{\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}}$$

**Contre-exemple** :

$$\sqrt{9 + 16} = \sqrt{25} = 5$$

mais

$$\sqrt{9} + \sqrt{16} = 3 + 4 = 7 \neq 5$$

**Attention** : ne jamais separer une somme sous une racine !

</Attention>

<Remark>

Pour simplifier une racine carree :

1. Decomposer le nombre en facteurs
2. Identifier les carres parfaits (4, 9, 16, 25, 36, etc.)
3. Extraire ces carres hors de la racine

Par exemple : $\sqrt{200} = \sqrt{100 \times 2} = 10\sqrt{2}$

</Remark>
