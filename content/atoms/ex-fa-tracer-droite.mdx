<Enonce>
Tracer dans un même repère orthonormé les représentations graphiques de :

1. $f(x) = 2x + 1$
2. $g(x) = -x + 4$
3. $h(x) = 3$ (fonction constante)
4. Déterminer graphiquement les coordonnées du point d'intersection de $(D_f)$ et $(D_g)$.
5. Vérifier par le calcul.
</Enonce>

<Solution>
**1. Tracé de la droite $(D_f)$ d'équation $y = 2x + 1$**

Calculons deux points :
- Pour $x = 0$ : $y = 2 \times 0 + 1 = 1$ → Point $A(0, 1)$
- Pour $x = 1$ : $y = 2 \times 1 + 1 = 3$ → Point $B(1, 3)$

On trace la droite passant par $A$ et $B$.

**2. Tracé de la droite $(D_g)$ d'équation $y = -x + 4$**

Calculons deux points :
- Pour $x = 0$ : $y = -0 + 4 = 4$ → Point $C(0, 4)$
- Pour $x = 2$ : $y = -2 + 4 = 2$ → Point $D(2, 2)$

On trace la droite passant par $C$ et $D$.

**3. Tracé de la droite $(D_h)$ d'équation $y = 3$**

C'est une droite horizontale passant par tous les points d'ordonnée $3$. Par exemple : $(0, 3)$, $(1, 3)$, $(2, 3)$, etc.

**4. Point d'intersection graphique de $(D_f)$ et $(D_g)$**

En traçant les deux droites, on observe qu'elles se coupent au point de coordonnées $(1, 3)$.

**5. Vérification par le calcul**

Le point d'intersection vérifie les deux équations simultanément :

$$2x + 1 = -x + 4$$

$$2x + x = 4 - 1$$

$$3x = 3$$

$$x = 1$$

On calcule $y$ :

$$y = 2 \times 1 + 1 = 3$$

Le point d'intersection est bien $I(1, 3)$.

**Vérification** :
- $f(1) = 2 \times 1 + 1 = 3$ ✓
- $g(1) = -1 + 4 = 3$ ✓
</Solution>

<Hint>
Pour tracer une droite, il suffit de calculer les coordonnées de deux points distincts, puis de les relier avec une règle.

Pour trouver le point d'intersection de deux droites, résoudre l'équation $f(x) = g(x)$.
</Hint>
