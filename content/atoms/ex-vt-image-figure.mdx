<Enonce>
Soit le triangle ABC avec A(1;1), B(4;2) et C(2;4).

On considere la translation de vecteur $\vec{v} = \begin{pmatrix} 2 \\ -3 \end{pmatrix}$.

1. Construire les images A', B', C' des points A, B, C par cette translation.
2. Calculer les longueurs AB et A'B'.
3. Que peut-on conclure sur la conservation des longueurs par translation ?
</Enonce>

<Solution>
**1.** Coordonnees des images :

La translation de vecteur $\vec{v} = \begin{pmatrix} 2 \\ -3 \end{pmatrix}$ transforme (x;y) en (x+2;y-3).

- $A' = (1+2 ; 1-3) = (3 ; -2)$
- $B' = (4+2 ; 2-3) = (6 ; -1)$
- $C' = (2+2 ; 4-3) = (4 ; 1)$

**2.** Calcul des longueurs :

Longueur AB :

$$AB = \sqrt{(x_B - x_A)^2 + (y_B - y_A)^2} = \sqrt{(4-1)^2 + (2-1)^2} = \sqrt{9 + 1} = \sqrt{10}$$

Longueur A'B' :

$$A'B' = \sqrt{(x_{B'} - x_{A'})^2 + (y_{B'} - y_{A'})^2} = \sqrt{(6-3)^2 + (-1-(-2))^2} = \sqrt{9 + 1} = \sqrt{10}$$

**3.** Conclusion :

On constate que $AB = A'B' = \sqrt{10}$.

**La translation conserve les longueurs.** Plus generalement, une translation conserve les distances, les angles, les aires et les formes des figures geometriques.

Le triangle A'B'C' est une **copie conforme** du triangle ABC, simplement deplace selon le vecteur $\vec{v}$.
</Solution>

<Methode>
Pour verifier la conservation des longueurs par translation :

1. Calculer les coordonnees des images par la translation
2. Calculer la longueur dans la figure initiale : $AB = \sqrt{(x_B - x_A)^2 + (y_B - y_A)^2}$
3. Calculer la longueur dans la figure image : $A'B' = \sqrt{(x_{B'} - x_{A'})^2 + (y_{B'} - y_{A'})^2}$
4. Comparer : si AB = A'B', la longueur est conservee
</Methode>

<Erreurs>
- Confondre le sens du vecteur de translation : $\vec{v} = \begin{pmatrix} 2 \\ -3 \end{pmatrix}$ signifie +2 en x et -3 en y (pas l'inverse)
- Oublier que la translation conserve TOUTES les proprietes geometriques (distances, angles, parallelisme, etc.)
</Erreurs>
