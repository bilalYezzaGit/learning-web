<Question>
Dans la configuration de Thales, si $\dfrac{AM}{AB} = \dfrac{2}{5}$, alors $\dfrac{MB}{AB}$ vaut :
</Question>

<Option>$\dfrac{2}{5}$</Option>
<Option correct>$\dfrac{3}{5}$</Option>
<Option>$\dfrac{5}{2}$</Option>
<Option>$\dfrac{2}{3}$</Option>

<Explanation>
Puisque $M$ est sur le segment $[AB]$, on a la relation :

$$AM + MB = AB$$

En divisant par $AB$ :

$$\dfrac{AM}{AB} + \dfrac{MB}{AB} = 1$$

Si $\dfrac{AM}{AB} = \dfrac{2}{5}$, alors :

$$\dfrac{MB}{AB} = 1 - \dfrac{2}{5} = \dfrac{5-2}{5} = \dfrac{3}{5}$$
</Explanation>
