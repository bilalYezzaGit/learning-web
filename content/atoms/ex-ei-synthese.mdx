<Enonce>
Soit $E(x) = (3x - 2)(x + 4) - (3x - 2)(2x - 1)$.

1. Developper et reduire $E(x)$.

2. Factoriser $E(x)$.

3. Resoudre l'equation $E(x) = 0$.

4. Resoudre l'inequation $E(x) \geq 0$.

5. Resoudre l'inequation $\frac{E(x)}{x - 1} \leq 0$.
</Enonce>

<Solution>
**1. Developpement et reduction de $E(x)$**

$$E(x) = (3x - 2)(x + 4) - (3x - 2)(2x - 1)$$

Developpons chaque produit :

$$E(x) = 3x^2 + 12x - 2x - 8 - (6x^2 - 3x - 4x + 2)$$

$$E(x) = 3x^2 + 10x - 8 - 6x^2 + 3x + 4x - 2$$

$$E(x) = (3x^2 - 6x^2) + (10x + 3x + 4x) + (-8 - 2)$$

$$E(x) = -3x^2 + 17x - 10$$

---

**2. Factorisation de $E(x)$**

On reprend l'expression initiale et on met $(3x - 2)$ en facteur commun :

$$E(x) = (3x - 2)(x + 4) - (3x - 2)(2x - 1)$$

$$E(x) = (3x - 2)[(x + 4) - (2x - 1)]$$

$$E(x) = (3x - 2)(x + 4 - 2x + 1)$$

$$E(x) = (3x - 2)(-x + 5)$$

---

**3. Resolution de $E(x) = 0$**

$$(3x - 2)(-x + 5) = 0$$

Un produit est nul si et seulement si l'un de ses facteurs est nul :

$$3x - 2 = 0 \quad \text{ou} \quad -x + 5 = 0$$

$$x = \frac{2}{3} \quad \text{ou} \quad x = 5$$

L'ensemble solution est $S = \left\{\frac{2}{3} ; 5\right\}$.

---

**4. Resolution de $E(x) \geq 0$**

On etudie le signe de $E(x) = (3x - 2)(-x + 5)$.

**Signe de $3x - 2$** :
- $3x - 2 = 0 \Rightarrow x = \frac{2}{3}$
- Si $x < \frac{2}{3}$ : negatif
- Si $x > \frac{2}{3}$ : positif

**Signe de $-x + 5$** :
- $-x + 5 = 0 \Rightarrow x = 5$
- Si $x < 5$ : positif
- Si $x > 5$ : negatif

**Tableau de signes** :

| $x$ | $-\infty$ |  | $\frac{2}{3}$ |  | $5$ |  | $+\infty$ |
|-----|-----------|--|---------------|--|-----|--|-----------|
| $3x-2$ | $-$ | | $0$ | $+$ | $+$ | | $+$ |
| $-x+5$ | $+$ | | $+$ | $+$ | $0$ | $-$ | $-$ |
| $E(x)$ | $-$ | | $0$ | $+$ | $0$ | $-$ | $-$ |

On cherche $E(x) \geq 0$ (positif ou nul).

L'ensemble solution est $S = \left[\frac{2}{3}, 5\right]$.

---

**5. Resolution de $\frac{E(x)}{x - 1} \leq 0$**

On a $E(x) = (3x - 2)(-x + 5)$, donc :

$$\frac{(3x - 2)(-x + 5)}{x - 1} \leq 0$$

**Valeur interdite** : $x = 1$ (denominateur nul).

On ajoute le facteur $x - 1$ au tableau de signes :

**Signe de $x - 1$** :
- $x - 1 = 0 \Rightarrow x = 1$
- Si $x < 1$ : negatif
- Si $x > 1$ : positif

**Tableau de signes complet** :

| $x$ | $-\infty$ |  | $\frac{2}{3}$ |  | $1$ |  | $5$ |  | $+\infty$ |
|-----|-----------|--|---------------|--|-----|--|-----|--|-----------|
| $3x-2$ | $-$ | | $0$ | $+$ | $+$ | | $+$ | | $+$ |
| $-x+5$ | $+$ | | $+$ | $+$ | $+$ | | $0$ | $-$ | $-$ |
| $x-1$ | $-$ | | $-$ | $-$ | $\|$ | $+$ | $+$ | $+$ | $+$ |
| Quotient | $+$ | | $0$ | $-$ | $\|$ | $+$ | $0$ | $-$ | $-$ |

On cherche les valeurs ou le quotient est negatif ou nul ($\leq 0$).

**Attention** : $x = 1$ est exclu (valeur interdite).

L'ensemble solution est $S = \left]-\infty, \frac{2}{3}\right] \cup ]1, 5]$.
</Solution>

<Erreurs>
- **Oublier d'exclure $x = 1$** dans la question 5 : le denominateur s'annule, cette valeur doit etre exclue de l'ensemble solution meme si le numerateur est non nul en 1.
- **Confondre developpement et factorisation** : bien verifier quelle forme est demandee.
- **Erreur de signe dans le tableau** : verifier soigneusement le signe de chaque facteur, surtout pour $-x + 5$ qui est negatif pour $x > 5$.
- **Oublier les crochets fermes** aux racines dans la question 4 : on cherche $E(x) \geq 0$, donc les valeurs ou $E(x) = 0$ font partie de la solution.
</Erreurs>
