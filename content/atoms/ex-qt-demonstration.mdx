<Enonce>
Soit $ABC$ un triangle rectangle isocele en $A$, c'est-a-dire $AB = AC$ et $\widehat{BAC} = 90°$.

On note $r$ le quart de tour direct de centre $A$.

1. Montrer que $r(B) = C$.
2. En deduire que les droites $(AB)$ et $(AC)$ sont perpendiculaires.
3. Calculer la longueur $BC$ en fonction de $AB$. On rappelle que $AB = AC$.
</Enonce>

<Solution>
**1. Montrer que $r(B) = C$**

Pour montrer que $C$ est l'image de $B$ par le quart de tour direct de centre $A$, il faut verifier deux conditions :

- $AB = AC$ (conservation de la distance)
- L'angle $\widehat{BAC} = 90°$ dans le sens positif

**Verification** :

D'apres l'enonce, le triangle $ABC$ est rectangle isocele en $A$, donc :

- $AB = AC$ ✓
- $\widehat{BAC} = 90°$ ✓

De plus, le sens de l'angle $\widehat{BAC}$ est positif (sens trigonometrique) si on passe de $[AB)$ a $[AC)$ dans le sens anti-horaire.

**Conclusion** : Le point $C$ verifie bien les deux conditions, donc $C = r(B)$.

**2. Perpendicularite de $(AB)$ et $(AC)$**

Par definition du quart de tour, l'angle entre $\vec{AB}$ et $\vec{AC}$ est de $90°$.

Cela signifie que les vecteurs $\vec{AB}$ et $\vec{AC}$ sont perpendiculaires.

Par consequent, les droites $(AB)$ et $(AC)$ sont perpendiculaires.

**Remarque** : Ceci est deja donne dans l'enonce puisque le triangle est rectangle en $A$, mais nous venons de le redemontrer en utilisant les proprietes du quart de tour.

**3. Calcul de $BC$**

On applique le **theoreme de Pythagore** dans le triangle rectangle $ABC$ :

$$BC^2 = AB^2 + AC^2$$

Puisque $AB = AC$, on peut ecrire :

$$BC^2 = AB^2 + AB^2 = 2AB^2$$

D'ou :

$$BC = \sqrt{2AB^2} = AB\sqrt{2}$$

**Conclusion** : Dans un triangle rectangle isocele, l'hypotenuse est egale au cote multiplie par $\sqrt{2}$.

$$\boxed{BC = AB\sqrt{2}}$$

**Interpretation geometrique** : Le quart de tour conserve les longueurs, donc $AC = AB$. Le triangle etant rectangle en $A$, le theoreme de Pythagore permet de calculer $BC$.
</Solution>

<Methode>
Pour demontrer qu'une transformation est un quart de tour :

1. Verifier la **conservation des distances** : $OA = OB$ ou $O$ est le centre
2. Verifier l'**angle de rotation** : $\widehat{AOB} = 90°$
3. Verifier le **sens** de rotation (direct ou indirect)

Pour calculer une longueur dans un triangle rectangle :

- Utiliser le **theoreme de Pythagore** : $(\text{hypotenuse})^2 = (\text{cote}_1)^2 + (\text{cote}_2)^2$
</Methode>

<Erreurs>
**Erreur 1** : Oublier de verifier le sens du quart de tour. Un angle de $90°$ peut etre mesure dans le sens positif ou negatif.

**Erreur 2** : Confondre le quart de tour direct et le quart de tour indirect. Ici, on doit verifier que l'angle $\widehat{BAC}$ est bien mesure dans le sens trigonometrique (sens positif).

**Erreur 3** : Oublier de simplifier $\sqrt{2AB^2}$ en $AB\sqrt{2}$.

**Erreur 4** : Appliquer le theoreme de Pythagore avec $BC$ comme un cote au lieu de l'hypotenuse. Dans un triangle rectangle en $A$, l'hypotenuse est le cote oppose a l'angle droit, donc $BC$.
</Erreurs>
