### Introduction

Le théorème de Thalès s'applique dans plusieurs configurations géométriques. Savoir **reconnaître** rapidement ces configurations dans une figure est essentiel pour appliquer le théorème efficacement.

### Configuration triangle

<Definition title="Configuration triangle">

Dans un triangle $ABC$, si une droite $(d)$ est **parallèle** à l'un des côtés du triangle (par exemple $(BC)$) et coupe les deux autres côtés en $M$ et $N$, alors on a une **configuration triangle**.

Cette configuration forme un petit triangle $AMN$ **à l'intérieur** du grand triangle $ABC$, avec $(MN) \parallel (BC)$.

</Definition>

Dans cette configuration, le théorème de Thalès s'écrit :

$$\frac{AM}{AB} = \frac{AN}{AC} = \frac{MN}{BC}$$

<Example title="Triangle avec droite parallele">

Dans le triangle $ABC$, la droite $(MN)$ est parallèle au côté $(BC)$.

Si $AM = 3$ cm, $MB = 2$ cm, $AN = 4,5$ cm, calculer $AC$.

**Solution :**

On a $AB = AM + MB = 3 + 2 = 5$ cm.

Par Thalès (configuration triangle) :

$$\frac{AM}{AB} = \frac{AN}{AC}$$

$$\frac{3}{5} = \frac{4,5}{AC}$$

$$3 \times AC = 5 \times 4,5 = 22,5$$

$$AC = \frac{22,5}{3} = 7,5 \text{ cm}$$

</Example>

### Configuration papillon

<Definition title="Configuration papillon">

La configuration **papillon** (ou **sablier**) se produit quand deux droites $(AB)$ et $(CD)$ se coupent en un point $O$ situé **entre** les deux droites parallèles $(AC)$ et $(BD)$.

Cette configuration ressemble visuellement à un papillon ou à un sablier.

</Definition>

Dans cette configuration, le théorème de Thalès s'écrit :

$$\frac{OA}{OB} = \frac{OC}{OD} = \frac{AC}{BD}$$

<Remark>

Dans la configuration papillon, le point d'intersection $O$ est **entre** les segments. On mesure les distances depuis $O$ jusqu'aux extrémités.

</Remark>

<Example title="Configuration papillon">

Deux droites $(AB)$ et $(CD)$ se coupent en $O$.

Les droites $(AC)$ et $(BD)$ sont parallèles.

On donne : $OA = 4$ cm, $OB = 6$ cm, $OC = 5$ cm.

Calculer $OD$.

**Solution :**

Les droites $(AC)$ et $(BD)$ sont parallèles, donc par Thalès (configuration papillon) :

$$\frac{OA}{OB} = \frac{OC}{OD}$$

$$\frac{4}{6} = \frac{5}{OD}$$

$$4 \times OD = 6 \times 5 = 30$$

$$OD = \frac{30}{4} = 7,5 \text{ cm}$$

</Example>

### Methode pour reperer une configuration de Thales

Pour identifier une configuration de Thalès dans une figure complexe, suivre cette démarche systématique :

**Étape 1 : Identifier les droites parallèles**

Chercher deux droites marquées comme parallèles (notation $\parallel$ ou indication dans l'énoncé).

**Étape 2 : Identifier le point d'intersection**

Chercher le point où deux autres droites se coupent. Ce point est le **sommet** de la configuration.

**Étape 3 : Déterminer la configuration**

- Si le point d'intersection est **à l'extérieur** des parallèles → **configuration triangle**
- Si le point d'intersection est **entre** les parallèles → **configuration papillon**

**Étape 4 : Écrire les rapports**

En fonction de la configuration identifiée, écrire les trois rapports égaux du théorème de Thalès.

<Attention>

Une erreur fréquente est de confondre les deux configurations et d'écrire les mauvais rapports. Toujours **dessiner clairement** la figure et identifier le type de configuration avant d'écrire les rapports !

</Attention>

### Cas particuliers

<Remark title="Droites qui se coupent hors de la figure">

Parfois, le point d'intersection des deux droites sécantes n'est pas visible sur la figure donnée. Il faut alors **prolonger** mentalement les droites pour identifier leur point d'intersection.

</Remark>

<Example title="Reconnaissance de configuration">

Dans une figure, on voit :
- Un triangle $ABC$
- Un point $M$ sur $[AB]$ et un point $N$ sur $[AC]$
- L'indication $(MN) \parallel (BC)$

**Quelle est la configuration ?**

C'est une **configuration triangle** car :
1. Les droites $(AB)$ et $(AC)$ se coupent en $A$
2. Le point $A$ est à l'extérieur de la zone comprise entre les parallèles $(MN)$ et $(BC)$
3. Un petit triangle $AMN$ est formé à l'intérieur du grand triangle $ABC$

On peut donc écrire :

$$\frac{AM}{AB} = \frac{AN}{AC} = \frac{MN}{BC}$$

</Example>
