La relation de Chasles stipule que :

- [ ] $\vec{AB} + \vec{BA} = \vec{0}$
- [x] $\vec{AB} + \vec{BC} = \vec{AC}$
- [ ] $\vec{AB} - \vec{BC} = \vec{AC}$
- [ ] $\vec{AB} = \vec{BC} + \vec{CA}$

> La relation de Chasles : pour tous points A, B, C, on a $\vec{AB} + \vec{BC} = \vec{AC}$.
