### Une nouvelle unite : le radian

En mathematiques, on utilise souvent une autre unite pour mesurer les angles : le **radian**.

<Definition title="Radian">

Le **radian** (note rad) est l'angle au centre d'un cercle qui intercepte un arc de longueur egale au rayon du cercle.

</Definition>

### Correspondance degres-radians

<Property title="Correspondances fondamentales">

Les correspondances cles entre degres et radians sont :

$$\pi \text{ rad} = 180°$$

$$2\pi \text{ rad} = 360°$$

</Property>

<Remark>

Le nombre $\pi \approx 3.14$ apparait naturellement car le perimetre d'un cercle de rayon $r$ est $2\pi r$. Un tour complet correspond donc a un angle de $2\pi$ radians.

</Remark>

### Formules de conversion

Pour convertir une mesure d'angle d'une unite a l'autre, on utilise les formules suivantes :

<Property title="Formules de conversion">

Soit $\alpha$ la mesure d'un angle.

**De degres vers radians** :

$$\alpha(\text{rad}) = \alpha(°) \times \frac{\pi}{180}$$

**De radians vers degres** :

$$\alpha(°) = \alpha(\text{rad}) \times \frac{180}{\pi}$$

</Property>

### Tableau de conversions courantes

Il est utile de memoriser les conversions suivantes :

| Degres | Radians |
|--------|---------|
| $0°$ | $0$ |
| $30°$ | $\frac{\pi}{6}$ |
| $45°$ | $\frac{\pi}{4}$ |
| $60°$ | $\frac{\pi}{3}$ |
| $90°$ | $\frac{\pi}{2}$ |
| $180°$ | $\pi$ |
| $270°$ | $\frac{3\pi}{2}$ |
| $360°$ | $2\pi$ |

<Remark title="Valeurs a memoriser">

Les angles $30°$, $45°$, $60°$ et $90°$ sont particulierement importants en trigonometrie. Leurs valeurs en radians sont a connaitre par coeur.

</Remark>

### Exemples de conversion

<Example title="De degres vers radians">

Convertir $120°$ en radians.

$$\alpha(\text{rad}) = 120 \times \frac{\pi}{180} = \frac{120\pi}{180} = \frac{2\pi}{3}$$

Donc $120° = \frac{2\pi}{3}$ rad.

</Example>

<Example title="De radians vers degres">

Convertir $\frac{5\pi}{6}$ rad en degres.

$$\alpha(°) = \frac{5\pi}{6} \times \frac{180}{\pi} = \frac{5 \times 180}{6} = \frac{900}{6} = 150°$$

Donc $\frac{5\pi}{6}$ rad $= 150°$.

</Example>

<Attention>

Ne pas confondre $\pi$ (le nombre) avec $\pi$ rad (l'angle de $180°$). Le contexte permet de distinguer les deux.

</Attention>
