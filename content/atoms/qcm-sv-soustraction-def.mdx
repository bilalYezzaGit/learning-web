<Question>
L'expression $\vec{OB} - \vec{OA}$ est egale a :
</Question>

<Option>$\vec{OA} + \vec{OB}$</Option>
<Option correct>$\vec{AB}$</Option>
<Option>$\vec{BA}$</Option>
<Option>$\vec{OB} + \vec{AO}$</Option>

<Explanation>
Par definition, soustraire un vecteur revient a ajouter son oppose :

$$\vec{OB} - \vec{OA} = \vec{OB} + (-\vec{OA}) = \vec{OB} + \vec{AO}$$

On applique ensuite la relation de Chasles :

$$\vec{OB} + \vec{AO} = \vec{AO} + \vec{OB} = \vec{AB}$$

**Regle generale** : Pour tout point $O$, on a toujours $\vec{OB} - \vec{OA} = \vec{AB}$.

**Les autres options sont fausses** :
- $\vec{OA} + \vec{OB}$ ne se simplifie pas par Chasles (les vecteurs ne sont pas consecutifs)
- $\vec{BA} = -\vec{AB}$ est le vecteur oppose du resultat
- $\vec{OB} + \vec{AO}$ est juste une etape intermediaire, pas le resultat final
</Explanation>
