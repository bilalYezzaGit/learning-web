<Question>
La tangente d'un angle aigu dans un triangle rectangle est :
</Question>

<Option>cote adjacent / cote oppose</Option>
<Option correct>cote oppose / cote adjacent</Option>
<Option>hypotenuse / cote oppose</Option>
<Option>cote oppose / hypotenuse</Option>

<Explanation>
La tangente d'un angle aigu $\alpha$ est definie par $\tan(\alpha) = \frac{\text{cote oppose}}{\text{cote adjacent}}$.

Pour retenir facilement : **TOA** (Tangente = Oppose / Adjacent).
</Explanation>
