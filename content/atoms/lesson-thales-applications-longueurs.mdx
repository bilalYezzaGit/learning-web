### Introduction

L'application principale du théorème de Thalès est le **calcul de longueurs** dans une figure géométrique. Cette leçon présente une méthode systématique pour résoudre ce type de problème.

### Methode systematique

<Property title="Methode de calcul avec Thales">

Pour calculer une longueur inconnue en utilisant le théorème de Thalès :

**Étape 1 : Identifier la configuration**

Repérer les droites parallèles et le point d'intersection. Déterminer s'il s'agit d'une configuration triangle ou papillon.

**Étape 2 : Vérifier les conditions**

S'assurer que les conditions du théorème sont remplies :
- Deux droites sécantes
- Deux droites parallèles
- Points alignés dans le bon ordre

**Étape 3 : Écrire les rapports de proportionnalité**

Selon la configuration, écrire les trois rapports égaux donnés par le théorème de Thalès.

**Étape 4 : Choisir les bons rapports**

Sélectionner deux rapports : un qui contient la longueur inconnue, l'autre qui contient uniquement des longueurs connues.

**Étape 5 : Résoudre l'équation**

Utiliser le produit en croix pour isoler l'inconnue et calculer sa valeur.

</Property>

### Application complete : configuration triangle

<Example title="Calcul dans un triangle">

Dans le triangle $DEF$, le point $G$ est sur $[DE]$ et le point $H$ est sur $[DF]$.

La droite $(GH)$ est parallèle au côté $(EF)$.

On donne :
- $DG = 4$ cm
- $GE = 2$ cm
- $DH = 5$ cm

Calculer $HF$.

**Solution :**

**Étape 1 :** Configuration triangle ($(GH) \parallel (EF)$, petit triangle $DGH$ dans grand triangle $DEF$).

**Étape 2 :** Les conditions sont vérifiées :
- $(DE)$ et $(DF)$ sont sécantes en $D$
- $(GH) \parallel (EF)$
- Les points sont dans le bon ordre

**Étape 3 :** Par le théorème de Thalès :

$$\frac{DG}{DE} = \frac{DH}{DF} = \frac{GH}{EF}$$

**Étape 4 :** Calculons d'abord les longueurs totales :

$$DE = DG + GE = 4 + 2 = 6 \text{ cm}$$

On cherche $HF$, donc on pose $DF = DH + HF = 5 + HF$.

Utilisons les deux premiers rapports :

$$\frac{DG}{DE} = \frac{DH}{DF}$$

$$\frac{4}{6} = \frac{5}{5 + HF}$$

**Étape 5 :** Produit en croix :

$$4 \times (5 + HF) = 6 \times 5$$

$$20 + 4 \times HF = 30$$

$$4 \times HF = 30 - 20 = 10$$

$$HF = \frac{10}{4} = 2,5 \text{ cm}$$

</Example>

<Remark>

Dans cet exemple, on a dû **calculer** d'abord la longueur totale $DE$ en additionnant les segments $DG$ et $GE$. Bien organiser les données avant d'écrire les rapports !

</Remark>

### Application complete : configuration papillon

<Example title="Calcul dans une configuration papillon">

Deux droites $(IJ)$ et $(KL)$ se coupent en un point $O$.

Les droites $(IK)$ et $(JL)$ sont parallèles.

On donne :
- $OI = 3$ cm
- $OJ = 4,5$ cm
- $OK = 5$ cm

Calculer $OL$.

**Solution :**

**Étape 1 :** Configuration papillon (les droites se coupent en $O$ qui est entre les parallèles).

**Étape 2 :** Les conditions sont vérifiées :
- $(IJ)$ et $(KL)$ sont sécantes en $O$
- $(IK) \parallel (JL)$

**Étape 3 :** Par le théorème de Thalès (configuration papillon) :

$$\frac{OI}{OJ} = \frac{OK}{OL} = \frac{IK}{JL}$$

**Étape 4 :** On cherche $OL$. Utilisons les deux premiers rapports :

$$\frac{OI}{OJ} = \frac{OK}{OL}$$

$$\frac{3}{4,5} = \frac{5}{OL}$$

**Étape 5 :** Produit en croix :

$$3 \times OL = 4,5 \times 5$$

$$3 \times OL = 22,5$$

$$OL = \frac{22,5}{3} = 7,5 \text{ cm}$$

</Example>

<Attention>

Dans la configuration papillon, tous les segments partent du point $O$. Ne pas essayer d'additionner des segments comme dans la configuration triangle !

</Attention>

### Verification du resultat

<Property title="Verification">

Après avoir calculé une longueur, il est toujours utile de **vérifier** le résultat :

1. **Vérifier l'égalité des rapports** : calculer tous les rapports numériquement et s'assurer qu'ils sont égaux
2. **Vérifier la cohérence** : la longueur trouvée doit être positive et avoir un ordre de grandeur raisonnable par rapport aux autres longueurs

</Property>

<Example title="Verification">

Reprenons l'exemple papillon précédent. On a trouvé $OL = 7,5$ cm.

Vérifions l'égalité des rapports :

$$\frac{OI}{OJ} = \frac{3}{4,5} = \frac{2}{3}$$

$$\frac{OK}{OL} = \frac{5}{7,5} = \frac{5}{7,5} = \frac{10}{15} = \frac{2}{3}$$

Les deux rapports sont bien égaux à $\frac{2}{3}$. Le résultat est correct !

</Example>

### Cas avec des fractions

<Example title="Calcul avec fractions">

Dans un triangle $ABC$, $(MN) \parallel (BC)$ avec $M \in [AB]$ et $N \in [AC]$.

On donne : $\frac{AM}{AB} = \frac{2}{5}$ et $BC = 15$ cm.

Calculer $MN$.

**Solution :**

Par Thalès :

$$\frac{AM}{AB} = \frac{MN}{BC}$$

$$\frac{2}{5} = \frac{MN}{15}$$

$$5 \times MN = 2 \times 15$$

$$5 \times MN = 30$$

$$MN = \frac{30}{5} = 6 \text{ cm}$$

</Example>

<Remark>

Quand un rapport est donné sous forme de fraction, on peut l'utiliser **directement** dans l'égalité de Thalès. Cela simplifie souvent les calculs.

</Remark>
