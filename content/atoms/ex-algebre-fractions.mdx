<Enonce>
Simplifier les fractions algebriques suivantes et preciser les conditions d'existence :

1. $\dfrac{x^2-9}{x+3}$

2. $\dfrac{2x^2+4x}{2x}$

3. $\dfrac{x^2-4x+4}{x^2-4}$
</Enonce>

<Solution>
**1.** Simplification de $\dfrac{x^2-9}{x+3}$ :

**Condition d'existence** : $x+3 \neq 0 \Rightarrow x \neq -3$

Factorisation du numerateur : $x^2 - 9 = (x-3)(x+3)$

$$\dfrac{x^2-9}{x+3} = \dfrac{(x-3)(x+3)}{x+3} = x-3 \quad \text{pour } x \neq -3$$

**2.** Simplification de $\dfrac{2x^2+4x}{2x}$ :

**Condition d'existence** : $2x \neq 0 \Rightarrow x \neq 0$

Factorisation du numerateur : $2x^2 + 4x = 2x(x+2)$

$$\dfrac{2x^2+4x}{2x} = \dfrac{2x(x+2)}{2x} = x+2 \quad \text{pour } x \neq 0$$

**3.** Simplification de $\dfrac{x^2-4x+4}{x^2-4}$ :

**Conditions d'existence** : $x^2-4 \neq 0 \Rightarrow (x-2)(x+2) \neq 0 \Rightarrow x \neq 2$ et $x \neq -2$

Factorisation :
- Numerateur : $x^2 - 4x + 4 = (x-2)^2$
- Denominateur : $x^2 - 4 = (x-2)(x+2)$

$$\dfrac{x^2-4x+4}{x^2-4} = \dfrac{(x-2)^2}{(x-2)(x+2)} = \dfrac{x-2}{x+2} \quad \text{pour } x \neq 2 \text{ et } x \neq -2$$
</Solution>

<Methode>
Pour simplifier une fraction algebrique :

1. **Determiner les conditions d'existence** : denominateur $\neq 0$
2. **Factoriser** le numerateur et le denominateur
3. **Simplifier** les facteurs communs
4. **Preciser** les valeurs interdites dans la reponse finale
</Methode>

<Erreurs>
**Erreur frequente** : Oublier de preciser les conditions d'existence.

Exemple : $\dfrac{x^2-9}{x+3} = x-3$ est **incomplet** car on ne precise pas que $x \neq -3$.

Meme si la forme simplifiee $x-3$ existe pour $x = -3$, la fraction initiale n'existe pas en ce point.
</Erreurs>
