<Enonce>
Etudier le signe des quotients suivants et resoudre les inequations :

a) $\frac{2x - 6}{x + 1} \geq 0$

b) $\frac{-x + 4}{3x - 9} < 0$
</Enonce>

<Solution>
**a)** $\frac{2x - 6}{x + 1} \geq 0$

**Etape 1** : Etudier le signe du numerateur et du denominateur.

Numerateur : $2x - 6 = 0 \Rightarrow x = 3$
- Si $x < 3$ : $2x - 6 < 0$ (negatif)
- Si $x > 3$ : $2x - 6 > 0$ (positif)

Denominateur : $x + 1 = 0 \Rightarrow x = -1$ (valeur interdite)
- Si $x < -1$ : $x + 1 < 0$ (negatif)
- Si $x > -1$ : $x + 1 > 0$ (positif)

**Etape 2** : Tableau de signes.

| $x$ | $-\infty$ |  | $-1$ |  | $3$ |  | $+\infty$ |
|-----|-----------|--|------|--|-----|--|-----------|
| $2x-6$ | $-$ | | $-$ | | $0$ | $+$ | $+$ |
| $x+1$ | $-$ | | $\|$ | $+$ | $+$ | $+$ | $+$ |
| Quotient | $+$ | | $\|$ | $-$ | $0$ | $+$ | $+$ |

**Etape 3** : Lecture du tableau.

On cherche les valeurs ou le quotient est positif ou nul ($\geq 0$).

Le quotient est positif ou nul pour $x \in ]-\infty, -1[ \cup [3, +\infty[$.

**Attention** : $x = -1$ est exclu car le denominateur s'annule.

L'ensemble solution est $S = ]-\infty, -1[ \cup [3, +\infty[$.

---

**b)** $\frac{-x + 4}{3x - 9} < 0$

**Etape 1** : Etudier le signe du numerateur et du denominateur.

Numerateur : $-x + 4 = 0 \Rightarrow x = 4$
- Si $x < 4$ : $-x + 4 > 0$ (positif)
- Si $x > 4$ : $-x + 4 < 0$ (negatif)

Denominateur : $3x - 9 = 0 \Rightarrow x = 3$ (valeur interdite)
- Si $x < 3$ : $3x - 9 < 0$ (negatif)
- Si $x > 3$ : $3x - 9 > 0$ (positif)

**Etape 2** : Tableau de signes.

| $x$ | $-\infty$ |  | $3$ |  | $4$ |  | $+\infty$ |
|-----|-----------|--|-----|--|-----|--|-----------|
| $-x+4$ | $+$ | | $+$ | | $0$ | $-$ | $-$ |
| $3x-9$ | $-$ | | $\|$ | $+$ | $+$ | $+$ | $+$ |
| Quotient | $-$ | | $\|$ | $+$ | $0$ | $-$ | $-$ |

**Etape 3** : Lecture du tableau.

On cherche les valeurs ou le quotient est strictement negatif ($< 0$).

Le quotient est negatif pour $x \in ]-\infty, 3[ \cup ]4, +\infty[$.

L'ensemble solution est $S = ]-\infty, 3[ \cup ]4, +\infty[$.
</Solution>

<Methode>
Pour etudier le signe d'un quotient $\frac{A(x)}{B(x)}$ :

1. **Determiner les valeurs interdites** : resoudre $B(x) = 0$.

2. **Etudier le signe du numerateur** $A(x)$ : trouver ses racines et son signe.

3. **Etudier le signe du denominateur** $B(x)$ : trouver ses racines et son signe.

4. **Construire un tableau de signes** avec trois lignes : numerateur, denominateur, quotient.

5. **Appliquer la regle des signes** :
   - $\frac{+}{+} = +$ et $\frac{-}{-} = +$
   - $\frac{+}{-} = -$ et $\frac{-}{+} = -$

6. **Lire la solution** sur la derniere ligne selon l'inequation demandee.

7. **Attention aux valeurs interdites** : toujours exclues des solutions.
</Methode>
