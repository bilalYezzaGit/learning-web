<Enonce>
Calculer et simplifier les expressions suivantes :

a) $3\sqrt{2} + 5\sqrt{2} - \sqrt{2}$

b) $\sqrt{12} + \sqrt{27} - \sqrt{48}$

c) $\sqrt{3} \times \sqrt{12}$

d) $(2\sqrt{5})^2$
</Enonce>

<Solution>
**a)** Les termes ont le même radical, on peut les additionner directement :

$$3\sqrt{2} + 5\sqrt{2} - \sqrt{2} = (3 + 5 - 1)\sqrt{2} = 7\sqrt{2}$$

**b)** On commence par simplifier chaque racine :

$$\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$$

$$\sqrt{27} = \sqrt{9 \times 3} = 3\sqrt{3}$$

$$\sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3}$$

Ensuite on additionne :

$$2\sqrt{3} + 3\sqrt{3} - 4\sqrt{3} = (2 + 3 - 4)\sqrt{3} = \sqrt{3}$$

**c)** On utilise la propriété $\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$ :

$$\sqrt{3} \times \sqrt{12} = \sqrt{3 \times 12} = \sqrt{36} = 6$$

**d)** On développe le carré :

$$(2\sqrt{5})^2 = 2^2 \times (\sqrt{5})^2 = 4 \times 5 = 20$$
</Solution>

<Methode>
Pour effectuer des opérations avec les racines :

1. **Addition/Soustraction** : simplifier d'abord chaque racine, puis regrouper les termes de même radical : $a\sqrt{c} + b\sqrt{c} = (a+b)\sqrt{c}$

2. **Multiplication** : utiliser $\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$

3. **Carrés** : $(\sqrt{a})^2 = a$ et $(k\sqrt{a})^2 = k^2 \times a$
</Methode>

<Hint>
Pour la question b), commencez par simplifier chaque racine en extrayant les carrés parfaits.
</Hint>
