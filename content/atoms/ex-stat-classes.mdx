<Enonce>
Les tailles (en cm) de 40 eleves d'un lycee ont ete groupees en classes :

| Taille (en cm) | Effectif |
|----------------|----------|
| $[140, 150[$ | 4 |
| $[150, 160[$ | 10 |
| $[160, 170[$ | 15 |
| $[170, 180[$ | 8 |
| $[180, 190[$ | 3 |

1. Calculer les **centres de classe**.
2. Calculer la **taille moyenne** des eleves.
3. Determiner la **classe mediane**.
</Enonce>

<Solution>
**1. Centres de classe**

Le centre d'une classe $[a, b[$ est :

$$c = \frac{a + b}{2}$$

| Classe | Centre |
|--------|--------|
| $[140, 150[$ | $\frac{140 + 150}{2} = 145$ |
| $[150, 160[$ | $\frac{150 + 160}{2} = 155$ |
| $[160, 170[$ | $\frac{160 + 170}{2} = 165$ |
| $[170, 180[$ | $\frac{170 + 180}{2} = 175$ |
| $[180, 190[$ | $\frac{180 + 190}{2} = 185$ |

**2. Taille moyenne**

On utilise les centres de classe comme valeurs representatives :

$$\bar{x} = \frac{\sum (\text{centre}_i \times n_i)}{N}$$

$$\bar{x} = \frac{4 \times 145 + 10 \times 155 + 15 \times 165 + 8 \times 175 + 3 \times 185}{40}$$

$$\bar{x} = \frac{580 + 1550 + 2475 + 1400 + 555}{40}$$

$$\bar{x} = \frac{6560}{40} = 164 \text{ cm}$$

La taille moyenne est **164 cm**.

**3. Classe mediane**

Construisons le tableau des effectifs cumules croissants :

| Classe | Effectif | Effectif cumule |
|--------|----------|-----------------|
| $[140, 150[$ | 4 | 4 |
| $[150, 160[$ | 10 | 14 |
| $[160, 170[$ | 15 | 29 |
| $[170, 180[$ | 8 | 37 |
| $[180, 190[$ | 3 | 40 |

La mediane correspond a la valeur de rang $\frac{N}{2} = \frac{40}{2} = 20$.

L'effectif cumule atteint 29 dans la classe $[160, 170[$, qui contient donc la 20e valeur.

**La classe mediane est $[160, 170[$**.
</Solution>

<Methode>
**Donnees groupees en classes** :

Pour calculer une moyenne avec des classes :

1. **Calculer les centres** de chaque classe : $c = \frac{\text{borne inf} + \text{borne sup}}{2}$
2. **Utiliser les centres** comme valeurs representatives
3. **Appliquer** la formule de la moyenne ponderee

Pour trouver la classe mediane :

1. **Calculer** les effectifs cumules croissants
2. **Identifier** le rang median : $\frac{N}{2}$ (si $N$ pair) ou $\frac{N+1}{2}$ (si $N$ impair)
3. **Trouver** la classe dont l'effectif cumule depasse ou atteint ce rang
</Methode>

<Hint>
Pour la classe mediane, commencez par construire le tableau des effectifs cumules croissants. La classe mediane est celle qui contient la valeur de rang $\frac{40}{2} = 20$.
</Hint>
