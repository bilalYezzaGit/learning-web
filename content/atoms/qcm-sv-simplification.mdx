<Question>
Que vaut $\vec{AB} + \vec{CD} + \vec{BA} + \vec{DC}$ ?
</Question>

<Option>$2\vec{AB}$</Option>
<Option>$\vec{AD}$</Option>
<Option correct>$\vec{0}$</Option>
<Option>$2\vec{AC}$</Option>

<Explanation>
On regroupe les vecteurs opposes en utilisant la commutativite :

$$\vec{AB} + \vec{CD} + \vec{BA} + \vec{DC} = \vec{AB} + \vec{BA} + \vec{CD} + \vec{DC}$$

Or :
- $\vec{AB} + \vec{BA} = \vec{0}$ (vecteurs opposes)
- $\vec{CD} + \vec{DC} = \vec{0}$ (vecteurs opposes)

Donc :

$$\vec{AB} + \vec{BA} + \vec{CD} + \vec{DC} = \vec{0} + \vec{0} = \vec{0}$$

**Methode generale** : Pour simplifier une somme de vecteurs, reperer les paires de vecteurs opposes ($\vec{XY}$ et $\vec{YX}$), qui s'annulent.

**Les autres options sont fausses** :
- $2\vec{AB}$ serait obtenu si on avait $\vec{AB} + \vec{AB}$
- $\vec{AD}$ ne correspond a aucune combinaison des vecteurs donnes
- $2\vec{AC}$ ne peut pas s'obtenir avec les vecteurs donnes
</Explanation>
