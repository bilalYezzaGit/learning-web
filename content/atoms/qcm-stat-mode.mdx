<Question>
Le mode de la serie $\{3, 5, 7, 5, 3, 5, 8\}$ est :
</Question>

<Option>$3$</Option>
<Option correct>$5$</Option>
<Option>$7$</Option>
<Option>$8$</Option>

<Explanation>
Le **mode** (ou valeur modale) est la valeur qui apparait le plus souvent dans la serie.

Comptage des effectifs :
- $3$ apparait **2 fois**
- $5$ apparait **3 fois**
- $7$ apparait **1 fois**
- $8$ apparait **1 fois**

Le mode est donc $5$ (effectif maximal de 3).

**Remarque** : Une serie peut avoir plusieurs modes (bimodale, multimodale) ou aucun mode si toutes les valeurs ont le meme effectif.
</Explanation>
