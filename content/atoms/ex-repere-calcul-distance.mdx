<Enonce>
Dans un repere orthonorme $(O, \vec{i}, \vec{j})$, on considere les points :
- $A(1; 2)$
- $B(4; 6)$
- $C(-2; 5)$

1. Calculer les distances $AB$, $AC$ et $BC$
2. Le triangle $ABC$ est-il isocele ?
3. Le triangle $ABC$ est-il rectangle ?
</Enonce>

<Solution>
**1. Calcul des distances**

La formule de la distance entre deux points $M(x_M; y_M)$ et $N(x_N; y_N)$ est :

$$MN = \sqrt{(x_N - x_M)^2 + (y_N - y_M)^2}$$

Pour $AB$ avec $A(1; 2)$ et $B(4; 6)$ :

$$AB = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

Pour $AC$ avec $A(1; 2)$ et $C(-2; 5)$ :

$$AC = \sqrt{(-2-1)^2 + (5-2)^2} = \sqrt{(-3)^2 + 3^2} = \sqrt{9 + 9} = \sqrt{18} = 3\sqrt{2}$$

Pour $BC$ avec $B(4; 6)$ et $C(-2; 5)$ :

$$BC = \sqrt{(-2-4)^2 + (5-6)^2} = \sqrt{(-6)^2 + (-1)^2} = \sqrt{36 + 1} = \sqrt{37}$$

**2. Triangle isocele ?**

Un triangle est isocele si deux de ses cotes ont la meme longueur.

On a $AB = 5$, $AC = 3\sqrt{2} \approx 4{,}24$ et $BC = \sqrt{37} \approx 6{,}08$.

Les trois longueurs sont differentes, donc **le triangle $ABC$ n'est pas isocele**.

**3. Triangle rectangle ?**

Un triangle est rectangle si le theoreme de Pythagore est verifie : $a^2 + b^2 = c^2$ pour le cote le plus long.

Le cote le plus long est $BC = \sqrt{37}$. Verifions si $AB^2 + AC^2 = BC^2$ :

$$AB^2 + AC^2 = 5^2 + (3\sqrt{2})^2 = 25 + 9 \times 2 = 25 + 18 = 43$$

$$BC^2 = (\sqrt{37})^2 = 37$$

On a $43 \neq 37$, donc **le triangle $ABC$ n'est pas rectangle**.
</Solution>

<Methode>
**Distance entre deux points** $A(x_A; y_A)$ et $B(x_B; y_B)$ dans un repere orthonorme :

$$AB = \sqrt{(x_B - x_A)^2 + (y_B - y_A)^2}$$

Cette formule vient du theoreme de Pythagore applique au triangle rectangle forme par les projections.

**Nature d'un triangle** :
- **Isocele** : deux cotes egaux
- **Equilateral** : trois cotes egaux
- **Rectangle** : theoreme de Pythagore verifie
</Methode>

<Hint>
Pour verifier si un triangle est rectangle, calculez d'abord les carres des distances (cela evite de manipuler des racines carrees).
</Hint>

<Erreurs>
- Oublier d'elever au carre les differences : $(4-1)^2 = 9$ et non $4-1 = 3$
- Faire des erreurs de signe : $(-3)^2 = 9$ (toujours positif)
- Comparer les distances au lieu des carres pour le theoreme de Pythagore
- Oublier de simplifier les racines : $\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$
</Erreurs>
