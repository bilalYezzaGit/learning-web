<Enonce>
Factoriser les expressions suivantes :

1. $6x^2 + 9x$
2. $4x(x-1) - 3(x-1)$
3. $x^2 - 5x$
4. $2x(3x+1) + (3x+1)$
</Enonce>

<Solution>
**1.** Factorisation de $6x^2 + 9x$ :

Le facteur commun est $3x$ :

$$6x^2 + 9x = 3x(2x + 3)$$

**2.** Factorisation de $4x(x-1) - 3(x-1)$ :

Le facteur commun est $(x-1)$ :

$$4x(x-1) - 3(x-1) = (x-1)(4x - 3)$$

**3.** Factorisation de $x^2 - 5x$ :

Le facteur commun est $x$ :

$$x^2 - 5x = x(x - 5)$$

**4.** Factorisation de $2x(3x+1) + (3x+1)$ :

Le facteur commun est $(3x+1)$ :

$$2x(3x+1) + (3x+1) = (3x+1)(2x + 1)$$
</Solution>

<Methode>
Pour factoriser par facteur commun :

1. **Identifier** le facteur commun a tous les termes
2. **Mettre en facteur** ce terme commun
3. **Verifier** en developpant le resultat
</Methode>
