<Question>
$\sqrt{9 + 16}$ est egal a :
</Question>

<Option>$\sqrt{9} + \sqrt{16} = 7$</Option>
<Option correct>$5$</Option>
<Option>$\sqrt{25} = 12{,}5$</Option>
<Option>$25$</Option>

<Explanation>
**PIEGE** : $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$ !

Il faut d'abord **calculer la somme sous la racine** : $\sqrt{9 + 16} = \sqrt{25} = 5$.

Verifier : $\sqrt{9} + \sqrt{16} = 3 + 4 = 7 \neq 5$.
</Explanation>
