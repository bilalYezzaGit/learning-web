<Question>
Le domaine de definition de $f(x) = \frac{1}{x^2 - 9}$ est :
</Question>

<Option>$\mathbb{R}$</Option>
<Option>$\mathbb{R} \setminus \{9\}$</Option>
<Option correct>$\mathbb{R} \setminus \{-3 ; 3\}$</Option>
<Option>$\mathbb{R} \setminus \{3\}$</Option>

<Explanation>
Une fraction n'est pas definie lorsque le denominateur est nul. On resout $x^2 - 9 = 0$, ce qui donne $(x-3)(x+3) = 0$, donc $x = 3$ ou $x = -3$. Le domaine de definition est donc $\mathbb{R} \setminus \{-3 ; 3\}$.
</Explanation>
