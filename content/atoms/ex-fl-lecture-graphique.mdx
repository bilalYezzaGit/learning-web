<Enonce>
Dans un repere orthonorme, une droite passe par l'origine $O$ et par le point $A(3, -6)$.

1. Justifier que cette droite est la representation graphique d'une fonction lineaire.
2. Determiner le coefficient de cette fonction.
3. Donner l'expression de $f(x)$.
4. La droite passe egalement par un point $B(-2, b)$. Determiner la valeur de $b$.
</Enonce>

<Solution>
**1.** Justification :

La droite passe par l'origine $O(0, 0)$. Or, le graphe d'une fonction lineaire est caracterise par le fait qu'il passe par l'origine.

Donc cette droite est bien la representation graphique d'une fonction lineaire.

**2.** Determination du coefficient :

Le point $A(3, -6)$ appartient au graphe, donc $f(3) = -6$.

Pour une fonction lineaire $f(x) = ax$, on a :

$$a = \frac{f(3)}{3} = \frac{-6}{3} = -2$$

Le coefficient est $a = -2$.

**3.** Expression de $f$ :

$$f(x) = -2x$$

**4.** Calcul de $b$ :

Le point $B(-2, b)$ appartient au graphe de $f$, donc :

$$b = f(-2) = -2 \times (-2) = 4$$

La valeur de $b$ est $4$, donc $B(-2, 4)$.
</Solution>

<Methode>
Pour determiner une fonction lineaire a partir de sa representation graphique :

1. **Verifier que le graphe passe par l'origine** $O(0, 0)$
2. **Identifier un point** $M(x_0, y_0)$ du graphe avec $x_0 \neq 0$
3. **Calculer le coefficient** : $a = \frac{y_0}{x_0}$
4. **Ecrire la fonction** : $f(x) = ax$

**Reciproque** : si une droite passe par l'origine, c'est le graphe d'une fonction lineaire.
</Methode>

<Hint>
Utilisez le fait que $A(3, -6)$ appartient au graphe, donc $f(3) = -6$.
</Hint>

<Erreurs>
- Confondre fonction lineaire et fonction affine : seules les fonctions lineaires ont un graphe passant par l'origine
- Oublier de verifier que la droite passe par $O$ avant de conclure qu'elle represente une fonction lineaire
- Se tromper de signe dans le calcul de $a$ (attention aux signes negatifs)
</Erreurs>
