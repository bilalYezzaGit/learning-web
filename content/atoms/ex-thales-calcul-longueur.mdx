<Enonce>
Dans un triangle $ABC$, la droite $(MN)$ est parallèle à $(BC)$ avec $M$ sur $[AB]$ et $N$ sur $[AC]$.

On donne : $AM = 3$ cm, $AB = 9$ cm, $AN = 2$ cm, $AC = 6$ cm.

1. Vérifier que les rapports $\frac{AM}{AB}$ et $\frac{AN}{AC}$ sont égaux.
2. Calculer $MN$ sachant que $BC = 12$ cm.
</Enonce>

<Solution>
**1.** Calculons les deux rapports :

$$\frac{AM}{AB} = \frac{3}{9} = \frac{1}{3}$$

$$\frac{AN}{AC} = \frac{2}{6} = \frac{1}{3}$$

Les deux rapports sont égaux : $\frac{AM}{AB} = \frac{AN}{AC} = \frac{1}{3}$ ✓

**2.** Puisque $(MN) \parallel (BC)$, on peut appliquer le théorème de Thalès dans le triangle $ABC$ :

$$\frac{AM}{AB} = \frac{AN}{AC} = \frac{MN}{BC}$$

On sait que $\frac{AM}{AB} = \frac{1}{3}$ et $BC = 12$ cm.

Donc :

$$\frac{MN}{BC} = \frac{1}{3}$$

$$MN = BC \times \frac{1}{3} = 12 \times \frac{1}{3} = 4 \text{ cm}$$

**Réponse** : $MN = 4$ cm.
</Solution>

<Methode>
Pour calculer une longueur avec le théorème de Thalès :

1. Vérifier que les droites sont parallèles et identifier la configuration de Thalès
2. Écrire l'égalité des rapports : $\frac{AM}{AB} = \frac{AN}{AC} = \frac{MN}{BC}$
3. Identifier la longueur inconnue et le rapport connu
4. Résoudre l'équation : $MN = BC \times \frac{AM}{AB}$
</Methode>

<Hint>
Commencez par calculer le rapport $\frac{AM}{AB}$, puis utilisez l'égalité $\frac{MN}{BC} = \frac{AM}{AB}$ pour trouver $MN$.
</Hint>
