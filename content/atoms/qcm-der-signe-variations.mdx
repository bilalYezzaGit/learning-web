<Question>
Si $f'(x) < 0$ pour tout $x \in ]a, b[$, alors $f$ est :
</Question>

<Option>Positive sur $]a, b[$</Option>
<Option>Negative sur $]a, b[$</Option>
<Option>Croissante sur $[a, b]$</Option>
<Option correct>Decroissante sur $[a, b]$</Option>

<Explanation>
Le signe de $f'$ indique le **sens de variation**, pas le signe de $f$.

- Si $f'(x) > 0$, alors $f$ est **croissante**
- Si $f'(x) < 0$, alors $f$ est **decroissante**
- Si $f'(x) = 0$, alors $f$ est **constante**

Ici $f'(x) < 0$ donc $f$ est decroissante sur $[a, b]$. Attention : $f'<0$ ne donne aucune information sur le signe de $f$ elle-meme.
</Explanation>
