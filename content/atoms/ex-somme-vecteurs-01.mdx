<Enonce>
Soit $ABCD$ un parallelogramme.

1. Simplifier $\vec{AB} + \vec{BC}$.
2. Simplifier $\vec{AB} + \vec{AD}$.
3. Simplifier $\vec{AB} + \vec{BC} + \vec{CD}$.
4. Montrer que $\vec{AB} + \vec{AD} + \vec{CB} + \vec{CD} = \vec{0}$.
</Enonce>

<Solution>
**1)** D'apres la relation de Chasles :

$\vec{AB} + \vec{BC} = \vec{AC}$

**2)** D'apres la regle du parallelogramme :

$\vec{AB} + \vec{AD} = \vec{AC}$

**3)** On applique la relation de Chasles deux fois :

$\vec{AB} + \vec{BC} + \vec{CD} = \vec{AC} + \vec{CD} = \vec{AD}$

**4)** On regroupe les vecteurs :

$\vec{AB} + \vec{AD} + \vec{CB} + \vec{CD}$

$= (\vec{AB} + \vec{CB}) + (\vec{AD} + \vec{CD})$

Or $\vec{CB} = -\vec{BC}$ et $\vec{CD} = -\vec{DC}$.

Dans le parallelogramme : $\vec{AB} = \vec{DC}$ donc $\vec{AB} + \vec{CD} = \vec{AB} - \vec{DC} = \vec{0}$

De meme : $\vec{AD} + \vec{CB} = \vec{AD} - \vec{BC} = \vec{0}$ car $\vec{AD} = \vec{BC}$.

Donc $\vec{AB} + \vec{AD} + \vec{CB} + \vec{CD} = \vec{0}$
</Solution>
