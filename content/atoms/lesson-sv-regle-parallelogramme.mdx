### La regle du parallelogramme

<Property title="Regle du parallelogramme">

Si $ABCD$ est un parallelogramme (dans cet ordre), alors :

$$\vec{AB} + \vec{AD} = \vec{AC}$$

La **diagonale** $\vec{AC}$ represente la somme des deux cotes consecutifs $\vec{AB}$ et $\vec{AD}$.

</Property>

### Construction graphique de la somme

Pour construire graphiquement $\vec{u} + \vec{v}$ :

1. Choisir un point $A$ comme origine commune
2. Placer le vecteur $\vec{AB} = \vec{u}$ a partir de $A$
3. Placer le vecteur $\vec{AD} = \vec{v}$ a partir de $A$
4. Completer le parallelogramme $ABCD$
5. La somme est $\vec{u} + \vec{v} = \vec{AC}$ (la diagonale)

<Remark>

Cette methode est appelee **methode du parallelogramme** car on place les deux vecteurs a la **meme origine**, contrairement a la methode de Chasles ou on les met bout a bout.

</Remark>

### Lien avec la relation de Chasles

Les deux methodes donnent le meme resultat :

<Example title="Equivalence des deux methodes">

Dans un parallelogramme $ABCD$, montrer que :

$$\vec{AB} + \vec{AD} = \vec{AB} + \vec{BC}$$

**Solution** :

- Methode du parallelogramme : $\vec{AB} + \vec{AD} = \vec{AC}$
- Methode de Chasles : $\vec{AB} + \vec{BC} = \vec{AC}$

Comme $ABCD$ est un parallelogramme, $\vec{AD} = \vec{BC}$, donc les deux expressions sont egales.

</Example>

### Propriete reciproque

<Property title="Reciproque">

Si $\vec{AB} + \vec{AD} = \vec{AC}$, alors $ABCD$ est un parallelogramme.

</Property>

Cette propriete permet de **demontrer qu'un quadrilatere est un parallelogramme** en verifiant une egalite vectorielle.

<Attention>

Dans la regle du parallelogramme, les quatre points doivent etre **dans l'ordre** : $ABCD$ signifie que $A$ et $C$ sont les sommets opposes, et $B$ et $D$ aussi.

</Attention>
