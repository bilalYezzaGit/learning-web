<Question>
Si $2 \leq x \leq 8$, alors $\frac{1}{x}$ verifie :
</Question>

<Option>$\frac{1}{2} \leq \frac{1}{x} \leq \frac{1}{8}$</Option>
<Option correct>$\frac{1}{8} \leq \frac{1}{x} \leq \frac{1}{2}$</Option>
<Option>$2 \leq \frac{1}{x} \leq 8$</Option>
<Option>$\frac{1}{8} \leq \frac{1}{x} \leq 8$</Option>

<Explanation>
ATTENTION : La fonction inverse est decroissante pour les nombres positifs. Donc elle inverse l'ordre des inegalites !

Plus $x$ est grand, plus $\frac{1}{x}$ est petit.

Quand $x = 2$ (minimum), $\frac{1}{x} = \frac{1}{2}$ (maximum).

Quand $x = 8$ (maximum), $\frac{1}{x} = \frac{1}{8}$ (minimum).

Donc $\frac{1}{8} \leq \frac{1}{x} \leq \frac{1}{2}$.

PIEGE CLASSIQUE : Les bornes sont inversees par rapport a l'encadrement de $x$ !
</Explanation>
