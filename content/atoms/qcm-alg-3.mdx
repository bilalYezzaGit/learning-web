L'expression $(a - b)(a + b)$ est egale a :

- [ ] $a^2 + b^2$
- [x] $a^2 - b^2$
- [ ] $a^2 - 2ab + b^2$
- [ ] $2a^2 - 2b^2$

> $(a - b)(a + b) = a^2 - b^2$ est l'identite de la difference de carres.
