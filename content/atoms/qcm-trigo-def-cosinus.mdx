<Question>
Dans un triangle rectangle, le cosinus d'un angle aigu est egal a :
</Question>

<Option>cote oppose / hypotenuse</Option>
<Option>hypotenuse / cote adjacent</Option>
<Option correct>cote adjacent / hypotenuse</Option>
<Option>cote oppose / cote adjacent</Option>

<Explanation>
Le cosinus d'un angle aigu $\alpha$ est defini par $\cos(\alpha) = \frac{\text{cote adjacent}}{\text{hypotenuse}}$.

Pour retenir facilement : **CAH** (Cosinus = Adjacent / Hypotenuse).
</Explanation>
