<Enonce>
Factoriser puis resoudre les equations suivantes :

a) $x^2 - 9 = 0$

b) $4x^2 - 12x + 9 = 0$

c) $(2x+1)^2 - (x-3)^2 = 0$

d) $x^2 - 5x = 0$
</Enonce>

<Solution>
**a)** $x^2 - 9 = 0$

On reconnait une difference de carres : $a^2 - b^2 = (a-b)(a+b)$

$$x^2 - 9 = x^2 - 3^2 = (x-3)(x+3) = 0$$

$$(x-3)(x+3) = 0$$

$$x = 3 \quad \text{ou} \quad x = -3$$

L'ensemble solution est $S = \{-3 ; 3\}$.

**b)** $4x^2 - 12x + 9 = 0$

On reconnait une identite remarquable : $(a-b)^2 = a^2 - 2ab + b^2$

Ici, $4x^2 = (2x)^2$, $9 = 3^2$ et $12x = 2 \times 2x \times 3$ :

$$4x^2 - 12x + 9 = (2x - 3)^2 = 0$$

$$(2x - 3)^2 = 0$$

$$2x - 3 = 0$$

$$x = \frac{3}{2}$$

L'ensemble solution est $S = \left\{\frac{3}{2}\right\}$ (racine double).

**c)** $(2x+1)^2 - (x-3)^2 = 0$

On reconnait une difference de carres : $A^2 - B^2 = (A-B)(A+B)$

Avec $A = 2x+1$ et $B = x-3$ :

$$(2x+1 - (x-3))(2x+1 + (x-3)) = 0$$

$$(2x+1 - x+3)(2x+1 + x-3) = 0$$

$$(x+4)(3x-2) = 0$$

$$x + 4 = 0 \quad \text{ou} \quad 3x - 2 = 0$$

$$x = -4 \quad \text{ou} \quad x = \frac{2}{3}$$

L'ensemble solution est $S = \left\{-4 ; \frac{2}{3}\right\}$.

**d)** $x^2 - 5x = 0$

On factorise par $x$ :

$$x(x - 5) = 0$$

$$x = 0 \quad \text{ou} \quad x - 5 = 0$$

$$x = 0 \quad \text{ou} \quad x = 5$$

L'ensemble solution est $S = \{0 ; 5\}$.
</Solution>

<Methode>
Pour resoudre une equation par factorisation :

1. **Reconnaitre les identites remarquables** :
   - $a^2 - b^2 = (a-b)(a+b)$ (difference de carres)
   - $a^2 + 2ab + b^2 = (a+b)^2$ (carre d'une somme)
   - $a^2 - 2ab + b^2 = (a-b)^2$ (carre d'une difference)

2. **Factoriser par un facteur commun** si possible (ex : $ax + bx = x(a+b)$).

3. **Ecrire sous forme de produit nul** : $A \times B = 0$.

4. **Appliquer la regle** : un produit est nul si et seulement si l'un des facteurs est nul.

5. **Resoudre chaque equation simple** obtenue.
</Methode>
