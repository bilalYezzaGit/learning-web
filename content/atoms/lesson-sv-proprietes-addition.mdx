### Les quatre proprietes fondamentales

L'addition vectorielle possede les memes proprietes que l'addition des nombres reels.

<Property title="Commutativite">

Pour tous vecteurs $\vec{u}$ et $\vec{v}$ :

$$\vec{u} + \vec{v} = \vec{v} + \vec{u}$$

L'ordre d'addition n'a pas d'importance.

</Property>

**Demonstration geometrique** : Dans un parallelogramme $ABCD$, on a $\vec{AB} + \vec{AD} = \vec{AC}$ et aussi $\vec{AD} + \vec{AB} = \vec{AC}$.

<Property title="Associativite">

Pour tous vecteurs $\vec{u}$, $\vec{v}$ et $\vec{w}$ :

$$(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})$$

On peut donc ecrire $\vec{u} + \vec{v} + \vec{w}$ sans parentheses.

</Property>

**Demonstration** : Par la relation de Chasles appliquee a quatre points $A$, $B$, $C$, $D$ :

$$\vec{AB} + \vec{BC} + \vec{CD} = \vec{AD}$$

qu'on peut grouper : $(\vec{AB} + \vec{BC}) + \vec{CD} = \vec{AC} + \vec{CD} = \vec{AD}$ ou $\vec{AB} + (\vec{BC} + \vec{CD}) = \vec{AB} + \vec{BD} = \vec{AD}$.

<Property title="Element neutre">

Le vecteur nul $\vec{0}$ est l'element neutre de l'addition :

$$\vec{u} + \vec{0} = \vec{0} + \vec{u} = \vec{u}$$

</Property>

**Interpretation** : Ajouter un deplacement nul ne change pas le vecteur.

<Property title="Vecteur oppose">

Pour tout vecteur $\vec{u}$, il existe un vecteur oppose $-\vec{u}$ tel que :

$$\vec{u} + (-\vec{u}) = \vec{0}$$

</Property>

**Rappel** : Si $\vec{u} = \vec{AB}$, alors $-\vec{u} = \vec{BA}$.

### Consequences pratiques

Ces quatre proprietes permettent de manipuler les sommes de vecteurs comme des sommes algebriques :

<Example title="Simplification d'une expression">

Simplifier : $\vec{u} + \vec{v} + (-\vec{u}) + \vec{w}$.

**Solution** :

En utilisant la commutativite et l'associativite :

$$\vec{u} + \vec{v} + (-\vec{u}) + \vec{w} = \vec{u} + (-\vec{u}) + \vec{v} + \vec{w} = \vec{0} + \vec{v} + \vec{w} = \vec{v} + \vec{w}$$

</Example>

<Remark>

Grace a ces proprietes, on peut **simplifier** les expressions vectorielles en regroupant les termes, en changeant l'ordre, et en eliminant les paires de vecteurs opposes.

</Remark>
