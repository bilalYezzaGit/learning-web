<Enonce>
Soit $f$ une fonction continue sur $[-2; 3]$ dont le tableau de variations est donne ci-dessous.

```typst
#import "@preview/vartable:0.2.3": tabvar

#tabvar(
  variable: $x$,
  label: (
    ([signe de $f'(x)$], "s"),
    ([variation de $f$], "v"),
  ),
  domain: ($-2$, $-1$, $1$, $3$),
  contents: (
    ($-$, $+$, $-$),
    (
      (center, $-1$),
      (top, $2$),
      (top, $2$),
      (bottom, $-3$),
    ),
  ),
)
```

En utilisant le TVI, determiner le nombre de solutions de l equation $f(x) = 0$ sur $[-2; 3]$.
</Enonce>

<Solution>
D apres le tableau de variations :

- $f(-2) = -1 < 0$
- $f(-1) = 2 > 0$
- $f(1) = 2 > 0$
- $f(3) = -3 < 0$

Par le TVI, il existe une unique solution dans $]-2; -1[$ et une dans $]1; 3[$.

Au total, l equation admet **2 solutions**.
</Solution>

<Methode>
Utiliser le tableau de variations pour reperer les changements de signe.
</Methode>
