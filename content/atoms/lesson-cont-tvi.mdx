### Intuition

Imaginons une courbe **continue** qui passe d'une valeur **positive** a une valeur **negative**. Pour aller de l'une a l'autre sans lever le crayon, elle **doit** croiser l'axe des abscisses.

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-0.5, 0), (6.5, 0), stroke: 0.8pt, mark: (end: ">", fill: black))
  line((0, -0.5), (0, 5), stroke: 0.8pt, mark: (end: ">", fill: black))

  // Courbe f (bezier)
  bezier((0.5, 1), (5.5, 4.2), (2, 4.5), (4, 0.5), stroke: 2pt + blue)

  // Points a et b sur l'axe x
  line((0.5, 0), (0.5, 1), stroke: (dash: "dashed", paint: gray))
  line((5.5, 0), (5.5, 4.2), stroke: (dash: "dashed", paint: gray))

  // f(a) et f(b) sur l'axe y
  line((0, 1), (0.5, 1), stroke: (dash: "dashed", paint: gray))
  line((0, 4.2), (5.5, 4.2), stroke: (dash: "dashed", paint: gray))

  // Ligne horizontale y = k
  line((-0.3, 2.5), (6.3, 2.5), stroke: (dash: "dotted", paint: red))

  // Point c (intersection)
  circle((2.8, 2.5), radius: 0.08, fill: red, stroke: red)

  // Labels
  content((0.5, -0.3), $a$)
  content((5.5, -0.3), $b$)
  content((2.8, -0.3), $c$)
  content((-0.5, 1), $f(a)$)
  content((-0.5, 4.2), $f(b)$)
  content((-0.3, 2.5), anchor: "east", $k$)
  content((6.5, -0.4), $x$)
  content((-0.3, 5), $y$)

  // Ligne verticale c
  line((2.8, 0), (2.8, 2.5), stroke: (dash: "dashed", paint: red))
})
```

C'est exactement ce que dit le theoreme des valeurs intermediaires : une fonction continue ne peut pas "sauter" par-dessus une valeur.

### Le theoreme

<Theorem title="Theoreme des valeurs intermediaires (TVI)">

Soit $f$ une fonction **continue** sur un intervalle $[a, b]$.

Pour tout reel $k$ compris entre $f(a)$ et $f(b)$, il existe au moins un $c \in [a, b]$ tel que :

$$f(c) = k$$

</Theorem>

<Remark>

Le TVI garantit l'**existence** d'une solution, mais pas son **unicite**. Il peut y avoir plusieurs valeurs de $c$ qui conviennent.

</Remark>

### Corollaire : zeros d'une fonction

<Property title="Corollaire du TVI (zeros)">

Si $f$ est **continue** sur $[a, b]$ et si $f(a)$ et $f(b)$ sont de **signes contraires**, alors il existe au moins un $c \in ]a, b[$ tel que $f(c) = 0$.

$$f(a) \times f(b) < 0 \implies \exists\, c \in ]a,b[,\ f(c) = 0$$

</Property>

### Unicite : le cas strictement monotone

<Property title="TVI + monotonie stricte">

Si $f$ est **continue** et **strictement monotone** sur $[a, b]$, alors pour tout $k$ compris entre $f(a)$ et $f(b)$, l'equation $f(x) = k$ admet une **unique** solution dans $[a, b]$.

</Property>

### Methode d'application

Pour montrer qu'une equation $f(x) = 0$ admet une solution dans $[a, b]$ :

1. Verifier que $f$ est **continue** sur $[a, b]$
2. Calculer $f(a)$ et $f(b)$
3. Verifier que $f(a) \times f(b) < 0$ (signes contraires)
4. Conclure par le **TVI**

<Example title="Application du TVI">

**Montrer que** $f(x) = x^3 - 3x + 1$ **admet une racine dans** $[0, 1]$.

**1.** $f$ est une fonction polynomiale, donc **continue sur $\mathbb{R}$**, et en particulier sur $[0, 1]$.

**2.** On calcule :

- $f(0) = 0 - 0 + 1 = 1 > 0$
- $f(1) = 1 - 3 + 1 = -1 < 0$

**3.** On a $f(0) \times f(1) = 1 \times (-1) = -1 < 0$.

**4.** $f$ est continue sur $[0, 1]$ et $f(0) \times f(1) < 0$, donc par le **TVI**, il existe $c \in ]0, 1[$ tel que $f(c) = 0$.

</Example>

### Affiner par dichotomie

Une fois l'existence etablie, on peut **affiner** l'encadrement en coupant l'intervalle en deux a chaque etape.

<Example title="Dichotomie">

On sait que $f(x) = x^3 - 3x + 1$ s'annule dans $[0, 1]$. Affinons :

- $f(0{,}5) = 0{,}125 - 1{,}5 + 1 = -0{,}375 < 0$ et $f(0) > 0$ donc la racine est dans $[0\,;\, 0{,}5]$
- $f(0{,}25) = 0{,}016 - 0{,}75 + 1 = 0{,}266 > 0$ donc la racine est dans $[0{,}25\,;\, 0{,}5]$

On a encadre la racine : $c \in [0{,}25\,;\, 0{,}5]$.

</Example>

<Attention>

Ne jamais oublier de **verifier la continuite** avant d'appliquer le TVI ! Le theoreme est **faux** si $f$ n'est pas continue. Par exemple, la fonction partie entiere saute de $0$ a $1$ sans passer par les valeurs intermediaires.

</Attention>
