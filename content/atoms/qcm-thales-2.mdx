<Question>
Dans une configuration de Thales avec $(BC) \parallel (DE)$, si $\frac{AB}{AD} = \frac{2}{3}$, alors :
</Question>

<Option correct>$\frac{AC}{AE} = \frac{2}{3}$</Option>
<Option>$\frac{AC}{AE} = \frac{3}{2}$</Option>
<Option>$\frac{AC}{AE} = \frac{1}{3}$</Option>
<Option>$\frac{BC}{DE} = \frac{3}{2}$</Option>

<Explanation>
D'apres le theoreme de Thales, les rapports sont egaux : $\frac{AB}{AD} = \frac{AC}{AE} = \frac{BC}{DE}$.
</Explanation>
