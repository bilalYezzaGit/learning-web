Dans une configuration de Thales avec $(BC) \parallel (DE)$, si $\frac{AB}{AD} = \frac{2}{3}$, alors :

- [x] $\frac{AC}{AE} = \frac{2}{3}$
- [ ] $\frac{AC}{AE} = \frac{3}{2}$
- [ ] $\frac{AC}{AE} = \frac{1}{3}$
- [ ] $\frac{BC}{DE} = \frac{3}{2}$

> D'apres le theoreme de Thales, les rapports sont egaux : $\frac{AB}{AD} = \frac{AC}{AE} = \frac{BC}{DE}$.
