<Enonce>
Factoriser les expressions suivantes :

1. $x^2 - 16$
2. $9x^2 + 6x + 1$
3. $4x^2 - 12x + 9$
4. $x^2 - 2\sqrt{5}x + 5$
5. $(x+1)^2 - (2x-3)^2$
</Enonce>

<Solution>
**1.** Factorisation de $x^2 - 16$ :

On reconnaît $a^2 - b^2 = (a+b)(a-b)$ avec $a = x$ et $b = 4$ :

$$x^2 - 16 = (x-4)(x+4)$$

**2.** Factorisation de $9x^2 + 6x + 1$ :

On reconnaît $a^2 + 2ab + b^2 = (a+b)^2$ avec $a = 3x$ et $b = 1$ :

$$9x^2 + 6x + 1 = (3x+1)^2$$

**3.** Factorisation de $4x^2 - 12x + 9$ :

On reconnaît $a^2 - 2ab + b^2 = (a-b)^2$ avec $a = 2x$ et $b = 3$ :

$$4x^2 - 12x + 9 = (2x-3)^2$$

**4.** Factorisation de $x^2 - 2\sqrt{5}x + 5$ :

On reconnaît $a^2 - 2ab + b^2 = (a-b)^2$ avec $a = x$ et $b = \sqrt{5}$ :

$$x^2 - 2\sqrt{5}x + 5 = (x-\sqrt{5})^2$$

**5.** Factorisation de $(x+1)^2 - (2x-3)^2$ :

On utilise $A^2 - B^2 = (A-B)(A+B)$ avec $A = x+1$ et $B = 2x-3$ :

$$\begin{align*}
(x+1)^2 - (2x-3)^2 &= [(x+1) - (2x-3)][(x+1) + (2x-3)] \\
&= (x+1-2x+3)(x+1+2x-3) \\
&= (-x+4)(3x-2)
\end{align*}$$

On peut aussi ecrire : $(4-x)(3x-2)$
</Solution>

<Methode>
**Reconnaitre les identites remarquables** :

1. $a^2 - b^2 = (a-b)(a+b)$ — difference de deux carres
2. $a^2 + 2ab + b^2 = (a+b)^2$ — carre d'une somme
3. $a^2 - 2ab + b^2 = (a-b)^2$ — carre d'une difference

**Astuces** :
- Pour $a^2 - b^2$ : chercher deux termes qui sont des carres
- Pour $a^2 \pm 2ab + b^2$ : verifier que le terme du milieu est bien $2ab$
</Methode>

<Hint>
Pour la question 5, identifiez d'abord les termes $A$ et $B$, puis appliquez la formule $A^2 - B^2$.
</Hint>
