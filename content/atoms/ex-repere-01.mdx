<Enonce>
Dans un repere orthonorme $(O, \vec{i}, \vec{j})$, on donne les points $A(2, 3)$, $B(6, 0)$ et $C(-1, 4)$.

1. Calculer les coordonnees du vecteur $\vec{AB}$.
2. Calculer les coordonnees du milieu $I$ du segment $[AB]$.
3. Calculer la distance $AB$.
4. Le triangle $ABC$ est-il isocele ?
</Enonce>

<Solution>
**1)** Coordonnees de $\vec{AB}$ :

$\vec{AB}\begin{pmatrix} 6 - 2 \\ 0 - 3 \end{pmatrix} = \begin{pmatrix} 4 \\ -3 \end{pmatrix}$

**2)** Coordonnees du milieu $I$ :

$I\left(\frac{2 + 6}{2}, \frac{3 + 0}{2}\right) = I(4; 1{,}5)$

**3)** Distance $AB$ :

$AB = \sqrt{(6-2)^2 + (0-3)^2} = \sqrt{16 + 9} = \sqrt{25} = 5$

**4)** Calculons les trois distances :

$AB = 5$ (calcule ci-dessus)

$AC = \sqrt{(-1-2)^2 + (4-3)^2} = \sqrt{9 + 1} = \sqrt{10}$

$BC = \sqrt{(-1-6)^2 + (4-0)^2} = \sqrt{49 + 16} = \sqrt{65}$

Comme $AB \neq AC \neq BC$, le triangle $ABC$ n'est pas isocele.
</Solution>
