<Question>
$(x + 4)^2$ est egal a :
</Question>

<Option>$x^2 + 16$</Option>
<Option>$x^2 + 4x + 16$</Option>
<Option correct>$x^2 + 8x + 16$</Option>
<Option>$x^2 + 4x + 8$</Option>

<Explanation>
L'erreur frequente est d'oublier le **double produit** dans l'identite remarquable.

La regle correcte est $(a+b)^2 = a^2 + \mathbf{2ab} + b^2$.

Ici, avec $a = x$ et $b = 4$ :

$$2ab = 2 \times x \times 4 = 8x$$

Donc $(x + 4)^2 = x^2 + 8x + 16$.

**Piege courant** : $(a+b)^2 \neq a^2 + b^2$ !
</Explanation>
