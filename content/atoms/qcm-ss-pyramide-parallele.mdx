<Question>
La section d'une pyramide a base carree par un plan parallele a la base est :
</Question>

<Option>Un triangle</Option>
<Option>Un rectangle</Option>
<Option correct>Un carre</Option>
<Option>Un losange</Option>

<Explanation>
Un plan parallele a la base d'une pyramide donne une figure **semblable a la base**. Si la base est un carre, la section est un carre (reduit selon le rapport de reduction $k$).
</Explanation>
