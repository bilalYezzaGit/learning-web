<Enonce>
On considere les droites $D_1 : y = 2x - 1$ et $D_2 : y = -x + 5$.

1. Tracer les deux droites dans le meme repere orthonorme.
2. Lire graphiquement les coordonnees du point d'intersection.
3. Verifier algebriquement le resultat.
</Enonce>

<Solution>
**1. Trace des droites**

Pour $D_1 : y = 2x - 1$ :
- Si $x = 0$ : $y = -1$, point $(0, -1)$
- Si $x = 1$ : $y = 1$, point $(1, 1)$

Pour $D_2 : y = -x + 5$ :
- Si $x = 0$ : $y = 5$, point $(0, 5)$
- Si $x = 5$ : $y = 0$, point $(5, 0)$

On trace les deux droites en reliant ces points.

**2. Lecture graphique**

Les deux droites se coupent au point de coordonnees $(2, 3)$.

**3. Verification algebrique**

Le point d'intersection est solution du systeme :

$$\begin{cases} y = 2x - 1 \\ y = -x + 5 \end{cases}$$

Par substitution : $2x - 1 = -x + 5$

$$3x = 6$$

$$x = 2$$

On en deduit $y = 2(2) - 1 = 3$.

Verification avec la deuxieme equation : $y = -2 + 5 = 3$. Correct.

**Le point d'intersection est bien $(2, 3)$.**
</Solution>
