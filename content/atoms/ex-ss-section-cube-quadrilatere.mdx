<Enonce>
ABCDEFGH est un cube d'arete a. On considere les points suivants :
- M est le milieu de [AB]
- N est le milieu de [BC]
- P est un point de [DH] tel que $DP = \frac{1}{3}DH$

Construire la section du cube par le plan (MNP). Quelle est la nature de ce quadrilatere ?
</Enonce>

<Solution>
**Etape 1 : Positionnement des points**

- M est le milieu de [AB] sur la face inferieure ABCD
- N est le milieu de [BC] sur la face inferieure ABCD
- P est sur [DH] avec $DP = \frac{a}{3}$ sur l'arete verticale arriere gauche

**Etape 2 : Construction methodique de la section**

On cherche les intersections du plan (MNP) avec les aretes du cube.

**Sur la face ABCD (base inferieure)** :
- M et N sont deja sur cette face
- Le segment [MN] appartient a l'intersection du plan avec ABCD

**Sur la face DHGC (face arriere)** :
- P est sur l'arete [DH]
- Le plan coupe [DC] : soit Q ce point d'intersection
- Pour trouver Q : (MN) et (DC) sont paralleles (car AB // DC)
- La droite passant par P et parallele a (MN) coupe [DC] en Q
- Par construction, $DQ = \frac{a}{2}$ (Q milieu de [DC])

**Sur la face ADHE (face gauche)** :
- Le plan coupe cette face selon [MP]

**Sur la face BCGF (face droite)** :
- Le plan coupe cette face selon [NQ]

**Etape 3 : Identification du polygone**

La section est le **quadrilatere MNQP**.

**Etape 4 : Nature du quadrilatere**

Analysons les cotes :
- M est milieu de [AB] et N est milieu de [BC]
- Dans le triangle ABC rectangle en B : $MN = \frac{AC}{2} = \frac{a\sqrt{2}}{2}$

Par ailleurs :
- (MN) // (DC) donc (MN) // (PQ)
- Les faces opposees du cube sont paralleles

Le plan (MNP) coupe deux faces opposees (ABCD et EFGH) selon des segments paralleles.

La section MNQP est un **trapeze** avec (MN) // (PQ).
</Solution>

<Methode>
Pour construire une section de cube etape par etape :

1. Identifier les points donnes et leur position
2. Chercher l'intersection du plan avec chaque face en utilisant :
   - Les droites paralleles aux aretes
   - Les proprietes des plans coupant des faces paralleles
3. Relier les points d'intersection dans l'ordre
4. Identifier la nature du polygone obtenu
</Methode>

<Hint>
Utilisez la propriete : un plan qui coupe deux faces paralleles d'un cube les coupe selon des segments paralleles.
</Hint>
