<Enonce>
Resoudre les equations suivantes :

a) $|x| = 5$

b) $|2x - 1| = 3$

c) $|x - 4| = |x + 2|$

d) $|3x + 2| = -1$
</Enonce>

<Solution>
**a)** $|x| = 5$

$$x = 5 \quad \text{ou} \quad x = -5$$

**Ensemble des solutions** : $S = \{-5, 5\}$

**b)** $|2x - 1| = 3$

On applique la definition : $2x - 1 = 3$ ou $2x - 1 = -3$

- **Cas 1** : $2x - 1 = 3 \Rightarrow 2x = 4 \Rightarrow x = 2$
- **Cas 2** : $2x - 1 = -3 \Rightarrow 2x = -2 \Rightarrow x = -1$

**Ensemble des solutions** : $S = \{-1, 2\}$

**c)** $|x - 4| = |x + 2|$

On peut elever au carre pour eliminer les valeurs absolues :

$$(x - 4)^2 = (x + 2)^2$$

$$x^2 - 8x + 16 = x^2 + 4x + 4$$

$$-8x + 16 = 4x + 4$$

$$-12x = -12$$

$$x = 1$$

**Verification** : $|1 - 4| = |-3| = 3$ et $|1 + 2| = |3| = 3$ âœ“

**Ensemble des solutions** : $S = \{1\}$

**d)** $|3x + 2| = -1$

Une valeur absolue est toujours **positive ou nulle**. Il est impossible qu'elle soit egale a $-1$.

**Ensemble des solutions** : $S = \emptyset$ (aucune solution)
</Solution>

<Methode>
Pour resoudre une equation de la forme $|f(x)| = a$ :

1. **Si $a < 0$** : aucune solution (une valeur absolue ne peut pas etre negative)

2. **Si $a = 0$** : resoudre $f(x) = 0$

3. **Si $a > 0$** : resoudre les deux equations :
   - $f(x) = a$
   - $f(x) = -a$

Pour une equation du type $|f(x)| = |g(x)|$, on peut :
- Soit considerer les cas $f(x) = g(x)$ et $f(x) = -g(x)$
- Soit elever au carre : $(f(x))^2 = (g(x))^2$
</Methode>
