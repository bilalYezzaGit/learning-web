### Qu'est-ce qu'une fonction lineaire ?

Une **fonction lineaire** est une fonction de la forme :

$$f : x \mapsto ax$$

ou $a$ est un nombre reel appele le **coefficient** de la fonction.

<Definition title="Fonction lineaire">

Une fonction $f$ est dite **lineaire** s'il existe un nombre reel $a$ tel que pour tout $x \in \mathbb{R}$ :

$$f(x) = ax$$

Le nombre $a$ est appele le **coefficient** (ou **coefficient directeur**) de la fonction lineaire.

</Definition>

### Proprietes immediates

Une fonction lineaire $f(x) = ax$ verifie toujours :

- **$f(0) = 0$** : l'image de $0$ est toujours $0$
- **$f(1) = a$** : l'image de $1$ est le coefficient $a$

<Example title="Fonction lineaire f(x) = 3x">

Soit $f$ la fonction lineaire definie par $f(x) = 3x$.

Calculons quelques images :
- $f(0) = 3 \times 0 = 0$
- $f(1) = 3 \times 1 = 3$
- $f(2) = 3 \times 2 = 6$
- $f(-1) = 3 \times (-1) = -3$
- $f(0,5) = 3 \times 0,5 = 1,5$

On remarque que **multiplier par 3** revient a appliquer $f$.

</Example>

### Vocabulaire

Pour une fonction lineaire $f(x) = ax$ :

- L'**image** de $x$ par $f$ est $f(x) = ax$
- L'**antecedent** de $y$ par $f$ est la valeur $x$ telle que $f(x) = y$

Pour trouver un antecedent : si $f(x) = y$ et $a \neq 0$, alors $x = \frac{y}{a}$.

<Remark>

Une fonction lineaire traduit une situation de **proportionnalite** : si on multiplie $x$ par un nombre, l'image $f(x)$ est multipliee par le meme nombre.

Par exemple, si $f(x) = 3x$ : multiplier $x$ par 2 donne $f(2x) = 3 \times 2x = 2 \times 3x = 2f(x)$.

</Remark>
