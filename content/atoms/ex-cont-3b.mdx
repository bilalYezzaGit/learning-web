<Enonce>
Soit $f(x) = x^2 - 2x - 1$ definie sur $[-1; 3]$. Le graphique suivant represente la fonction $f$.

```typst
#import "@preview/cetz:0.3.4": canvas, draw
#import "@preview/cetz-plot:0.1.1": plot

#canvas({
  plot.plot(
    size: (8, 6),
    x-label: $x$,
    y-label: $y$,
    x-tick-step: 1,
    y-tick-step: 1,
    x-min: -1, x-max: 3,
    y-min: -3, y-max: 3,
    {
      plot.add(domain: (-1, 3), x => calc.pow(x, 2) - 2 * x - 1,
        style: (stroke: 2pt + blue), label: $f(x) = x^2 - 2x - 1$)
      plot.add(((- 1, 2), (3, 2)),
        style: (stroke: none), mark: "o", mark-size: 0.15,
        mark-style: (stroke: red, fill: red))
    },
  )
})
```

A l aide du graphique, determiner le nombre de solutions de l equation $f(x) = 0$ dans $[-1; 3]$.
</Enonce>

<Solution>
On observe sur le graphique que $f(-1) = 2 > 0$ et $f(3) = 2 > 0$.

Cependant, $f(1) = -2 < 0$.

Par le TVI applique sur $[-1; 1]$ et sur $[1; 3]$, il existe deux solutions : une dans $]-1; 1[$ et une dans $]1; 3[$.
</Solution>

<Methode>
Utiliser le graphique pour reperer les changements de signe et appliquer le TVI.
</Methode>
