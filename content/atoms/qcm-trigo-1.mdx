Dans un triangle rectangle, le cosinus d'un angle aigu est egal a :

- [ ] cote oppose / hypotenuse
- [x] cote adjacent / hypotenuse
- [ ] cote oppose / cote adjacent
- [ ] hypotenuse / cote adjacent

> $\cos(\alpha) = \frac{\text{cote adjacent}}{\text{hypotenuse}}$
