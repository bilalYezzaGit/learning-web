<Question>
Pour le systeme $\begin{cases} x + 3y = 7 \\ x - 3y = 1 \end{cases}$, en additionnant les deux equations membre a membre, on obtient :
</Question>

<Option correct>$2x = 8$</Option>
<Option>$6y = 8$</Option>
<Option>$2x + 6y = 8$</Option>
<Option>$2x = 6$</Option>

<Explanation>
La methode par combinaison lineaire consiste a additionner (ou soustraire) les deux equations pour eliminer une inconnue.

En additionnant les deux equations :

$$(x + 3y) + (x - 3y) = 7 + 1$$

$$2x = 8$$

Les termes en $y$ s'annulent ($+3y - 3y = 0$), ce qui permet de trouver $x = 4$. Puis on peut deduire $y = 1$.
</Explanation>
