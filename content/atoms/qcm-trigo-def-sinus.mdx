<Question>
Dans un triangle rectangle, le sinus d'un angle aigu est egal a :
</Question>

<Option>cote adjacent / hypotenuse</Option>
<Option correct>cote oppose / hypotenuse</Option>
<Option>cote oppose / cote adjacent</Option>
<Option>hypotenuse / cote oppose</Option>

<Explanation>
Le sinus d'un angle aigu $\alpha$ est defini par $\sin(\alpha) = \frac{\text{cote oppose}}{\text{hypotenuse}}$.

Pour retenir facilement : **SOH** (Sinus = Oppose / Hypotenuse).
</Explanation>
