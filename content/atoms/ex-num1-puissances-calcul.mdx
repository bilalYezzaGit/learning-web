<Enonce>
Calculer et simplifier les expressions suivantes :

a) $2^3 \times 2^5$

b) $\frac{3^7}{3^4}$

c) $(5^2)^3$

d) $\left(\frac{2}{3}\right)^{-2}$
</Enonce>

<Solution>
**a)** On utilise la règle $a^m \times a^n = a^{m+n}$ :

$$2^3 \times 2^5 = 2^{3+5} = 2^8 = 256$$

**b)** On utilise la règle $\frac{a^m}{a^n} = a^{m-n}$ :

$$\frac{3^7}{3^4} = 3^{7-4} = 3^3 = 27$$

**c)** On utilise la règle $(a^m)^n = a^{m \times n}$ :

$$(5^2)^3 = 5^{2 \times 3} = 5^6 = 15625$$

**d)** On utilise la règle $\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$ :

$$\left(\frac{2}{3}\right)^{-2} = \left(\frac{3}{2}\right)^2 = \frac{9}{4}$$
</Solution>

<Methode>
Pour calculer avec les puissances, on utilise les règles suivantes :

1. **Produit** : $a^m \times a^n = a^{m+n}$
2. **Quotient** : $\frac{a^m}{a^n} = a^{m-n}$
3. **Puissance de puissance** : $(a^m)^n = a^{m \times n}$
4. **Exposant négatif** : $a^{-n} = \frac{1}{a^n}$ et $\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$
</Methode>
