### Representation graphique

Chaque equation lineaire $ax + by = c$ avec $b \neq 0$ represente une **droite** dans le plan.

Resoudre graphiquement un systeme revient a chercher le **point d'intersection** des deux droites.

<Property title="Interpretation graphique">

Resoudre graphiquement le systeme :

$$\begin{cases}
a_1 x + b_1 y = c_1 \\
a_2 x + b_2 y = c_2
\end{cases}$$

revient a chercher les coordonnees du point d'intersection des deux droites $(D_1)$ et $(D_2)$ d'equations respectives $a_1 x + b_1 y = c_1$ et $a_2 x + b_2 y = c_2$.

</Property>

### Les trois cas possibles

<Definition title="Position relative de deux droites">

Selon la position relative des droites $(D_1)$ et $(D_2)$, trois cas sont possibles :

**1. Les droites sont secantes :**
- Elles se coupent en **un seul point**
- Le systeme admet **une solution unique**
- Les coordonnees du point d'intersection sont la solution

**2. Les droites sont paralleles :**
- Elles n'ont **aucun point commun**
- Le systeme n'admet **aucune solution**
- On dit que le systeme est **impossible**

**3. Les droites sont confondues :**
- Elles ont **tous leurs points en commun**
- Le systeme admet **une infinite de solutions**
- Tous les points de la droite sont solutions

</Definition>

### Lien avec les coefficients

On peut determiner la nature du systeme en comparant les coefficients des equations.

<Property title="Critere algebrique">

Pour le systeme $\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}$ avec $a_2, b_2, a_1, b_1 \neq 0$ :

- Si $\frac{a_1}{a_2} \neq \frac{b_1}{b_2}$ : les droites sont **secantes** → **une solution unique**

- Si $\frac{a_1}{a_2} = \frac{b_1}{b_2} \neq \frac{c_1}{c_2}$ : les droites sont **paralleles** → **aucune solution**

- Si $\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2}$ : les droites sont **confondues** → **infinite de solutions**

</Property>

<Example title="Determination graphique du nombre de solutions">

Determiner graphiquement le nombre de solutions du systeme :

$$\begin{cases}
x + 2y = 4 \\
2x - y = 1
\end{cases}$$

Verifions avec les coefficients : $a_1 = 1$, $b_1 = 2$, $a_2 = 2$, $b_2 = -1$.

$$\frac{a_1}{a_2} = \frac{1}{2} \quad \text{et} \quad \frac{b_1}{b_2} = \frac{2}{-1} = -2$$

On a $\frac{1}{2} \neq -2$, donc les droites sont **secantes** et le systeme admet **une solution unique**.

</Example>

<Remark>

L'interpretation graphique est un **outil de visualisation** tres utile pour comprendre la nature des solutions, mais pour obtenir les valeurs exactes, on utilise les methodes algebriques (substitution ou combinaison lineaire).

</Remark>
