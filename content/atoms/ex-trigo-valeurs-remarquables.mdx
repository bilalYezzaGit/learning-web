<Enonce>
Dans un triangle ABC rectangle en A, l'angle B mesure 30° et AB = $6\sqrt{3}$.

1. Calculer AC (donner la valeur exacte).
2. Calculer BC (donner la valeur exacte).
3. Verifier avec le theoreme de Pythagore.
4. Que vaut l'angle C?
</Enonce>

<Solution>
**1.** Calcul de AC :

AB est le cote adjacent a l'angle B, et AC est le cote oppose.

$$\tan(30°) = \frac{AC}{AB}$$

On sait que $\tan(30°) = \frac{\sqrt{3}}{3} = \frac{1}{\sqrt{3}}$

$$AC = AB \times \tan(30°) = 6\sqrt{3} \times \frac{\sqrt{3}}{3} = \frac{6\sqrt{3} \times \sqrt{3}}{3} = \frac{6 \times 3}{3} = 6$$

Donc **AC = 6**.

**2.** Calcul de BC :

BC est l'hypotenuse.

$$\cos(30°) = \frac{AB}{BC}$$

On sait que $\cos(30°) = \frac{\sqrt{3}}{2}$

$$BC = \frac{AB}{\cos(30°)} = \frac{6\sqrt{3}}{\frac{\sqrt{3}}{2}} = 6\sqrt{3} \times \frac{2}{\sqrt{3}} = \frac{12\sqrt{3}}{\sqrt{3}} = 12$$

Donc **BC = 12**.

**3.** Verification avec Pythagore :

$$AC^2 + AB^2 = 6^2 + (6\sqrt{3})^2 = 36 + 36 \times 3 = 36 + 108 = 144$$

$$BC^2 = 12^2 = 144$$

On a bien $AC^2 + AB^2 = BC^2$, donc le resultat est correct.

**4.** L'angle C :

Dans un triangle, la somme des angles vaut 180°.

$$\hat{A} + \hat{B} + \hat{C} = 180°$$

$$90° + 30° + \hat{C} = 180°$$

$$\hat{C} = 60°$$

L'angle C mesure **60°**.
</Solution>

<Erreurs>
- Confondre les valeurs de sin(30°) et sin(60°) : $\sin(30°) = \frac{1}{2}$ alors que $\sin(60°) = \frac{\sqrt{3}}{2}$
- Confondre les valeurs de cos(30°) et cos(60°) : $\cos(30°) = \frac{\sqrt{3}}{2}$ alors que $\cos(60°) = \frac{1}{2}$
- Oublier de simplifier les expressions avec racines carrees
- Ne pas donner les valeurs exactes et arrondir trop tot
</Erreurs>
