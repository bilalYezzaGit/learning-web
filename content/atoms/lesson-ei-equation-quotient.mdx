### Signe d'un quotient

<Property title="Regle du signe d'un quotient">

Le signe d'un quotient $\frac{A}{B}$ (avec $B \neq 0$) est determine par les signes de $A$ et de $B$ :

| Signe de $A$ | Signe de $B$ | Signe de $\frac{A}{B}$ |
|--------------|--------------|------------------------|
| $+$ | $+$ | $+$ |
| $+$ | $-$ | $-$ |
| $-$ | $+$ | $-$ |
| $-$ | $-$ | $+$ |

**Regle simple :** Le quotient est **positif** si $A$ et $B$ sont de **meme signe**, **negatif** s'ils sont de **signes opposes**.

</Property>

### Tableau de signes

Pour etudier le signe d'un quotient $\frac{A(x)}{B(x)}$ :

1. **Etudier le signe** de $A(x)$
2. **Etudier le signe** de $B(x)$
3. **Construire** le tableau de signes en appliquant la regle du quotient

<Example title="Etude de signe d'un quotient">

Etudier le signe de $f(x) = \frac{2x - 6}{x + 1}$.

**Solution :**

**Etape 1 :** Signe de $A(x) = 2x - 6$

$2x - 6 = 0 \Leftrightarrow x = 3$

- Si $x < 3$ : $A(x) < 0$
- Si $x > 3$ : $A(x) > 0$

**Etape 2 :** Signe de $B(x) = x + 1$

$x + 1 = 0 \Leftrightarrow x = -1$

- Si $x < -1$ : $B(x) < 0$
- Si $x > -1$ : $B(x) > 0$

**Etape 3 :** Tableau de signes

| $x$ | $-\infty$ | | $-1$ | | $3$ | | $+\infty$ |
|-----|-----------|---|------|---|-----|---|-----------|
| $2x - 6$ | | $-$ | | $-$ | $0$ | $+$ | |
| $x + 1$ | | $-$ | $0$ | $+$ | | $+$ | |
| $\frac{2x - 6}{x + 1}$ | | $+$ | $\|$ | $-$ | $0$ | $+$ | |

**Conclusion :**

- $f(x) > 0$ sur $]-\infty, -1[ \cup ]3, +\infty[$
- $f(x) = 0$ pour $x = 3$
- $f(x) < 0$ sur $]-1, 3[$
- $f(x)$ n'existe pas en $x = -1$ (valeur interdite)

</Example>

### Resolution d'inequations quotient

<Property title="Methode de resolution">

Pour resoudre une inequation du type $\frac{A(x)}{B(x)} \geq 0$ (ou $>$, $\leq$, $<$) :

1. **Construire** le tableau de signes de $\frac{A(x)}{B(x)}$
2. **Lire** sur le tableau les intervalles ou le quotient a le signe demande
3. **Attention** : ne jamais inclure les valeurs qui annulent le denominateur

</Property>

<Example title="Inequation quotient">

Resoudre $\frac{2x - 6}{x + 1} \geq 0$.

**Solution :**

D'apres le tableau de signes precedent, $\frac{2x - 6}{x + 1} \geq 0$ signifie que le quotient est **positif ou nul**.

D'apres le tableau :

- Le quotient est positif sur $]-\infty, -1[$ et sur $]3, +\infty[$
- Le quotient est nul en $x = 3$

**L'ensemble des solutions est** $S = ]-\infty, -1[ \cup [3, +\infty[$.

</Example>

<Attention title="Condition d'existence">

Le denominateur ne doit **jamais** etre nul.

Dans l'exemple precedent, $x = -1$ annule le denominateur, donc $x = -1$ est **toujours exclu** de l'ensemble des solutions, meme si l'inequation comporte le signe $\geq$ ou $\leq$.

</Attention>

<Remark>

Pour une inequation du type $\frac{A(x)}{B(x)} \leq 0$, on cherche les valeurs ou le quotient est **negatif ou nul**.

Attention : le quotient est nul quand $A(x) = 0$ (et $B(x) \neq 0$), pas quand $B(x) = 0$.

</Remark>
