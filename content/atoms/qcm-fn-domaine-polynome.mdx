:::question
Le domaine de definition de $f(x) = 3x^4 - 2x^2 + 5$ est :
:::

:::option
$\mathbb{R}^+$
:::
:::option
$\mathbb{R}^*$
:::
:::option{correct}
$\mathbb{R}$
:::
:::option
$[-5, 5]$
:::

:::explanation
Une fonction polynomiale est definie sur tout $\mathbb{R}$. Il n'y a aucune restriction car toutes les operations (puissances entieres positives, somme, produit) sont definies pour tout reel.
:::
