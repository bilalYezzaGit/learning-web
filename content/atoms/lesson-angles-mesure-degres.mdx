### Qu'est-ce qu'un angle ?

<Definition title="Angle">

Un **angle** est une figure formee par deux **demi-droites** ayant la meme origine. Le point commun s'appelle le **sommet** de l'angle, et les deux demi-droites s'appellent les **cotes** de l'angle.

</Definition>

### Unite de mesure : le degre

L'unite de mesure des angles est le **degre**, note $°$.

<Property title="Reference : le tour complet">

Un tour complet autour d'un point mesure **360 degres** : $360°$.

</Property>

Par consequent :
- Un demi-tour mesure $180°$
- Un quart de tour mesure $90°$

### Classification des angles

Selon leur mesure, on distingue plusieurs types d'angles :

<Definition title="Types d'angles">

Soit $\alpha$ la mesure d'un angle. On dit que l'angle est :

- **Nul** si $\alpha = 0°$
- **Aigu** si $0° < \alpha < 90°$
- **Droit** si $\alpha = 90°$
- **Obtus** si $90° < \alpha < 180°$
- **Plat** si $\alpha = 180°$

</Definition>

<Example title="Exemples d'angles">

- Un angle de $45°$ est **aigu** car $0° < 45° < 90°$
- Un angle de $90°$ est **droit**
- Un angle de $120°$ est **obtus** car $90° < 120° < 180°$
- Un angle de $180°$ est **plat**

</Example>

### Mesure avec un rapporteur

En pratique, on mesure un angle avec un **rapporteur**, instrument gradue de $0°$ a $180°$. On place le centre du rapporteur sur le sommet de l'angle et on lit la mesure sur la graduation.

<Remark>

Pour mesurer un angle superieur a $180°$, on peut completer par rapport a un tour complet ($360°$).

</Remark>

### Propriete fondamentale dans un triangle

<Property title="Somme des angles d'un triangle">

Dans tout triangle, la somme des mesures des trois angles est egale a $180°$.

Si un triangle $ABC$ a des angles de mesures $\alpha$, $\beta$ et $\gamma$, alors :

$$\alpha + \beta + \gamma = 180°$$

</Property>

<Example title="Application">

Dans un triangle, deux angles mesurent $50°$ et $70°$. Quelle est la mesure du troisieme angle ?

Soit $\gamma$ la mesure du troisieme angle :

$$50° + 70° + \gamma = 180°$$

$$120° + \gamma = 180°$$

$$\gamma = 180° - 120° = 60°$$

Le troisieme angle mesure $60°$.

</Example>
