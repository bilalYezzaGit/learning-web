<Question>
La racine (le zéro) de $f(x) = 4x - 12$ est :
</Question>

<Option>$x = 12$</Option>
<Option>$x = 4$</Option>
<Option correct>$x = 3$</Option>
<Option>$x = -3$</Option>

<Explanation>
La racine est la valeur de $x$ pour laquelle $f(x) = 0$. On résout : $4x - 12 = 0$, donc $4x = 12$, d'où $x = 3$. On peut aussi utiliser la formule $x_0 = -\frac{b}{a} = -\frac{-12}{4} = \frac{12}{4} = 3$. L'erreur fréquente est de confondre la racine avec les coefficients $a$ ou $b$.
</Explanation>
