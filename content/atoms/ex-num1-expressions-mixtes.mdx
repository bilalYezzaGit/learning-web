<Enonce>
Simplifier les expressions suivantes :

a) $\frac{\sqrt{8} \times \sqrt{18}}{2^3}$

b) $\frac{(3\sqrt{2})^2}{6}$

c) $\sqrt{5^4 \times 2^2}$

d) $\frac{2^{-1} \times \sqrt{12}}{3^{-1}}$
</Enonce>

<Solution>
**a)** On commence par calculer le numérateur :

$$\sqrt{8} \times \sqrt{18} = \sqrt{8 \times 18} = \sqrt{144} = 12$$

Puis :

$$\frac{12}{2^3} = \frac{12}{8} = \frac{3}{2}$$

**b)** On développe le carré au numérateur :

$$(3\sqrt{2})^2 = 3^2 \times (\sqrt{2})^2 = 9 \times 2 = 18$$

Donc :

$$\frac{18}{6} = 3$$

**c)** On extrait les racines :

$$\sqrt{5^4 \times 2^2} = \sqrt{5^4} \times \sqrt{2^2}$$

$$= \sqrt{(5^2)^2} \times \sqrt{2^2} = 5^2 \times 2 = 25 \times 2 = 50$$

**d)** On simplifie d'abord les puissances négatives :

$$\frac{2^{-1} \times \sqrt{12}}{3^{-1}} = \frac{\frac{1}{2} \times \sqrt{12}}{\frac{1}{3}} = \frac{\sqrt{12}}{2} \times 3$$

On simplifie $\sqrt{12} = \sqrt{4 \times 3} = 2\sqrt{3}$ :

$$= \frac{2\sqrt{3}}{2} \times 3 = \sqrt{3} \times 3 = 3\sqrt{3}$$
</Solution>

<Methode>
Pour simplifier des expressions mixtes avec puissances et racines :

1. **Simplifier** les racines en extrayant les carrés parfaits
2. **Utiliser** les règles de puissances pour les exposants
3. **Appliquer** $\sqrt{a \times b} = \sqrt{a} \times \sqrt{b}$ et $\sqrt{a^2} = a$ (pour $a \geq 0$)
4. **Gérer** les exposants négatifs : $a^{-n} = \frac{1}{a^n}$
5. **Simplifier** l'expression finale
</Methode>

<Hint>
Pour la question d), commencez par transformer les exposants négatifs, puis simplifiez $\sqrt{12}$.
</Hint>
