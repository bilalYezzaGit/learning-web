:::question
Si $f(3) = 7$, alors :
:::

:::option
3 est l'image de 7
:::
:::option{correct}
7 est l'image de 3
:::
:::option
3 est l'antecedent de 3
:::
:::option
7 est l'antecedent de 3
:::

:::explanation
Par definition, si $f(a) = b$, alors $b$ est **l'image de $a$ par $f$**, et $a$ est **un antecedent de $b$ par $f$**. Donc $f(3) = 7$ signifie que 7 est l'image de 3, et que 3 est un antecedent de 7.
:::
