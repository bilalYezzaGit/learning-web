### Valeur absolue

:::definition[Valeur absolue]

La **valeur absolue** d'un reel $a$, notee $|a|$, est definie par :

$$|a| = \begin{cases} a & \text{si } a \geq 0 \\ -a & \text{si } a < 0 \end{cases}$$

:::

:::remark

Geometriquement, $|a|$ represente la **distance** du nombre $a$ a l'origine $0$ sur la droite reelle. On a toujours $|a| \geq 0$.

:::

### Fonctions avec valeur absolue

Pour etudier une fonction contenant des valeurs absolues, on **leve les valeurs absolues** sur chaque intervalle.

:::example[Fonction avec une valeur absolue]

Soit $f(x) = |2x - 1| - 3x$.

Le terme $2x - 1$ change de signe en $x = \frac{1}{2}$.

- Si $x \geq \frac{1}{2}$ : $|2x - 1| = 2x - 1$, donc $f(x) = (2x - 1) - 3x = -x - 1$
- Si $x < \frac{1}{2}$ : $|2x - 1| = -(2x - 1) = -2x + 1$, donc $f(x) = (-2x + 1) - 3x = -5x + 1$

$$f(x) = \begin{cases} -5x + 1 & \text{si } x < \frac{1}{2} \\ -x - 1 & \text{si } x \geq \frac{1}{2} \end{cases}$$

Sur chaque intervalle, $f$ est une fonction affine. On parle de **fonction affine par morceaux**.

:::

### Fonctions affines par morceaux

:::definition[Fonction affine par morceaux]

Une fonction est **affine par morceaux** si son domaine peut etre decoupe en intervalles sur chacun desquels la fonction est affine (de la forme $ax + b$).

:::

:::property[Representation graphique]

La courbe d'une fonction affine par morceaux est composee de **segments de droite**. Aux points de jonction, la courbe peut presenter un "angle" (un point anguleux).

:::

:::example[Fonction $g(x) = 2 - 2|x|$]

Le terme $|x|$ change de signe en $x = 0$.

- Si $x \geq 0$ : $g(x) = 2 - 2x$ (droite de pente $-2$)
- Si $x < 0$ : $g(x) = 2 - 2(-x) = 2 + 2x$ (droite de pente $+2$)

La courbe forme un "V" renverse avec un sommet en $(0, 2)$.

:::

### Fonction partie entiere

:::definition[Partie entiere]

La **partie entiere** d'un reel $x$, notee $E(x)$, est le plus grand entier relatif inferieur ou egal a $x$ :

$$E(x) = n \quad \text{si} \quad n \leq x < n + 1$$

:::

:::example[Valeurs de $E(x)$]

- $E(2{,}7) = 2$ car $2 \leq 2{,}7 < 3$
- $E(3) = 3$ car $3 \leq 3 < 4$
- $E(-1{,}3) = -2$ car $-2 \leq -1{,}3 < -1$
- $E(0{,}5) = 0$ car $0 \leq 0{,}5 < 1$

:::

:::attention

Pour les nombres negatifs, la partie entiere "descend" : $E(-1{,}3) = -2$ et non $-1$. C'est le plus grand entier **inferieur ou egal** a $x$.

:::

### Representation graphique de $E(x)$

:::property[Courbe en escalier]

La courbe de $y = E(x)$ est une **fonction en escalier** :

- Sur chaque intervalle $[n, n+1[$ (avec $n \in \mathbb{Z}$), la fonction est constante egale a $n$
- Le point $(n, n)$ est **inclus** (point ferme) : $E(n) = n$
- Le point $(n+1, n)$ est **exclu** (point ouvert) : on "saute" a la marche suivante

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-3.5, 0), (3.5, 0), stroke: 0.8pt, mark: (end: ">", fill: black))
  line((0, -3.5), (0, 3.5), stroke: 0.8pt, mark: (end: ">", fill: black))
  content((3.3, -0.3), $x$)
  content((-0.3, 3.3), $y$)
  content((-0.3, -0.3), $O$)

  // Ticks on x-axis
  for x in (-3, -2, -1, 1, 2, 3) {
    line((x, -0.08), (x, 0.08), stroke: 0.5pt)
    content((x, -0.3), text(size: 7pt, str(x)))
  }

  // Ticks on y-axis
  for y in (-3, -2, -1, 1, 2, 3) {
    line((-0.08, y), (0.08, y), stroke: 0.5pt)
    content((-0.3, y), text(size: 7pt, str(y)))
  }

  // Staircase for E(x) from -3 to 3
  for n in (-3, -2, -1, 0, 1, 2) {
    // Horizontal segment [n, n+1[
    line((n, n), (n + 1, n), stroke: 2pt + blue)
    // Closed dot at (n, n)
    circle((n, n), radius: 0.06, fill: blue, stroke: 1pt + blue)
    // Open dot at (n+1, n)
    circle((n + 1, n), radius: 0.06, fill: white, stroke: 1pt + blue)
  }

  // Add label
  content((2.5, 2.8), text(size: 8pt, fill: blue, $E(x)$))
})
```

:::

:::remark

La partie entiere est un exemple de fonction **discontinue** : elle presente des "sauts" aux points entiers. C'est un premier exemple de fonction non continue, notion qui sera approfondie dans le module sur la continuite.

:::

:::example[Partie fractionnaire]

La **partie fractionnaire** de $x$ est definie par $\{x\} = x - E(x)$.

On a toujours $0 \leq \{x\} < 1$. Par exemple, $\{2{,}7\} = 2{,}7 - 2 = 0{,}7$.

Sa courbe est une "dent de scie" : sur chaque intervalle $[n, n+1[$, c'est la droite $y = x - n$, partant du point ferme $(n, 0)$ et allant vers le point ouvert $(n+1, 1)$.

:::
