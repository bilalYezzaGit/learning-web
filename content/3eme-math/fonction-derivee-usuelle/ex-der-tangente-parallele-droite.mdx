:::enonce
Soit $f$ la fonction definie sur $\mathbb{R}$ par $f(x) = x^3 - 6x^2 + 9x + 1$.

1. Calculer $f'(x)$ et factoriser.
2. Dresser le tableau de variation de $f$.
3. Determiner les points de la courbe $\mathcal{C}_f$ ou la tangente est parallele a la droite $D : y = -3x + 5$.
4. Ecrire les equations de ces tangentes.
:::

:::solution
**1.** $f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x - 1)(x - 3)$.

**2.** $f'(x) = 0 \Leftrightarrow x = 1$ ou $x = 3$.

$f(1) = 1 - 6 + 9 + 1 = 5$ et $f(3) = 27 - 54 + 27 + 1 = 1$.

**3.** La pente de $D$ est $-3$. La tangente a $\mathcal{C}_f$ est parallele a $D$ lorsque $f'(x) = -3$ :

$$3x^2 - 12x + 9 = -3$$

$$3x^2 - 12x + 12 = 0 \quad \Leftrightarrow \quad x^2 - 4x + 4 = 0 \quad \Leftrightarrow \quad (x - 2)^2 = 0$$

Donc $x = 2$ est l'unique solution.

$f(2) = 8 - 24 + 18 + 1 = 3$. Le point est $(2, 3)$.

**4.** La tangente au point $(2, 3)$ a pour pente $f'(2) = -3$ :

$$y = -3(x - 2) + 3 = -3x + 9$$

Il y a une seule tangente parallele a $D$ : $y = -3x + 9$.
:::
