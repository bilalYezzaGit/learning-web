:::enonce
Soit $f$ la fonction definie par :

$$f(x) = \frac{2x^2 + 3x + 4}{x + 1}$$

1. Determiner le domaine de definition de $f$.
2. Montrer que $f(x) = 2x + 1 + \dfrac{3}{x + 1}$.
3. Calculer $f'(x)$ et etudier le signe de $f'$.
4. Dresser le tableau de variation de $f$ et preciser les extrema locaux.
:::

:::solution
**1.** $f$ est definie lorsque $x + 1 \neq 0$, donc $D_f = \mathbb{R} \setminus \{-1\}$.

**2.** On effectue la division euclidienne de $2x^2 + 3x + 4$ par $x + 1$ :

$2x^2 + 3x + 4 = (x + 1)(2x + 1) + 3$.

En effet : $(x+1)(2x+1) = 2x^2 + x + 2x + 1 = 2x^2 + 3x + 1$.

Donc $2x^2 + 3x + 4 = (x+1)(2x+1) + 3$, et :

$$f(x) = 2x + 1 + \frac{3}{x + 1}$$

**3.** $f(x) = 2x + 1 + 3(x+1)^{-1}$, donc :

$$f'(x) = 2 - \frac{3}{(x+1)^2} = \frac{2(x+1)^2 - 3}{(x+1)^2}$$

$f'(x) = 0 \Leftrightarrow 2(x+1)^2 = 3 \Leftrightarrow (x+1)^2 = \dfrac{3}{2} \Leftrightarrow x + 1 = \pm\sqrt{\dfrac{3}{2}}$.

Donc $x_1 = -1 - \sqrt{\dfrac{3}{2}}$ et $x_2 = -1 + \sqrt{\dfrac{3}{2}}$.

**4.** $f(x_2) = 2\left(-1 + \sqrt{\frac{3}{2}}\right) + 1 + \dfrac{3}{\sqrt{3/2}} = -2 + 2\sqrt{\frac{3}{2}} + 1 + \sqrt{6} = -1 + 2\sqrt{\frac{3}{2}} + \sqrt{6}$.

Simplifions : $\sqrt{\dfrac{3}{2}} = \dfrac{\sqrt{6}}{2}$, donc $2\sqrt{\dfrac{3}{2}} = \sqrt{6}$ et $\dfrac{3}{\sqrt{3/2}} = \dfrac{3 \cdot \sqrt{2}}{\sqrt{3}} = \sqrt{6}$.

$f(x_2) = -1 + \sqrt{6} + \sqrt{6} = -1 + 2\sqrt{6}$.

Par symetrie : $f(x_1) = -1 - 2\sqrt{6}$.
