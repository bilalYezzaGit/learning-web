:::enonce
Soit $f$ la fonction definie sur $\mathbb{R}$ par $f(x) = (x - 1)^2(x + 2)$.

1. Developper $f(x)$ et calculer $f'(x)$.
2. Factoriser $f'(x)$.
3. Dresser le tableau de variation de $f$. Preciser les extrema.
4. Determiner les points de la courbe ou la tangente est horizontale.
5. Discuter selon les valeurs de $k$ le nombre de solutions de l'equation $f(x) = k$.
:::

:::solution
**1.** $f(x) = (x-1)^2(x+2) = (x^2 - 2x + 1)(x + 2)$

$= x^3 + 2x^2 - 2x^2 - 4x + x + 2 = x^3 - 3x + 2$.

$f'(x) = 3x^2 - 3$.

**2.** $f'(x) = 3(x^2 - 1) = 3(x - 1)(x + 1)$.

**3.** $f'(x) = 0 \Leftrightarrow x = -1$ ou $x = 1$.

$f(-1) = (-1)^3 - 3(-1) + 2 = -1 + 3 + 2 = 4$.

$f(1) = 1 - 3 + 2 = 0$.

$f$ admet un **maximum local** en $x = -1$ egal a $4$ et un **minimum local** en $x = 1$ egal a $0$.

**4.** La tangente est horizontale lorsque $f'(x) = 0$, soit en $x = -1$ et $x = 1$.

Les points sont $(-1, 4)$ et $(1, 0)$.

**5.** D'apres le tableau de variation, le nombre de solutions de $f(x) = k$ :

- Si $k < 0$ : **1 solution** (la courbe ne coupe $y = k$ qu'une fois sur $]-\infty, -1[$).
- Si $k = 0$ : **2 solutions** ($x = 1$ est un minimum local et une racine double, et il y a une autre solution $x = -2$).
- Si $0 < k < 4$ : **3 solutions**.
- Si $k = 4$ : **2 solutions** ($x = -1$ est un maximum local, et il y a une autre solution a droite).
- Si $k > 4$ : **1 solution**.
:::
