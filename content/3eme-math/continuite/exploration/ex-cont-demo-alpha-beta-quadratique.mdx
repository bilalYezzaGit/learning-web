:::enonce
Soit $f$ la fonction definie sur $\mathbb{R}$ par $f(x) = x^2 + x$.

1. Montrer, en utilisant la definition de la continuite ($\alpha$-$\beta$), que $f$ est continue en $a = 1$.
2. Generaliser : montrer que $f$ est continue en tout reel $a$.
:::

:::solution
**1.** On veut montrer : pour tout $\beta > 0$, il existe $\alpha > 0$ tel que $|x - 1| < \alpha \Rightarrow |f(x) - f(1)| < \beta$.

**Calcul de $|f(x) - f(1)|$ :**

$$f(1) = 1 + 1 = 2$$

$$|f(x) - f(1)| = |x^2 + x - 2| = |(x-1)(x+2)| = |x - 1| \cdot |x + 2|$$

**Majoration de $|x + 2|$ :** on se restreint a $|x - 1| < 1$, c'est-a-dire $x \in {]0, 2[}$. Dans ce cas :

$$|x + 2| < |2 + 2| = 4$$

Donc, sous la restriction $|x - 1| < 1$ :

$$|f(x) - f(1)| < 4|x - 1|$$

**Choix de $\alpha$ :** soit $\beta > 0$. On pose :

$$\alpha = \min\!\left(1,\; \frac{\beta}{4}\right)$$

Alors, si $|x - 1| < \alpha$ :
- On a $|x - 1| < 1$ (restriction valide)
- On a $|f(x) - f(1)| < 4|x - 1| < 4 \cdot \dfrac{\beta}{4} = \beta$

Conclusion : $f$ est continue en $1$.

---

**2.** Soit $a \in \mathbb{R}$. On veut montrer que pour tout $\beta > 0$, il existe $\alpha > 0$ tel que $|x - a| < \alpha \Rightarrow |f(x) - f(a)| < \beta$.

**Calcul de $|f(x) - f(a)|$ :**

$$f(a) = a^2 + a$$

$$|f(x) - f(a)| = |x^2 + x - a^2 - a| = |(x^2 - a^2) + (x - a)|$$

$$= |(x - a)(x + a) + (x - a)| = |x - a| \cdot |x + a + 1|$$

**Majoration de $|x + a + 1|$ :** on se restreint a $|x - a| < 1$, c'est-a-dire $x \in {]a - 1, a + 1[}$. Dans ce cas :

$$|x + a + 1| \leq |x - a| + |2a + 1| < 1 + |2a + 1|$$

Posons $M = 1 + |2a + 1|$. On a :

$$|f(x) - f(a)| < M \cdot |x - a|$$

**Choix de $\alpha$ :** soit $\beta > 0$. On pose :

$$\alpha = \min\!\left(1,\; \frac{\beta}{M}\right) = \min\!\left(1,\; \frac{\beta}{1 + |2a + 1|}\right)$$

Alors, si $|x - a| < \alpha$ :

$$|f(x) - f(a)| < M \cdot \frac{\beta}{M} = \beta$$

Conclusion : $f$ est continue en tout reel $a$, donc $f$ est continue sur $\mathbb{R}$.
:::

:::methode
Pour une demonstration $\alpha$-$\beta$ avec une fonction quadratique :
1. Ecrire $|f(x) - f(a)|$ et factoriser $|x - a|$
2. Restreindre $|x - a| < 1$ pour borner le facteur restant par une constante $M$
3. Choisir $\alpha = \min(1,\; \beta / M)$
:::

:::erreurs
- Oublier la restriction $|x - a| < 1$ : sans cette restriction, le facteur $|x + a + 1|$ n'est pas borne et on ne peut pas conclure.
- Ecrire $\alpha = \beta / M$ sans le $\min(1, \ldots)$ : il faut que la restriction $|x - a| < 1$ soit garantie.
:::
