:::question
Soit $f$ une fonction **continue** sur $[0, 4]$ dont le tableau de variation est :

- $f$ croissante sur $[0, 2]$, de $f(0) = -1$ a $f(2) = 3$
- $f$ decroissante sur $[2, 4]$, de $f(2) = 3$ a $f(4) = 0$

L'image $f([0, 4])$ est :
:::

:::option{correct}
$[-1, 3]$
:::

:::option
$[-1, 0]$
:::

:::option
$[0, 3]$
:::

:::option
$\{-1, 0, 3\}$
:::

:::explanation
D'apres le tableau de variation :

- Sur $[0, 2]$, $f$ est croissante de $-1$ a $3$, donc $f([0, 2]) = [-1, 3]$.
- Sur $[2, 4]$, $f$ est decroissante de $3$ a $0$, donc $f([2, 4]) = [0, 3]$.

L'image totale est la reunion :

$$f([0, 4]) = f([0, 2]) \cup f([2, 4]) = [-1, 3] \cup [0, 3] = [-1, 3]$$

**Analyse des erreurs :**
- $[-1, 0]$ : confusion entre image et ensemble des valeurs aux bornes seulement.
- $[0, 3]$ : oubli de la valeur minimale $f(0) = -1$ (on ne regarde que le deuxieme morceau).
- $\{-1, 0, 3\}$ : confusion entre image (intervalle) et ensemble des valeurs remarquables (ensemble fini). L'image d'un intervalle par une fonction continue est toujours un **intervalle**.
:::
