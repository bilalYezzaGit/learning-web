:::question
La fonction $f(x) = \sqrt{|x - 1|}$ est continue sur :
:::

:::option{correct}
$\mathbb{R}$
:::
:::option
$\mathbb{R} \setminus \{1\}$
:::
:::option
$[1, +\infty[$
:::

:::explanation
La fonction $x \mapsto |x - 1|$ est continue sur $\mathbb{R}$ et positive. Donc $f(x) = \sqrt{|x - 1|}$ est continue sur $D_f = \mathbb{R}$.
:::
