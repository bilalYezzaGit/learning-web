:::question
On veut montrer que $f(x) = x^3$ est continue en $a = 1$ par la definition ($\alpha$-$\beta$).

On ecrit : $|x^3 - 1| = |x - 1| \cdot |x^2 + x + 1|$.

En se restreignant a $|x - 1| < 1$ (c'est-a-dire $x \in {]0, 2[}$), quelle est la **meilleure majoration** de $|x^2 + x + 1|$ ?
:::

:::option
$|x^2 + x + 1| < 3$
:::

:::option
$|x^2 + x + 1| < 5$
:::

:::option{correct}
$|x^2 + x + 1| < 7$
:::

:::option
$|x^2 + x + 1| < 10$
:::

:::explanation
Sous la restriction $|x - 1| < 1$, on a $x \in {]0, 2[}$, donc $0 < x < 2$.

Le trinome $g(x) = x^2 + x + 1$ est croissant sur $]0, 2[$ (car $g'(x) = 2x + 1 > 0$).

La valeur maximale est atteinte quand $x$ s'approche de $2$ :

$$g(2) = 4 + 2 + 1 = 7$$

Donc $|x^2 + x + 1| < 7$ est la majoration correcte.

- $< 3$ est faux : $g(1) = 3$ deja, et $g$ depasse $3$ sur $]1, 2[$.
- $< 5$ est insuffisant : $g(1{,}5) = 2{,}25 + 1{,}5 + 1 = 4{,}75$ et $g(1{,}8) = 6{,}04$.
- $< 10$ est vrai mais pas optimal : on choisit toujours la majoration la plus fine possible.

Avec cette majoration, on choisit $\alpha = \min\!\left(1,\; \dfrac{\beta}{7}\right)$ pour conclure.
:::
