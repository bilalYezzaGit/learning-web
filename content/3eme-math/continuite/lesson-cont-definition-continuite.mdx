### Intuition : tracer sans lever le crayon

Observe la courbe d'une fonction $f$. Si tu peux la tracer **sans lever le crayon** au voisinage d'un point $a$, alors $f$ est **continue en $a$**.

A l'inverse, si la courbe presente un **saut** en $a$ (un trou, un point isole, ou un changement brusque), alors $f$ n'est **pas continue en $a$**.

### Definition formelle

:::definition[Continuite en un point]
Soit $f$ une fonction definie sur un intervalle ouvert $I$ et $a \in I$.

$f$ est **continue en $a$** si : pour tout $\beta > 0$, il existe $\alpha > 0$ tel que :

$$|x - a| < \alpha \implies |f(x) - f(a)| < \beta$$
:::

:::remark
La convention tunisienne utilise $\alpha$ et $\beta$ (et non $\varepsilon$ et $\delta$). L'idee reste la meme : en prenant $x$ suffisamment proche de $a$, on peut rendre $f(x)$ aussi proche de $f(a)$ que l'on veut.
:::

### Contre-exemple : la partie entiere

La fonction **partie entiere** $E(x)$ est definie par : $E(x) = n$ si $n \leqslant x < n + 1$.

En tout entier $n$, on observe un **saut** :
- $\lim_{x \to n^-} E(x) = n - 1$
- $\lim_{x \to n^+} E(x) = n$
- $E(n) = n$

Les limites a gauche et a droite sont differentes, donc $E$ est **discontinue en tout entier**.

:::attention[Piege classique]
$E(x)$ est continue entre les entiers (sur chaque intervalle $]n, n+1[$), mais **discontinue en chaque entier $n$**.
:::

### Fonctions usuelles continues

:::property[Continuite des fonctions usuelles]
Les fonctions suivantes sont continues sur leur domaine de definition :

| Fonction | Domaine de continuite |
|---|---|
| Constante $f(x) = c$ | $\mathbb{R}$ |
| Identite $f(x) = x$ | $\mathbb{R}$ |
| Affine $f(x) = ax + b$ | $\mathbb{R}$ |
| Polynome $f(x) = a_n x^n + \cdots + a_0$ | $\mathbb{R}$ |
| Rationnelle $f(x) = \dfrac{P(x)}{Q(x)}$ | $\{x \in \mathbb{R} : Q(x) \neq 0\}$ |
| Racine carree $f(x) = \sqrt{x}$ | $\mathbb{R}^{+*}$ |
| Valeur absolue $f(x) = \lvert x \rvert$ | $\mathbb{R}$ |
:::

:::remark
Ces resultats sont admis. Ils servent de **briques de base** pour justifier la continuite de fonctions plus complexes (voir la lecon sur les operations).
:::

:::example[Exemple]
- $f(x) = 3x^4 - 2x + 7$ est un polynome, donc $f$ est **continue sur $\mathbb{R}$**.
- $g(x) = \dfrac{x - 1}{x^2 + 3}$ est une fonction rationnelle. Comme $x^2 + 3 > 0$ pour tout $x \in \mathbb{R}$, $g$ est **continue sur $\mathbb{R}$**.
- $h(x) = \sqrt{x}$ est continue sur $\mathbb{R}^{+*}$, mais on ne peut pas affirmer qu'elle est continue en tout reel.
:::
