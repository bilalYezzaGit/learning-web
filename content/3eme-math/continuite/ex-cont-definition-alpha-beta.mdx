:::enonce
Montrer par la definition ($\alpha$-$\beta$) la continuite des fonctions suivantes au point indique.

**a)** $f(x) = 3x - 2$ en $a = 1$

**b)** $f(x) = x^2$ en $a = 2$
:::

:::solution
**a)** Il faut montrer : pour tout $\beta > 0$, il existe $\alpha > 0$ tel que $|x - 1| < \alpha \implies |f(x) - f(1)| < \beta$.

On calcule :

$$|f(x) - f(1)| = |(3x - 2) - (3 \times 1 - 2)| = |3x - 2 - 1| = |3x - 3| = 3|x - 1|$$

On veut $3|x - 1| < \beta$, soit $|x - 1| < \dfrac{\beta}{3}$.

On choisit $\alpha = \dfrac{\beta}{3}$.

**Verification** : si $|x - 1| < \alpha = \dfrac{\beta}{3}$, alors :

$$|f(x) - f(1)| = 3|x - 1| < 3 \times \frac{\beta}{3} = \beta$$

Donc $f$ est **continue en $a = 1$** par la definition.

**b)** Il faut montrer : pour tout $\beta > 0$, il existe $\alpha > 0$ tel que $|x - 2| < \alpha \implies |f(x) - f(2)| < \beta$.

On calcule :

$$|f(x) - f(2)| = |x^2 - 4| = |x - 2| \cdot |x + 2|$$

On a besoin de majorer $|x + 2|$. On se place au voisinage de $2$ : supposons $|x - 2| < 1$, ce qui donne $1 < x < 3$, donc $3 < x + 2 < 5$, d'ou $|x + 2| < 5$.

Dans ce cas :

$$|f(x) - f(2)| = |x - 2| \cdot |x + 2| < 5|x - 2|$$

On veut $5|x - 2| < \beta$, soit $|x - 2| < \dfrac{\beta}{5}$.

On choisit $\alpha = \min\!\left(1{,}\ \dfrac{\beta}{5}\right)$.

**Verification** : si $|x - 2| < \alpha = \min\!\left(1{,}\ \dfrac{\beta}{5}\right)$, alors :
- $|x - 2| < 1$, donc $|x + 2| < 5$
- $|x - 2| < \dfrac{\beta}{5}$

D'ou :

$$|f(x) - f(2)| = |x - 2| \cdot |x + 2| < \frac{\beta}{5} \times 5 = \beta$$

Donc $f$ est **continue en $a = 2$** par la definition.
:::
