:::enonce
Dans chacun des cas suivants, justifier la continuite de la fonction $f$ en $a$.

1. $f(x) = \sqrt{3}\,x^8 - \sqrt{2}\,x^3 + 3x + 1$ ; $a = -\sqrt{3}$.
2. $f(x) = \dfrac{2x^5 - 2x + 1}{x^3 - x + 1}$ ; $a = 0{,}2$.
3. $f(x) = \sqrt{6}(2x - 5)^3$ ; $a = -2{,}8$.
4. $f(x) = 2x^4 - 3x(x + 1) + 5x^5 - 4$ ; $a = -\sqrt{3}$.
5. $f(x) = \dfrac{x}{10^2} - 10x^4 + \sqrt{3}\,x - 1$ ; $a = -1000500$.
6. $f(x) = \dfrac{-10^{-3}x^4 + 2}{3x + 10}$ ; $a = -\dfrac{1}{2}$.
7. $f(x) = \dfrac{x - 5}{x^4 + 10x^2 + 3}$ ; $a = -0{,}000251$.
:::

:::solution
1. $f(x) = \sqrt{3}\,x^8 - \sqrt{2}\,x^3 + 3x + 1$. $f$ est une fonction polynome, elle est donc continue en tout reel, en particulier en $a = -\sqrt{3}$.

2. $f(x) = \dfrac{2x^5 - 2x + 1}{x^3 - x + 1}$. $f$ est une fonction rationnelle. $(0{,}2)^3 - (0{,}2) + 1 \neq 0$ donc $0{,}2 \in D_f$. Donc $f$ est continue en $a = 0{,}2$.

3. $f(x) = \sqrt{6}(2x - 5)^3$. Comme etant fonction polynome, $f$ est continue en $a = -2{,}8$.

4. De meme, fonction polynome, $f$ est continue en $a$.

5. De meme, fonction polynome, $f$ est continue en $a$.

6. $f(x) = \dfrac{-10^{-3}x^4 + 2}{3x + 10}$. $D_f = \mathbb{R} \setminus \left\{-\dfrac{10}{3}\right\}$. $f$ est une fonction rationnelle et $\left(-\dfrac{1}{2}\right) \in D_f$. Donc $f$ est continue en $a = -\dfrac{1}{2}$.

7. $f(x) = \dfrac{x - 5}{x^4 + 10x^2 + 3}$. $f$ est une fonction rationnelle et $(-0{,}000251)^4 + 10(-0{,}000251)^2 + 3 \neq 0$ (somme de trois reels positifs). Donc $a \in D_f$ et $f$ est continue en $a$.
:::
