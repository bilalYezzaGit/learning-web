:::question
La fonction $f(x) = \dfrac{x^2 + 3x - 5}{x - 1}$ est continue sur :
:::

:::option
$]-\infty, 1[$
:::
:::option{correct}
$\mathbb{R} \setminus \{1\}$
:::
:::option
$]1, +\infty[$
:::

:::explanation
$f$ est une fonction rationnelle donc continue sur son domaine de definition $D_f = \mathbb{R} \setminus \{1\}$.
:::
