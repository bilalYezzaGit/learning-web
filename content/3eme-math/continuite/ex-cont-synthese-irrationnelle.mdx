:::enonce
Soit $f(x) = \sqrt{x + 2} - x$.

**a)** Determiner l'ensemble de definition $D_f$ de $f$.

**b)** Justifier que $f$ est continue sur $D_f$.

**c)** Calculer $f'(x)$ et etudier son signe. En deduire le tableau de variation de $f$.

**d)** Montrer que l'equation $f(x) = 0$ admet exactement une solution dans $D_f$. La determiner.

**e)** Determiner $f([-2{,}\ +\infty[)$.
:::

:::methode
Pour une fonction irrationnelle $f(x) = \sqrt{u(x)} - v(x)$, la strategie d'etude est :
1. Domaine : $u(x) \geq 0$.
2. Continuite : composee/somme de fonctions continues.
3. Derivee : $f'(x) = \dfrac{u'(x)}{2\sqrt{u(x)}} - v'(x)$, puis signe par mise au meme denominateur.
4. Pour les zeros : resoudre $\sqrt{u(x)} = v(x)$ impose $v(x) \geq 0$, puis elever au carre.
:::

:::solution
**a)** La fonction $f$ est definie si et seulement si $x + 2 \geq 0$, soit $x \geq -2$.

$$D_f = [-2{,}\ +\infty[$$

---

**b)** Sur $]{-}2{,}\ +\infty[$ :
- $x \mapsto x + 2$ est continue (fonction affine)
- $t \mapsto \sqrt{t}$ est continue sur $]0{,}\ +\infty[$
- par composition, $x \mapsto \sqrt{x + 2}$ est continue sur $]{-}2{,}\ +\infty[$
- $x \mapsto x$ est continue
- par difference, $f(x) = \sqrt{x + 2} - x$ est **continue** sur $]{-}2{,}\ +\infty[$

En $x = -2$ : $f(-2) = \sqrt{0} - (-2) = 2$ et $\displaystyle\lim_{x \to -2^+} f(x) = 0 + 2 = 2$, donc $f$ est continue a droite en $-2$.

Ainsi $f$ est **continue sur $[-2{,}\ +\infty[$**.

---

**c)** Pour $x > -2$ :

$$f'(x) = \frac{1}{2\sqrt{x + 2}} - 1$$

On etudie le signe de $f'(x)$ :

$$f'(x) = 0 \iff \frac{1}{2\sqrt{x + 2}} = 1 \iff 2\sqrt{x + 2} = 1 \iff \sqrt{x + 2} = \frac{1}{2}$$

$$\iff x + 2 = \frac{1}{4} \iff x = -\frac{7}{4}$$

Signe de $f'(x)$ :
- Pour $x \in {]{-}2{,}\ -\frac{7}{4}[}$ : $\sqrt{x+2} < \dfrac{1}{2}$, donc $\dfrac{1}{2\sqrt{x+2}} > 1$, d'ou $f'(x) > 0$.
- Pour $x \in {]{-}\frac{7}{4}{,}\ +\infty[}$ : $\sqrt{x+2} > \dfrac{1}{2}$, donc $\dfrac{1}{2\sqrt{x+2}} < 1$, d'ou $f'(x) < 0$.

Valeurs aux bornes et a l'extremum :
- $f(-2) = \sqrt{0} + 2 = 2$
- $f\!\left(-\dfrac{7}{4}\right) = \sqrt{-\dfrac{7}{4} + 2} - \left(-\dfrac{7}{4}\right) = \sqrt{\dfrac{1}{4}} + \dfrac{7}{4} = \dfrac{1}{2} + \dfrac{7}{4} = \dfrac{9}{4}$
- $\displaystyle\lim_{x \to +\infty} f(x) = \lim_{x \to +\infty} (\sqrt{x+2} - x) = -\infty$ (car $x$ domine $\sqrt{x+2}$)

Tableau de variation :

| $x$ | $-2$ | | $-\dfrac{7}{4}$ | | $+\infty$ |
|-----|------|---|-----------------|---|-----------|
| $f'(x)$ | | $+$ | $0$ | $-$ | |
| $f(x)$ | $2$ | $\nearrow$ | $\dfrac{9}{4}$ | $\searrow$ | $-\infty$ |

---

**d)** On cherche les solutions de $f(x) = 0$, c'est-a-dire $\sqrt{x + 2} = x$.

**Condition necessaire** : $x \geq 0$ (car $\sqrt{x+2} \geq 0$).

Sous cette condition, on eleve au carre :

$$x + 2 = x^2 \iff x^2 - x - 2 = 0 \iff (x - 2)(x + 1) = 0$$

Les solutions de l'equation algebrique sont $x = 2$ et $x = -1$.

**Verification** (car l'elevation au carre peut introduire des solutions parasites) :
- $x = 2$ : $f(2) = \sqrt{4} - 2 = 2 - 2 = 0$ **--- solution valide.**
- $x = -1$ : ne verifie pas $x \geq 0$, donc **rejetee**.

On peut aussi verifier directement : $f(-1) = \sqrt{1} - (-1) = 1 + 1 = 2 \neq 0$.

**Par le tableau de variation** : $f$ est decroissante sur $\left[-\dfrac{7}{4}{,}\ +\infty\right[$ de $\dfrac{9}{4}$ a $-\infty$. Comme $0 \in \left]{-}\infty{,}\ \dfrac{9}{4}\right[$, le TVI applique a $f$ strictement decroissante garantit l'existence et l'unicite d'un zero, qui est $x = 2$.

Sur $\left[-2{,}\ -\dfrac{7}{4}\right]$, $f$ est croissante de $2$ a $\dfrac{9}{4}$, donc $f(x) \geq 2 > 0$ : aucun zero.

L'equation $f(x) = 0$ admet **exactement une solution : $x = 2$**.

---

**e)** D'apres le tableau de variation :

- $f$ est continue sur $[-2{,}\ +\infty[$
- $f$ croit de $2$ a $\dfrac{9}{4}$ sur $\left[-2{,}\ -\dfrac{7}{4}\right]$, puis decroit de $\dfrac{9}{4}$ a $-\infty$ sur $\left[-\dfrac{7}{4}{,}\ +\infty\right[$

Le maximum de $f$ est $\dfrac{9}{4}$, atteint en $x = -\dfrac{7}{4}$.

La fonction $f$ prend toutes les valeurs de $]{-}\infty{,}\ \dfrac{9}{4}]$ sur $\left[-\dfrac{7}{4}{,}\ +\infty\right[$ (decroissante de $\dfrac{9}{4}$ a $-\infty$).

Donc :

$$f([-2{,}\ +\infty[) = \left]{-}\infty{,}\ \frac{9}{4}\right]$$
:::

:::erreurs
- **Oublier de verifier $x \geq 0$ avant d'elever au carre** : l'equation $\sqrt{x+2} = x$ impose $x \geq 0$. Sans cette verification, on garde la fausse solution $x = -1$.
- **Ecrire $f(-1) = 0$** : attention, $f(-1) = \sqrt{1} - (-1) = 1 + 1 = 2$, pas $0$.
- **Se tromper dans le calcul de $f\!\left(-\frac{7}{4}\right)$** : le maximum est $\dfrac{9}{4}$, pas $\dfrac{7}{4}$ ou $2$.
:::
