:::question
Dans un repere, $C_f$, $C_g$ et $C_k$ sont les courbes representatives de trois fonctions $f$, $g$ et $k$. La courbe de $f$ est continue, celle de $g$ presente un saut en $a$, et celle de $k$ est continue. Une seulement parmi ces fonctions est discontinue en $a$, laquelle ?

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-0.5, 0), (5.5, 0), stroke: 0.8pt, mark: (end: ">", fill: black))
  line((0, -0.5), (0, 3), stroke: 0.8pt, mark: (end: ">", fill: black))

  // Courbe C_f : continue, lisse, passe par a sans rupture
  bezier((0.3, 0.8), (5, 2.2), (1.5, 2.5), (3.5, 0.8), stroke: 1.5pt + blue)

  // Point plein en a (continuite)
  circle((2.5, 1.65), radius: 0.06, fill: blue, stroke: 1pt + blue)

  // Projection de a
  line((2.5, 0), (2.5, 1.65), stroke: (dash: "dashed", paint: gray))

  // Labels
  content((2.5, -0.3), $a$)
  content((-0.3, -0.3), $O$)
  content((5, 2.5), text(fill: blue)[$C_f$])
})
```

  bezier((0.3, 0.5), (2.5, 2.5), (1.2, 0.6), (2, 2.2), stroke: 1.5pt + green.darken(20%))
  bezier((2.5, 2.5), (5, 1), (3, 2.2), (4, 1.2), stroke: 1.5pt + green.darken(20%))

  // Point plein en a (continuite malgre la pointe)
  circle((2.5, 2.5), radius: 0.06, fill: green.darken(20%), stroke: 1pt + green.darken(20%))

  // Projection de a
  line((2.5, 0), (2.5, 2.5), stroke: (dash: "dashed", paint: gray))

  // Labels
  content((2.5, -0.3), $a$)

  content((5, 1.3), text(fill: green.darken(20%))[$C_k$])

:::

:::option
$f$
:::
:::option{correct}
$g$
:::
:::option
$k$
:::

:::explanation
La fonction $g$ est discontinue en $a$ car sa courbe representative presente un saut au point d'abscisse $a$ : les valeurs a gauche et a droite de $a$ ne se rejoignent pas en $g(a)$.
:::
