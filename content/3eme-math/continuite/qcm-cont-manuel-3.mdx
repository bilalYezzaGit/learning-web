:::question
Dans un repere, $C_f$ est la courbe representative de la fonction $f$. La courbe arrive par la gauche en $(-1, 1{,}5)$ et repart par la droite en $(-1, 2)$. Alors $f$ est continue :

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-3.5, 0), (2.5, 0), stroke: 0.8pt, mark: (end: ">", fill: black))
  line((0, -0.5), (0, 3.2), stroke: 0.8pt, mark: (end: ">", fill: black))

  // Branche gauche : arrive en (-1, 1.5), point ouvert
  bezier((-3, 0.5), (-1, 1.5), (-2.2, 0.8), (-1.5, 1.3), stroke: 1.5pt + blue)
  circle((-1, 1.5), radius: 0.06, fill: white, stroke: 1pt + blue)

  // Branche droite : repart de (-1, 2), point plein
  bezier((-1, 2), (2, 2.5), (-0.3, 2.1), (1.2, 2.4), stroke: 1.5pt + blue)
  circle((-1, 2), radius: 0.06, fill: blue, stroke: 1pt + blue)

  // Projections horizontales
  line((-1, 1.5), (0, 1.5), stroke: (dash: "dashed", paint: gray))
  line((-1, 2), (0, 2), stroke: (dash: "dashed", paint: gray))

  // Projection verticale
  line((-1, 0), (-1, 1.5), stroke: (dash: "dashed", paint: gray))

  // Labels
  content((-1, -0.3), $-1$)
  content((0.35, 1.5), $1{,}5$)
  content((0.25, 2), $2$)
  content((-0.25, -0.3), $O$)
  content((2, 2.8), text(fill: blue)[$C_f$])

  // Vecteurs unitaires
  line((0, 0), (1, 0), stroke: 1pt, mark: (end: ">", fill: black))
  content((0.5, -0.3), $arrow(i)$)
  line((0, 0), (0, 1), stroke: 1pt, mark: (end: ">", fill: black))
  content((-0.3, 0.5), $arrow(j)$)
})
```
:::

:::option
en $-1$
:::
:::option{correct}
a droite en $-1$
:::
:::option
a gauche en $-1$
:::

:::explanation
Contre-exemple : $a = -1$. La courbe montre que $f(-1) = 2$ (point plein a droite). Quand $x$ se rapproche de $-1$ par la droite, $f(x) \to 2 = f(-1)$, donc $f$ est continue a droite en $-1$. Mais par la gauche, $f(x) \to 1{,}5 \neq 2 = f(-1)$, donc $f$ n'est pas continue a gauche en $-1$, ni continue en $-1$.
:::
