:::question
Vrai ou Faux : $f(x) = x^3 - 3x^2 + x - 5$ est continue sur $\mathbb{R}$.
:::

:::option{correct}
Vrai
:::
:::option
Faux
:::

:::explanation
Vrai. $f$ est une fonction polynome donc continue sur $\mathbb{R}$.
:::
