:::enonce
$\zeta_1$, $\zeta_2$ et $\zeta_3$ sont les representations graphiques des trois fonctions $f$, $g$ et $h$ definies sur $[-1, 2]$.

  import draw: *

  // Tick marks
  line((-1, -0.1), (-1, 0.1), stroke: 0.5pt)
  line((1, -0.1), (1, 0.1), stroke: 0.5pt)
  line((2, -0.1), (2, 0.1), stroke: 0.5pt)
  line((-0.1, 1), (0.1, 1), stroke: 0.5pt)
  line((-0.1, 2), (0.1, 2), stroke: 0.5pt)
  line((-0.1, -1), (0.1, -1), stroke: 0.5pt)

  // Tick labels
  content((-1, -0.3), $-1$, anchor: "north")
  content((1, -0.3), $1$, anchor: "north")
  content((2, -0.3), $2$, anchor: "north")
  content((-0.3, 1), $1$, anchor: "east")
  content((-0.3, 2), $2$, anchor: "east")
  content((-0.3, -1), $-1$, anchor: "east")

  // Courbe f — continue, lisse
  bezier((-1, -1), (2, 0.5), (0, 2.5), (1.5, -0.5), stroke: 1.5pt + blue)

  // Filled dots aux extremites
  circle((-1, -1), radius: 0.07, fill: blue, stroke: 1pt + blue)
  circle((2, 0.5), radius: 0.07, fill: blue, stroke: 1pt + blue)

  // Label
  content((2.2, 1), $zeta_1$, anchor: "west")
})
```

  content((-0.3, 2), $2$, anchor: "east")
  content((-0.3, 3), $3$, anchor: "east")

  // Courbe h — continue, monte haut
  bezier((-1, 0), (2, 1), (0.5, 3.5), (1.5, 3), stroke: 1.5pt + green.darken(20%))

  // Filled dots
  circle((-1, 0), radius: 0.07, fill: green.darken(20%), stroke: 1pt + green.darken(20%))
  circle((2, 1), radius: 0.07, fill: green.darken(20%), stroke: 1pt + green.darken(20%))

  // Label
  content((2.2, 1.5), $zeta_3$, anchor: "west")

```

a. Dire dans chaque cas si la fonction est continue sur $[-1, 2]$.
b. Si la fonction n'est pas continue sur $[-1, 2]$, citer un intervalle sur lequel elle est continue.
:::

:::solution
a. Les courbes representatives de $f$ et $h$ sont tracees sans lever le crayon, donc $f$ et $h$ sont continues sur $[-1, 2]$.

$\zeta_g$ n'est pas continue en $1$ (il faut lever le crayon) donc $g$ n'est pas continue sur $[-1, 2]$.

b. $g$ est continue sur $[-1, 0]$ par exemple (ou sur $[1, 2]$).
:::
