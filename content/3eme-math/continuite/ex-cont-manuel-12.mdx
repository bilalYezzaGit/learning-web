:::enonce
Soit $f : x \mapsto -\dfrac{2}{x - 1}$.

1. Determiner l'ensemble de definition de $f$.
2. Tracer la courbe representative $C$ dans un repere orthonorme.
3. Quelles sont les images par $f$ des intervalles $I = [2, 4]$, $J = [-1, 0]$ et $K = ]-\infty, -3[$ ?
4. Determiner l'ensemble des antecedents par $f$ des reels de l'intervalle $]-\infty, 0[$.
:::

:::solution
1. $f$ est definie pour $x \neq 1$. Donc $D_f = \mathbb{R} \setminus \{1\}$.

2. $f$ est une hyperbole de centre $(1, 0)$. $f$ est croissante sur $]-\infty, 1[$ et sur $]1, +\infty[$.

3. - $f(I) = f([2, 4])$ : $f(2) = -2$, $f(4) = -\dfrac{2}{3}$. $f$ est croissante sur $]1, +\infty[$, donc $f([2, 4]) = [-2, -\dfrac{2}{3}]$.
- $f(J) = f([-1, 0])$ : $f(-1) = -\dfrac{2}{-2} = 1$, $f(0) = -\dfrac{2}{-1} = 2$. $f$ est croissante sur $]-\infty, 1[$, donc $f([-1, 0]) = [1, 2]$.
- $f(K) = f(]-\infty, -3[)$ : $f(-3) = -\dfrac{2}{-4} = \dfrac{1}{2}$ (non atteint), $\lim_{x \to -\infty} f(x) = 0$ (non atteint). Donc $f(]-\infty, -3[) = ]0, \dfrac{1}{2}[$.

4. $f(x) < 0$ : $-\dfrac{2}{x - 1} < 0 \Rightarrow \dfrac{2}{x - 1} > 0 \Rightarrow x - 1 > 0 \Rightarrow x > 1$. L'ensemble des antecedents est $]1, +\infty[$.
:::

:::methode
Pour une fonction homographique $\dfrac{a}{x - c} + d$ : identifier le centre de l'hyperbole, les variations (monotone sur chaque intervalle de continuite), puis determiner les images des intervalles.
:::
