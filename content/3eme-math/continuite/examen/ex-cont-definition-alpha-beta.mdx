:::enonce
En utilisant la definition de la continuite (avec $\alpha$ et $\beta$), montrer que les fonctions suivantes sont continues au point $a$ indique.

**a)** $f(x) = 3x - 2$ en $a = 1$

**b)** $f(x) = x^2$ en $a = 2$
:::

:::solution
**Rappel** : $f$ est continue en $a$ si, pour tout $\beta > 0$, il existe $\alpha > 0$ tel que :

$$|x - a| < \alpha \implies |f(x) - f(a)| < \beta$$

**a)** $f(x) = 3x - 2$ en $a = 1$.

On a $f(1) = 3 \times 1 - 2 = 1$.

Calculons $|f(x) - f(1)|$ :

$$|f(x) - f(1)| = |(3x - 2) - 1| = |3x - 3| = 3|x - 1|$$

On veut $|f(x) - f(1)| < \beta$, c'est-a-dire $3|x - 1| < \beta$, soit $|x - 1| < \dfrac{\beta}{3}$.

**Choix** : on prend $\alpha = \dfrac{\beta}{3}$.

Verification : si $|x - 1| < \alpha = \dfrac{\beta}{3}$, alors :

$$|f(x) - f(1)| = 3|x - 1| < 3 \times \frac{\beta}{3} = \beta$$

Donc $f$ est continue en $1$.

**b)** $f(x) = x^2$ en $a = 2$.

On a $f(2) = 4$.

Calculons $|f(x) - f(2)|$ :

$$|f(x) - 4| = |x^2 - 4| = |x - 2| \cdot |x + 2|$$

On a besoin de majorer $|x + 2|$. On se restreint a un voisinage de $2$ : supposons $|x - 2| < 1$.

Alors $1 < x < 3$, donc $3 < x + 2 < 5$, et ainsi $|x + 2| < 5$.

Dans ce cas :

$$|f(x) - 4| = |x - 2| \cdot |x + 2| < 5|x - 2|$$

On veut $5|x - 2| < \beta$, soit $|x - 2| < \dfrac{\beta}{5}$.

**Choix** : on prend $\alpha = \min\!\left(1,\ \dfrac{\beta}{5}\right)$.

Verification : si $|x - 2| < \alpha$, alors :
- $|x - 2| < 1$, donc $|x + 2| < 5$
- $|x - 2| < \dfrac{\beta}{5}$

Ainsi :

$$|f(x) - 4| = |x - 2| \cdot |x + 2| < \frac{\beta}{5} \times 5 = \beta$$

Donc $f$ est continue en $2$.
:::

:::methode
Pour montrer la continuite par la definition $(\alpha, \beta)$ :

1. Calculer $|f(x) - f(a)|$ et exprimer en fonction de $|x - a|$
2. Si necessaire, se restreindre a $|x - a| < 1$ pour majorer les termes restants
3. En deduire $|f(x) - f(a)| \leq K|x - a|$ pour une constante $K$
4. Choisir $\alpha = \min\!\left(1, \dfrac{\beta}{K}\right)$ (ou $\alpha = \dfrac{\beta}{K}$ si pas de restriction)
5. Verifier l'implication et conclure
:::
