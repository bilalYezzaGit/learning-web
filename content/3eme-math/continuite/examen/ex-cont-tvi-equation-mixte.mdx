:::enonce
Montrer que l'equation $\sqrt{x + 1} = 3 - x$ admet une unique solution dans $[0, 3]$.
:::

:::solution
On pose $h(x) = \sqrt{x + 1} - 3 + x$ de sorte que l'equation $\sqrt{x + 1} = 3 - x$ equivaut a $h(x) = 0$.

### Continuite de $h$ sur $[0, 3]$

- $x + 1$ est un polynome, continu sur $\mathbb{R}$ (T1). Pour tout $x \in [0, 3]$, $x + 1 \geq 1 > 0$, donc $\sqrt{x + 1}$ est continue sur $[0, 3]$ (T5).
- $-3 + x$ est un polynome, continu sur $\mathbb{R}$ (T1).
- Donc $h = \sqrt{x + 1} + x - 3$ est continue sur $[0, 3]$ comme somme de fonctions continues (T2).

### Existence (TVI)

$$h(0) = \sqrt{0 + 1} - 3 + 0 = 1 - 3 = -2 < 0$$

$$h(3) = \sqrt{3 + 1} - 3 + 3 = \sqrt{4} = 2 > 0$$

Comme $h$ est continue sur $[0, 3]$ et $h(0) \cdot h(3) = (-2) \times 2 = -4 < 0$, par le TVI, l'equation $h(x) = 0$ admet au moins une solution dans $]0, 3[$.

### Unicite (monotonie)

$h$ est derivable sur $]0, 3[$ et :

$$h'(x) = \frac{1}{2\sqrt{x + 1}} + 1$$

Pour tout $x \in [0, 3]$ : $\sqrt{x + 1} > 0$, donc $\dfrac{1}{2\sqrt{x + 1}} > 0$, et ainsi :

$$h'(x) = \frac{1}{2\sqrt{x + 1}} + 1 > 1 > 0$$

Donc $h$ est **strictement croissante** sur $[0, 3]$. Par T9, la solution est **unique**.

L'equation $\sqrt{x + 1} = 3 - x$ admet donc une unique solution dans $[0, 3]$.
:::
