:::enonce
Soit $f$ la fonction definie sur $\mathbb{R}$ par :

$$f(x) = \begin{cases} \dfrac{x^2 - 1}{x - 1} & \text{si } x < 1 \\ a & \text{si } x = 1 \\ \sqrt{x} + b & \text{si } x > 1 \end{cases}$$

ou $a$ et $b$ sont des constantes reelles.

**a)** Simplifier $f(x)$ pour $x < 1$.

**b)** Determiner les valeurs de $a$ et $b$ pour que $f$ soit continue en $1$.

**c)** Pour ces valeurs de $a$ et $b$, $f$ est-elle continue sur $\mathbb{R}$ ?
:::

:::solution
**a)** Pour $x < 1$, on a $x \neq 1$, donc on peut factoriser :

$$f(x) = \frac{x^2 - 1}{x - 1} = \frac{(x - 1)(x + 1)}{x - 1} = x + 1$$

Ainsi, pour $x < 1$ : $f(x) = x + 1$.

**b)** Pour que $f$ soit continue en $1$, il faut :

$$\lim_{x \to 1^-} f(x) = \lim_{x \to 1^+} f(x) = f(1)$$

**Limite a gauche** :

$$\lim_{x \to 1^-} f(x) = \lim_{x \to 1^-} (x + 1) = 1 + 1 = 2$$

**Valeur en $1$** : $f(1) = a$.

**Limite a droite** :

$$\lim_{x \to 1^+} f(x) = \lim_{x \to 1^+} (\sqrt{x} + b) = \sqrt{1} + b = 1 + b$$

La continuite en $1$ exige :

$$2 = a = 1 + b$$

Donc : $a = 2$ et $b = 1$.

**c)** Avec $a = 2$ et $b = 1$, on a :

$$f(x) = \begin{cases} x + 1 & \text{si } x < 1 \\ 2 & \text{si } x = 1 \\ \sqrt{x} + 1 & \text{si } x > 1 \end{cases}$$

- Sur $]{-}\infty, 1[$ : $f(x) = x + 1$ est un polynome, continu sur $\mathbb{R}$ (T1).
- Sur $]1, +\infty[$ : $\sqrt{x}$ est continue sur $]0, +\infty[$ (T1) et la constante $1$ est continue. Donc $f(x) = \sqrt{x} + 1$ est continue sur $]1, +\infty[$ (T2).
- En $1$ : on a montre que $f$ est continue en $1$.

Donc $f$ est continue sur $\mathbb{R}$.
:::
