:::enonce
On considere la fonction $f$ definie sur $\mathbb{R}$ par :

$$f(x) = \begin{cases} \sqrt{2x - 4} & \text{si } x \geq 2 \\ x & \text{si } x < 2 \end{cases}$$

1. Tracer la courbe representative de $f$ dans un repere.
2. Justifier la continuite de $f$ sur $[2, +\infty[$.
3. Justifier la continuite de $f$ sur $]-\infty, 2[$.
4. Verifier, a l'aide du graphique, que $f$ n'est pas continue sur $\mathbb{R}$.
:::

:::solution
1. Pour $x < 2$ : $f(x) = x$ (droite d'equation $y = x$). Pour $x \geq 2$ : $f(x) = \sqrt{2x - 4}$ (courbe de racine).

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({

  content((5.3, -0.4), $x$)
  content((-0.4, 3.3), $y$)
  content((-0.3, -0.3), $O$)

  // Graduations axe x
  for i in (-1, 1, 2, 3, 4, 5) {
    line((i, -0.1), (i, 0.1), stroke: 0.6pt)
    content((i, -0.4), text(size: 8pt, str(i)))
  }

  // Graduations axe y
  for i in (-1, 1, 2, 3) {
    line((-0.1, i), (0.1, i), stroke: 0.6pt)
    content((-0.4, i), text(size: 8pt, str(i)))
  }

  // Droite y = x pour x < 2
  line((-1, -1), (2, 2), stroke: 1.5pt + blue)

  // Open dot at (2, 2) — left limit
  circle((2, 2), radius: 0.08, fill: white, stroke: 1.5pt + blue)

  // Curve y = sqrt(2x - 4) for x >= 2
  // Points: (2, 0), (2.5, 1), (3, sqrt(2)≈1.41), (4, 2), (5, sqrt(6)≈2.45)
  bezier((2, 0), (5, 2.45), (3, 1.6), (4.2, 2.3), stroke: 1.5pt + blue)

  // Filled dot at (2, 0) — f(2)
  circle((2, 0), radius: 0.08, fill: blue, stroke: 1.5pt + blue)

  // Dashed vertical line at x = 2
  line((2, 0), (2, 2), stroke: (dash: "dashed", paint: gray))

  // Labels
  content((0.3, 1.5), text(size: 9pt, fill: blue, $y = x$))
  content((4.3, 1.2), text(size: 9pt, fill: blue, $y = sqrt(2x - 4)$))
```

2. Sur $[2, +\infty[$, $f(x) = \sqrt{2x - 4}$. La fonction $2x - 4$ est affine, positive sur $[2, +\infty[$, et continue. Par le theoreme de $\sqrt{f}$ (en incluant la continuite a droite en $2$), $f$ est continue sur $[2, +\infty[$.

3. Sur $]-\infty, 2[$, $f(x) = x$ est une fonction affine, donc continue sur $]-\infty, 2[$.

4. En $x = 2$ : $f(2) = \sqrt{2 \cdot 2 - 4} = 0$. Par la gauche : $\lim_{x \to 2^-} f(x) = \lim_{x \to 2^-} x = 2 \neq 0 = f(2)$. Donc $f$ n'est pas continue a gauche en $2$, donc $f$ n'est pas continue en $2$. La fonction n'est pas continue sur $\mathbb{R}$.
:::

:::methode
Pour une fonction definie par morceaux : etudier la continuite sur chaque morceau separement, puis verifier la continuite aux points de raccordement en comparant les limites a gauche et a droite avec la valeur de la fonction.
:::
