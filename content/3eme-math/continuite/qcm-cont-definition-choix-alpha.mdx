:::question
Soit $f(x) = 5x + 1$. Pour montrer la continuite de $f$ en $a = 0$ par la definition, quel $\alpha$ choisir en fonction de $\beta$ ?
:::

:::option
$\alpha = \beta$
:::

:::option{correct}
$\alpha = \dfrac{\beta}{5}$
:::

:::option
$\alpha = 5\beta$
:::

:::option
$\alpha = \beta^2$
:::

:::explanation
On calcule $|f(x) - f(0)|$ :

$$|f(x) - f(0)| = |(5x + 1) - 1| = |5x| = 5|x| = 5|x - 0|$$

On veut $|f(x) - f(0)| < \beta$, c'est-a-dire $5|x| < \beta$, soit $|x| < \dfrac{\beta}{5}$.

Donc le bon choix est $\alpha = \dfrac{\beta}{5}$.

- $\alpha = \beta$ ne suffit pas : $5|x| < 5\beta \neq \beta$ en general.
- $\alpha = 5\beta$ est trop grand : on obtiendrait $5|x| < 25\beta$.
- $\alpha = \beta^2$ n'a pas la bonne forme : la relation est lineaire, pas quadratique.
:::
