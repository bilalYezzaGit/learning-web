:::question
Parmi les affirmations suivantes, laquelle est **fausse** ?
:::

:::option
Toute fonction polynôme est continue sur $\mathbb{R}$
:::

:::option
La fonction $x \mapsto \dfrac{1}{x}$ est continue sur $\mathbb{R}^*$
:::

:::option{correct}
La fonction $x \mapsto \sqrt{x}$ est continue sur $\mathbb{R}$
:::

:::option
Toute fonction affine est continue sur $\mathbb{R}$
:::

:::explanation
La fonction $x \mapsto \sqrt{x}$ est continue sur $]0{,}\ +\infty[$ (et continue à droite en $0$), mais elle n'est **pas définie** sur $\mathbb{R}$ tout entier (pas définie pour $x < 0$). On ne peut donc pas dire qu'elle est continue sur $\mathbb{R}$.
:::
