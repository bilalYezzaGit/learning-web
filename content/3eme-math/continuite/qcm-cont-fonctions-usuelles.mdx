:::question
Parmi les affirmations suivantes, laquelle est **fausse** ?
:::

:::option
Toute fonction polynomiale est continue sur $\mathbb{R}$
:::

:::option
La fonction $f(x) = \dfrac{1}{x^2 + 1}$ est continue sur $\mathbb{R}$
:::

:::option{correct}
La fonction $f(x) = \sqrt{x}$ est continue sur $\mathbb{R}$
:::

:::option
La fonction $f(x) = |2x - 5|$ est continue sur $\mathbb{R}$
:::

:::explanation
La fonction $\sqrt{x}$ n'est definie que pour $x \geqslant 0$ et elle est continue sur $\mathbb{R}^{+*}$. Elle n'est **pas definie** sur $\mathbb{R}$ tout entier, donc on ne peut pas dire qu'elle est continue sur $\mathbb{R}$.

Les autres affirmations sont vraies :
- Un polynome est toujours continu sur $\mathbb{R}$.
- $\dfrac{1}{x^2 + 1}$ est une fonction rationnelle dont le denominateur ne s'annule jamais, donc elle est continue sur $\mathbb{R}$.
- $|2x - 5|$ est la valeur absolue d'un polynome, donc continue sur $\mathbb{R}$.
:::
