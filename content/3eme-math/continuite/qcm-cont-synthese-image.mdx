:::question
Soit $f$ une fonction continue sur $[0{,}\ 4]$ dont le tableau de variation est :

| $x$ | $0$ | | $2$ | | $4$ |
|-----|-----|---|------|---|------|
| $f(x)$ | $-1$ | $\nearrow$ | $3$ | $\searrow$ | $0$ |

($f$ croissante sur $[0{,}\ 2]$ de $-1$ a $3$, decroissante sur $[2{,}\ 4]$ de $3$ a $0$.)

Que vaut $f([0{,}\ 4])$ ?
:::

:::option
$[-1{,}\ 4]$
:::

:::option
$\{-1{,}\ 0{,}\ 3\}$
:::

:::option{correct}
$[-1{,}\ 3]$
:::

:::option
$[0{,}\ 3]$
:::

:::explanation
Pour determiner l'image d'un intervalle, on utilise le tableau de variation :

- Le **minimum** de $f$ sur $[0{,}\ 4]$ est $f(0) = -1$ (valeur la plus basse du tableau).
- Le **maximum** de $f$ sur $[0{,}\ 4]$ est $f(2) = 3$ (valeur la plus haute du tableau).

Comme $f$ est continue sur $[0{,}\ 4]$, par le theoreme des valeurs intermediaires, $f$ prend toutes les valeurs entre son minimum et son maximum.

Donc $f([0{,}\ 4]) = [-1{,}\ 3]$.

**Analyse des distracteurs** :
- $[-1{,}\ 4]$ : le $4$ est la borne de l'intervalle de depart, pas une valeur de $f$.
- $\{-1{,}\ 0{,}\ 3\}$ : ce sont les valeurs remarquables, mais $f$ prend toutes les valeurs intermediaires (pas seulement ces trois valeurs).
- $[0{,}\ 3]$ : le $0$ est $f(4)$, mais le minimum global est $f(0) = -1 < 0$.
:::
