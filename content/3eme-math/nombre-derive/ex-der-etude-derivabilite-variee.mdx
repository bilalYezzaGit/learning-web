:::enonce
Étudier la dérivabilité de chacune des fonctions suivantes en $a$ :

1. $f(x) = |x^2 - 9| \cdot (x + 3)$ en $a = 3$, $a = -3$ et $a \in {]{-3},\, 3[}$.
2. $g(x) = (x - 2)\sqrt{x - 2}$ en $a = 3$ et $a = 2$.
3. $h(x) = |x^2 - 4| + |x - 2|$ en $a = 2$, $a = -2$ et $a > 2$.
4. $f(x) = \dfrac{2x + 1}{x + 2}$ en $a = 0$ et $a = 1$.
5. $f(x) = \sqrt{x^2 - 9}$ en $a = 3$ et $a = 5$.
6. $f$ définie par $f(x) = \dfrac{3x^2 + x - 2}{x + 1}$ si $x \neq -1$ et $f(-1) = -5$, en $a = -1$ et $a = 1$.

Pour chaque cas, écrire le taux de variation, simplifier et calculer la limite.
:::

:::solution
**1. $f(x) = |x^2 - 9| \cdot (x + 3)$**

On factorise : $x^2 - 9 = (x - 3)(x + 3)$, donc $|x^2 - 9| = |x - 3| \cdot |x + 3|$.

- Pour $x > 3$ : $f(x) = (x - 3)(x + 3)^2$
- Pour $-3 < x < 3$ : $f(x) = (3 - x)(x + 3)^2$
- Pour $x < -3$ : $f(x) = (x - 3)(x + 3)^2$

**En $a = 3$ :** on a $f(3) = 0$.

$$f'_d(3) = \lim_{x \to 3^+} \frac{(x - 3)(x + 3)^2 - 0}{x - 3} = \lim_{x \to 3^+} (x + 3)^2 = 36$$

$$f'_g(3) = \lim_{x \to 3^-} \frac{(3 - x)(x + 3)^2 - 0}{x - 3} = \lim_{x \to 3^-} \frac{-(x - 3)(x + 3)^2}{x - 3} = \lim_{x \to 3^-} -(x + 3)^2 = -36$$

$f'_d(3) = 36 \neq -36 = f'_g(3)$. Donc $f$ n'est **pas dérivable en $3$** : **point anguleux**.

**En $a = -3$ :** on a $f(-3) = 0$.

- Pour $x > -3$ (au voisinage de $-3$) : $\dfrac{f(x) - f(-3)}{x - (-3)} = \dfrac{(3 - x)(x + 3)^2}{x + 3} = (3 - x)(x + 3)$, et $\lim_{x \to -3^+} (3 - x)(x + 3) = 6 \times 0 = 0$.

- Pour $x < -3$ : $\dfrac{f(x) - f(-3)}{x + 3} = \dfrac{(x - 3)(x + 3)^2}{x + 3} = (x - 3)(x + 3)$, et $\lim_{x \to -3^-} (x - 3)(x + 3) = (-6) \times 0 = 0$.

$f'_d(-3) = f'_g(-3) = 0$. Donc $f$ est **dérivable en $-3$** et $f'(-3) = 0$.

**Pour $a \in {]{-3},\, 3[}$ :** $f(x) = (3 - x)(x + 3)^2$ est une fonction polynomiale au voisinage de $a$, donc **dérivable**.

$f'(x) = -(x + 3)^2 + 2(3 - x)(x + 3) = (x + 3)[-(x + 3) + 2(3 - x)] = (x + 3)(3 - 3x)$

D'où $f'(a) = (a + 3)(3 - 3a) = -3(a + 3)(a - 1)$.

**2. $g(x) = (x - 2)\sqrt{x - 2}$**

Le domaine est $D_g = [2, +\infty[$. On peut écrire $g(x) = (x - 2)^{3/2}$.

**En $a = 3$ :** $g(3) = 1 \times 1 = 1$.

$$\frac{g(3 + h) - g(3)}{h} = \frac{(1 + h)^{3/2} - 1}{h}$$

On reconnaît la dérivée de $t \mapsto t^{3/2}$ en $t = 1$, qui vaut $\dfrac{3}{2} \times 1^{1/2} = \dfrac{3}{2}$.

Donc $g$ est **dérivable en $3$** et $g'(3) = \dfrac{3}{2}$.

**En $a = 2$ :** $g(2) = 0$. Pour $h > 0$ :

$$\frac{g(2 + h) - g(2)}{h} = \frac{h\sqrt{h}}{h} = \sqrt{h} \xrightarrow[h \to 0^+]{} 0$$

Donc $g$ est **dérivable à droite en $2$** (bord du domaine) et $g'_d(2) = 0$.

**3. $h(x) = |x^2 - 4| + |x - 2|$**

On a $x^2 - 4 = (x - 2)(x + 2)$ et $|x^2 - 4| = |x - 2| \cdot |x + 2|$.

Les points critiques sont $x = 2$ et $x = -2$.

**En $a = 2$ :** $h(2) = 0 + 0 = 0$.

- Pour $x > 2$ : $h(x) = (x^2 - 4) + (x - 2) = x^2 + x - 6$.

$$h'_d(2) = \lim_{x \to 2^+} \frac{x^2 + x - 6}{x - 2} = \lim_{x \to 2^+} \frac{(x - 2)(x + 3)}{x - 2} = \lim_{x \to 2^+} (x + 3) = 5$$

- Pour $-2 < x < 2$ : $h(x) = (4 - x^2) + (2 - x) = -x^2 - x + 6$.

$$h'_g(2) = \lim_{x \to 2^-} \frac{-x^2 - x + 6}{x - 2} = \lim_{x \to 2^-} \frac{-(x - 2)(x + 3)}{x - 2} = \lim_{x \to 2^-} -(x + 3) = -5$$

$h'_d(2) = 5 \neq -5 = h'_g(2)$. Donc $h$ n'est **pas dérivable en $2$** : **point anguleux**.

**En $a = -2$ :** $h(-2) = 0 + 4 = 4$.

- Pour $x > -2$ (au voisinage de $-2$, avec $x < 2$) : $h(x) = (4 - x^2) + (2 - x) = -x^2 - x + 6$.

$$h'_d(-2) = \lim_{x \to -2^+} \frac{-x^2 - x + 6 - 4}{x + 2} = \lim_{x \to -2^+} \frac{-x^2 - x + 2}{x + 2} = \lim_{x \to -2^+} \frac{-(x + 2)(x - 1)}{x + 2} = \lim_{x \to -2^+} -(x - 1) = 3$$

- Pour $x < -2$ : $h(x) = (x^2 - 4) + (2 - x) = x^2 - x - 2$.

$$h'_g(-2) = \lim_{x \to -2^-} \frac{x^2 - x - 2 - 4}{x + 2} = \lim_{x \to -2^-} \frac{x^2 - x - 6}{x + 2} = \lim_{x \to -2^-} \frac{(x + 2)(x - 3)}{x + 2} = \lim_{x \to -2^-} (x - 3) = -5$$

$h'_d(-2) = 3 \neq -5 = h'_g(-2)$. Donc $h$ n'est **pas dérivable en $-2$** : **point anguleux**.

**Pour $a > 2$ :** $h(x) = x^2 + x - 6$ est polynomiale au voisinage de $a$, donc **dérivable** et $h'(a) = 2a + 1$.

**4. $f(x) = \dfrac{2x + 1}{x + 2}$**

$f$ est une fonction rationnelle définie pour $x \neq -2$.

En appliquant la formule du quotient :

$$f'(x) = \frac{2(x + 2) - (2x + 1) \times 1}{(x + 2)^2} = \frac{2x + 4 - 2x - 1}{(x + 2)^2} = \frac{3}{(x + 2)^2}$$

**En $a = 0$ :** $f'(0) = \dfrac{3}{4}$. **Dérivable**.

**En $a = 1$ :** $f'(1) = \dfrac{3}{9} = \dfrac{1}{3}$. **Dérivable**.

**5. $f(x) = \sqrt{x^2 - 9}$**

Le domaine est $D_f = {]{-\infty},\, -3]} \cup [3, +\infty[$.

**En $a = 3$ :** $f(3) = 0$. Pour $x > 3$ :

$$\frac{f(x) - f(3)}{x - 3} = \frac{\sqrt{x^2 - 9}}{x - 3} = \frac{\sqrt{(x - 3)(x + 3)}}{x - 3} = \frac{\sqrt{x + 3}}{\sqrt{x - 3}}$$

Quand $x \to 3^+$ : $\sqrt{x + 3} \to \sqrt{6}$ et $\sqrt{x - 3} \to 0^+$, donc la limite vaut $+\infty$.

Donc $f$ n'est **pas dérivable en $3$** : **demi-tangente verticale** à droite.

**En $a = 5$ :** $f(5) = \sqrt{25 - 9} = 4$. On a $f(x) = \sqrt{x^2 - 9}$ avec $x^2 - 9 > 0$ au voisinage de $5$.

$$f'(5) = \frac{2 \times 5}{2\sqrt{25 - 9}} = \frac{10}{2 \times 4} = \frac{5}{4}$$

Donc $f$ est **dérivable en $5$** et $f'(5) = \dfrac{5}{4}$.

**6. $f(x) = \dfrac{3x^2 + x - 2}{x + 1}$ si $x \neq -1$, $f(-1) = -5$**

Factorisons : $3x^2 + x - 2 = (3x - 2)(x + 1)$.

Donc pour $x \neq -1$ : $f(x) = \dfrac{(3x - 2)(x + 1)}{x + 1} = 3x - 2$.

Vérifions la continuité en $-1$ : $\lim_{x \to -1} f(x) = \lim_{x \to -1} (3x - 2) = -5 = f(-1)$.

Donc $f$ est continue en $-1$ et $f(x) = 3x - 2$ pour tout $x$.

**En $a = -1$ :**

$$\frac{f(-1 + h) - f(-1)}{h} = \frac{3(-1 + h) - 2 - (-5)}{h} = \frac{-3 + 3h - 2 + 5}{h} = \frac{3h}{h} = 3$$

Donc $f$ est **dérivable en $-1$** et $f'(-1) = 3$.

**En $a = 1$ :** $f(1) = 3 - 2 = 1$. $f'(1) = 3$. **Dérivable**.
:::

:::methode
Pour étudier la dérivabilité d'une fonction en un point $a$ :

1. **Identifier le type de fonction** au voisinage de $a$ (valeur absolue, racine, quotient, morceaux)
2. **Écrire le taux de variation** $\dfrac{f(x) - f(a)}{x - a}$ ou $\dfrac{f(a + h) - f(a)}{h}$
3. **Simplifier** algébriquement (factoriser, rationaliser, simplifier)
4. **Calculer la limite** : si elle est finie, $f$ est dérivable ; si elle est infinie, demi-tangente verticale ; si les limites latérales diffèrent, point anguleux

Pour les fonctions avec valeur absolue : distinguer les cas selon le signe de l'expression sous la valeur absolue.
:::

:::erreurs
- Oublier de distinguer les dérivées à droite et à gauche pour les fonctions avec valeur absolue
- Pour $\sqrt{x^2 - 9}$ en $a = 3$ : appliquer la formule $\dfrac{x}{\sqrt{x^2 - 9}}$ donne $\dfrac{3}{0}$, ce qui confirme la non-dérivabilité mais ne remplace pas le calcul par la définition
- Pour la question **6.** : oublier de vérifier la continuité avant de conclure à la dérivabilité
:::
