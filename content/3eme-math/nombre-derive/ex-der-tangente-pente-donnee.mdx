:::enonce
Soit $f$ la fonction définie sur $\mathbb{R}$ par $f(x) = x^2 - 4x + 3$.

1. Soit $a$ un réel quelconque, calculer $f'(a)$.
2. Déterminer les abscisses des points de $\mathcal{C}_f$ où la tangente :
   a) est horizontale ;
   b) admet $-2$ pour coefficient directeur ;
   c) est parallèle à la droite $\Delta : y = \dfrac{2}{3}x - 1$.
:::

:::solution
**1. Calcul de $f'(a)$**

On utilise la définition du nombre dérivé :

$$f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

Calculons $f(a + h) - f(a)$ :

$$f(a + h) = (a + h)^2 - 4(a + h) + 3 = a^2 + 2ah + h^2 - 4a - 4h + 3$$

$$f(a + h) - f(a) = a^2 + 2ah + h^2 - 4a - 4h + 3 - (a^2 - 4a + 3)$$

$$= 2ah + h^2 - 4h$$

Le taux d'accroissement vaut :

$$\frac{f(a + h) - f(a)}{h} = \frac{2ah + h^2 - 4h}{h} = 2a + h - 4$$

$$f'(a) = \lim_{h \to 0} (2a + h - 4) = 2a - 4$$

Donc $f'(a) = 2a - 4$.

**2a. Tangente horizontale**

La tangente est horizontale lorsque sa pente est nulle :

$$f'(a) = 0 \Leftrightarrow 2a - 4 = 0 \Leftrightarrow a = 2$$

Calculons $f(2) = 4 - 8 + 3 = -1$.

La tangente est horizontale au point $(2, -1)$.

**2b. Tangente de coefficient directeur $-2$**

$$f'(a) = -2 \Leftrightarrow 2a - 4 = -2 \Leftrightarrow 2a = 2 \Leftrightarrow a = 1$$

Calculons $f(1) = 1 - 4 + 3 = 0$.

La tangente admet $-2$ pour coefficient directeur au point $(1, 0)$.

**2c. Tangente parallèle à $\Delta$**

La droite $\Delta : y = \dfrac{2}{3}x - 1$ a pour coefficient directeur $\dfrac{2}{3}$.

Deux droites sont parallèles si et seulement si elles ont le même coefficient directeur.

$$f'(a) = \frac{2}{3} \Leftrightarrow 2a - 4 = \frac{2}{3} \Leftrightarrow 2a = 4 + \frac{2}{3} = \frac{14}{3} \Leftrightarrow a = \frac{7}{3}$$

Calculons $f\!\left(\dfrac{7}{3}\right)$ :

$$f\!\left(\frac{7}{3}\right) = \left(\frac{7}{3}\right)^2 - 4 \times \frac{7}{3} + 3 = \frac{49}{9} - \frac{28}{3} + 3 = \frac{49}{9} - \frac{84}{9} + \frac{27}{9} = -\frac{8}{9}$$

La tangente est parallèle à $\Delta$ au point $\left(\dfrac{7}{3}, -\dfrac{8}{9}\right)$.
:::

:::methode
Pour trouver les points où la tangente a une propriété donnée :

1. **Calculer $f'(a)$** pour un réel $a$ quelconque (par la définition ou les formules de dérivation).
2. **Traduire la condition** en une équation sur $f'(a)$ :
   - Tangente horizontale : $f'(a) = 0$
   - Pente donnée $m$ : $f'(a) = m$
   - Parallèle à $y = mx + p$ : $f'(a) = m$
3. **Résoudre l'équation** pour trouver $a$, puis calculer $f(a)$ pour obtenir le point.
:::

:::erreurs
- **Confondre coefficient directeur et ordonnée à l'origine** : pour $\Delta : y = \dfrac{2}{3}x - 1$, le coefficient directeur est $\dfrac{2}{3}$ (et non $-1$).
- **Oublier de calculer l'ordonnée** : la question demande les abscisses, mais il est bon de donner aussi le point complet pour l'interprétation.
:::
