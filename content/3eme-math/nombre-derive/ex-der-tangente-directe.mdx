:::enonce
Determiner l'equation de la tangente a la courbe $C_f$ au point d'abscisse $a$ dans chacun des cas suivants :

**a)** $f(x) = 2x^3 - 4x + 3$ en $a = 1$

**b)** $f(x) = \dfrac{x - 1}{x + 1}$ en $a = -2$

**c)** $f(x) = \sqrt{x + 4}$ en $a = 0$
:::

:::solution
L'equation de la tangente en $a$ est $y = f'(a)(x - a) + f(a)$.

**a)** $f(x) = 2x^3 - 4x + 3$ en $a = 1$

- $f(1) = 2 - 4 + 3 = 1$

- On calcule $f'(1)$ par la definition :

$$\frac{f(x) - f(1)}{x - 1} = \frac{(2x^3 - 4x + 3) - 1}{x - 1} = \frac{2x^3 - 4x + 2}{x - 1}$$

On verifie que $x = 1$ est racine : $2 - 4 + 2 = 0$. On factorise :

$$2x^3 - 4x + 2 = (x - 1)(2x^2 + 2x - 2)$$

Donc $f'(1) = 2(1) + 2(1) - 2 = 2$.

- L'equation de la tangente est :

$$y = 2(x - 1) + 1 = 2x - 1$$

**b)** $f(x) = \dfrac{x - 1}{x + 1}$ en $a = -2$

- $f(-2) = \dfrac{-2 - 1}{-2 + 1} = \dfrac{-3}{-1} = 3$

- On calcule $f'(-2)$ :

$$\frac{f(x) - f(-2)}{x - (-2)} = \frac{\dfrac{x - 1}{x + 1} - 3}{x + 2}$$

On reduit au meme denominateur :

$$= \frac{\dfrac{x - 1 - 3(x + 1)}{x + 1}}{x + 2} = \frac{\dfrac{x - 1 - 3x - 3}{x + 1}}{x + 2} = \frac{\dfrac{-2x - 4}{x + 1}}{x + 2} = \frac{-2(x + 2)}{(x + 1)(x + 2)} = \frac{-2}{x + 1}$$

Donc $f'(-2) = \dfrac{-2}{-2 + 1} = \dfrac{-2}{-1} = 2$.

- L'equation de la tangente est :

$$y = 2(x - (-2)) + 3 = 2(x + 2) + 3 = 2x + 7$$

**c)** $f(x) = \sqrt{x + 4}$ en $a = 0$

- $f(0) = \sqrt{4} = 2$

- On calcule $f'(0)$ par la technique du conjugue :

$$\frac{f(x) - f(0)}{x - 0} = \frac{\sqrt{x + 4} - 2}{x} = \frac{(\sqrt{x + 4} - 2)(\sqrt{x + 4} + 2)}{x(\sqrt{x + 4} + 2)} = \frac{x + 4 - 4}{x(\sqrt{x + 4} + 2)} = \frac{1}{\sqrt{x + 4} + 2}$$

Donc $f'(0) = \dfrac{1}{\sqrt{4} + 2} = \dfrac{1}{4}$.

- L'equation de la tangente est :

$$y = \frac{1}{4}(x - 0) + 2 = \frac{1}{4}x + 2$$
:::

:::methode
Pour determiner l'equation de la tangente a $C_f$ au point d'abscisse $a$ :

1. Calculer $f(a)$
2. Calculer $f'(a)$ en utilisant la definition (taux de variation + limite)
3. Ecrire l'equation : $y = f'(a)(x - a) + f(a)$
:::
