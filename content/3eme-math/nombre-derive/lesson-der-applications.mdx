### Calculer une limite par reconnaissance d'un nombre derive

Certaines limites ont la forme d'un taux de variation. On peut les calculer en reconnaissant un nombre derive.

:::definition[Principe de reconnaissance]
Si l'on peut ecrire une limite sous la forme :

$$\lim_{x \to a} \frac{g(x) - g(a)}{x - a}$$

alors cette limite vaut $g'(a)$, a condition que $g$ soit derivable en $a$.
:::

**Methode** : identifier la fonction $g$ et le point $a$, puis calculer $g'(a)$ avec les formules de derivation.

:::example[Limite d'un quotient polynomial]
Calculer :

$$\lim_{x \to 1} \frac{x^3 - 1}{x - 1}$$

On reconnait le taux de variation de $g(x) = x^3$ entre $1$ et $x$.

En effet : $\frac{x^3 - 1}{x - 1} = \frac{g(x) - g(1)}{x - 1}$ avec $g(1) = 1$.

On a $g'(x) = 3x^2$, donc $g'(1) = 3$.

D'ou :

$$\lim_{x \to 1} \frac{x^3 - 1}{x - 1} = g'(1) = 3$$
:::

:::example[Limite avec une racine carree]
Calculer :

$$\lim_{x \to 0} \frac{\sqrt{1 + x} - 1}{x}$$

On pose $g(x) = \sqrt{1 + x}$. On a $g(0) = 1$, donc :

$$\frac{\sqrt{1 + x} - 1}{x} = \frac{g(x) - g(0)}{x - 0}$$

On calcule $g'(x) = \frac{1}{2\sqrt{1 + x}}$, d'ou $g'(0) = \frac{1}{2}$.

Par suite :

$$\lim_{x \to 0} \frac{\sqrt{1 + x} - 1}{x} = g'(0) = \frac{1}{2}$$
:::

:::remark
L'astuce consiste souvent a reecrire la limite pour faire apparaitre la forme $\frac{g(x) - g(a)}{x - a}$. Parfois, un changement de variable est necessaire.
:::

### Lecture graphique du nombre derive

A partir du graphe d'une fonction $f$, on peut lire directement $f(a)$ et $f'(a)$.

:::definition[Lecture graphique]
Soit $\mathcal{C}_f$ la courbe representative de $f$ dans un repere $(O, \vec{i}, \vec{j})$.

- **$f(a)$** se lit comme l'ordonnee du point de $\mathcal{C}_f$ d'abscisse $a$.
- **$f'(a)$** se lit comme le **coefficient directeur** (la pente) de la tangente a $\mathcal{C}_f$ au point d'abscisse $a$.
:::

```typst
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-0.5, 0), (6, 0), stroke: 0.8pt, mark: (end: ">", fill: black))
  line((0, -0.5), (0, 5.5), stroke: 0.8pt, mark: (end: ">", fill: black))
  content((5.8, -0.3), $x$)
  content((-0.3, 5.3), $y$)
  content((-0.3, -0.3), $O$)

  // Courbe schematique passant par (1,4), (2,3), (4,2), (5,4)
  bezier((0.3, 4.5), (2, 3), (1, 4.2), stroke: 2pt + blue)
  bezier((2, 3), (5.5, 4.5), (3, 1.5), (4.5, 2), stroke: 2pt + blue)

  // Tangente en A(2,3) : pente 2, y = 2(x-2)+3 = 2x-1
  line((0.8, 0.6), (4.5, 8 - 1), stroke: (dash: "dashed", paint: red, thickness: 1.5pt))

  // Point A
  circle((2, 3), radius: 0.08, fill: red, stroke: red)
  content((2.3, 3.4), text(fill: red, $A(2, 3)$))

  // Tangente en (4,2) : pente ~1, y = 1(x-4)+2 = x-2
  line((2.5, 0.5), (5.5, 3.5), stroke: (dash: "dashed", paint: green, thickness: 1.5pt))

  // Point B
  circle((4, 2), radius: 0.08, fill: green, stroke: green)
  content((4.3, 1.6), text(fill: green, $B$))

  // Tangente horizontale en sommet (~1, 4.2)
  line((0, 4.2), (2.2, 4.2), stroke: (dash: "dashed", paint: purple, thickness: 1.5pt))

  // Point C
  circle((1, 4.2), radius: 0.08, fill: purple, stroke: purple)
  content((1.3, 4.6), text(fill: purple, $C : f'=0$))

  // Label courbe
  content((5, 3.8), text(fill: blue, $C_f$))
})
```

:::example[Lecture sur un graphe]
Si la tangente a la courbe au point $A(2, 3)$ passe aussi par le point $B(4, 7)$, alors :

- $f(2) = 3$ (ordonnee du point de tangence)
- $f'(2) = \frac{7 - 3}{4 - 2} = \frac{4}{2} = 2$ (pente de la tangente)

L'equation de la tangente en $a = 2$ est :

$$y = f'(2)(x - 2) + f(2) = 2(x - 2) + 3 = 2x - 1$$
:::

:::attention
Si la tangente est **horizontale** en un point $a$, cela signifie que $f'(a) = 0$, et **non** que $f$ n'est pas derivable en $a$. Une tangente horizontale est bien une tangente.
:::
