1) a) Determiner l'ensemble de definition de $f$.

   b) Determiner $\lim_{x \to +\infty} f(x)$ et $\lim_{x \to -\infty} f(x)$.

2) a) $f$ est-elle derivable en $-1$ ? Justifier.

   b) Calculer les limites suivantes en reconnaissant un nombre derive :

$$\lim_{x \to -3} \dfrac{f(x)}{x + 3} \quad \text{et} \quad \lim_{x \to 0} \dfrac{f(x) - 2}{x}$$

3) Determiner les intervalles sur lesquels $f$ est derivable.

4) Dresser un tableau de variations complet de $f$.
:::

:::solution
**1) a)** La droite $x = 1$ est une asymptote verticale. La fonction $f$ n'est pas definie en $x = 1$.

$$D_f = \mathbb{R} \setminus \{1\}$$

**b)** D'apres la courbe :

$$\lim_{x \to +\infty} f(x) = 0 \quad \text{et} \quad \lim_{x \to -\infty} f(x) = 0$$

L'axe des abscisses ($y = 0$) est une asymptote horizontale en $+\infty$ et en $-\infty$.

**2) a)** Non, $f$ n'est pas derivable en $-1$. La courbe presente un **point anguleux** en $x = -1$ : on observe deux demi-tangentes de pentes differentes.

- La demi-tangente a gauche a pour pente $f'_g(-1) \approx -2$ (la courbe descend depuis la bosse)
- La demi-tangente a droite a pour pente $f'_d(-1) \approx 4$ (la courbe monte vers le maximum)

Comme $f'_g(-1) \neq f'_d(-1)$, $f$ n'est pas derivable en $-1$.

**b)** On remarque que $f(-3) = 0$ et $f(0) = 2$.

Pour la premiere limite :

$$\lim_{x \to -3} \dfrac{f(x)}{x + 3} = \lim_{x \to -3} \dfrac{f(x) - f(-3)}{x - (-3)} = f'(-3)$$

D'apres la tangente a la courbe au point $(-3, 0)$, la pente est $2$.

$$\lim_{x \to -3} \dfrac{f(x)}{x + 3} = f'(-3) = 2$$

Pour la seconde limite :

$$\lim_{x \to 0} \dfrac{f(x) - 2}{x} = \lim_{x \to 0} \dfrac{f(x) - f(0)}{x - 0} = f'(0)$$

Le point $(0, 2)$ est un **maximum local** de $f$ : la tangente y est horizontale.

$$\lim_{x \to 0} \dfrac{f(x) - 2}{x} = f'(0) = 0$$

**3)** Les points de non-derivabilite sont :
- $x = -1$ : point anguleux
- $x = 1$ : $f$ n'est pas definie

Par suite, $f$ est derivable sur chacun des intervalles :

$$]-\infty, -1[ \quad ; \quad ]-1, 1[ \quad ; \quad ]1, +\infty[$$

**4.** Tableau de variations de $f$ :

Variations :
- Sur $]-\infty, -3[$ : $f$ est decroissante (tend vers $0$ depuis le dessous)
- Sur $[-3, -2]$ : $f$ est croissante (de $0$ a $1$)
- Sur $[-2, -1]$ : $f$ est decroissante (de $1$ a $0$)
- Sur $[-1, 0]$ : $f$ est croissante (de $0$ a $2$)
- Sur $[0, 1[$ : $f$ est d'abord decroissante puis monte vers $+\infty$ en $1^-$

$$\lim_{x \to 1^-} f(x) = +\infty \quad ; \quad \lim_{x \to 1^+} f(x) = -\infty$$

- Sur $]1, +\infty[$ : $f$ est croissante (de $-\infty$ vers $0$)
:::

:::methode
Pour etudier les intervalles de derivabilite :

1. Identifier les points ou $f$ n'est pas definie (asymptotes verticales)
2. Identifier les points ou $f$ est continue mais non derivable (points anguleux, tangentes verticales)
3. Les intervalles de derivabilite sont les intervalles du domaine prives de ces points singuliers

Pour reconnaitre un nombre derive dans une limite :
- Si $f(a) = 0$ : $\lim_{x \to a} \dfrac{f(x)}{x - a} = f'(a)$
- Si $f(a) = k$ : $\lim_{x \to a} \dfrac{f(x) - k}{x - a} = f'(a)$
:::
