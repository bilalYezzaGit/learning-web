:::enonce
En utilisant la définition du nombre dérivé, calculer $f'(a)$ dans chacun des cas suivants :

**a)** $f(x) = \sqrt{x + 3}$ en $a = -2$

**b)** $f(x) = \dfrac{1}{x + 1}$ en $a = 0$
:::

:::solution
**a)** On calcule le taux de variation de $f(x) = \sqrt{x + 3}$ en $a = -2$ :

$$\frac{f(x) - f(-2)}{x - (-2)} = \frac{\sqrt{x + 3} - \sqrt{-2 + 3}}{x + 2} = \frac{\sqrt{x + 3} - 1}{x + 2}$$

On multiplie par l'expression conjuguée :

$$= \frac{(\sqrt{x + 3} - 1)(\sqrt{x + 3} + 1)}{(x + 2)(\sqrt{x + 3} + 1)} = \frac{(x + 3) - 1}{(x + 2)(\sqrt{x + 3} + 1)} = \frac{x + 2}{(x + 2)(\sqrt{x + 3} + 1)}$$

On simplifie par $(x + 2)$ :

$$= \frac{1}{\sqrt{x + 3} + 1}$$

En passant à la limite :

$$f'(-2) = \lim_{x \to -2} \frac{1}{\sqrt{x + 3} + 1} = \frac{1}{\sqrt{1} + 1} = \frac{1}{2}$$

**b)** On calcule le taux de variation de $f(x) = \dfrac{1}{x + 1}$ en $a = 0$ :

$$\frac{f(x) - f(0)}{x - 0} = \frac{\dfrac{1}{x + 1} - 1}{x}$$

On réduit au même dénominateur :

$$= \frac{\dfrac{1 - (x + 1)}{x + 1}}{x} = \frac{\dfrac{-x}{x + 1}}{x} = \frac{-x}{x(x + 1)} = \frac{-1}{x + 1}$$

En passant à la limite :

$$f'(0) = \lim_{x \to 0} \frac{-1}{x + 1} = \frac{-1}{1} = -1$$
:::

:::methode
- Pour les **racines carrées** : multiplier numérateur et dénominateur par l'expression conjuguée $(\sqrt{A} + \sqrt{B})$ pour éliminer la racine.
- Pour les **fractions** : réduire au même dénominateur pour obtenir une seule fraction, puis simplifier le facteur $(x - a)$.
:::
