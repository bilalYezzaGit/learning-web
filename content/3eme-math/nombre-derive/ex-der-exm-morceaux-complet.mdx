:::enonce
Soit $f$ la fonction definie sur $\mathbb{R}$ par :

$$f(x) = \begin{cases} x^2 + x - 6 & \text{si } x \leq 2 \\ \sqrt{x^2 - 4} + x - 2 & \text{si } x > 2 \end{cases}$$

1. Montrer que $f$ est continue en $2$.
2. Etudier la derivabilite de $f$ en $2$.
3. Ecrire les equations des demi-tangentes a la courbe $\mathcal{C}_f$ au point d'abscisse $2$.
:::

:::solution
**1. Continuite en $2$**

$$f(2) = 2^2 + 2 - 6 = 0$$

$$\lim_{x \to 2^-} f(x) = \lim_{x \to 2^-} (x^2 + x - 6) = 4 + 2 - 6 = 0$$

$$\lim_{x \to 2^+} f(x) = \lim_{x \to 2^+} \left(\sqrt{x^2 - 4} + x - 2\right) = \sqrt{0} + 2 - 2 = 0$$

On a $\lim_{x \to 2^-} f(x) = \lim_{x \to 2^+} f(x) = f(2) = 0$, donc $f$ est **continue en $2$**.

---

**2. Derivabilite en $2$**

**Derivee a gauche** : pour $x < 2$, $f(x) = x^2 + x - 6$, donc :

$$f'_g(2) = \lim_{x \to 2^-} \frac{f(x) - f(2)}{x - 2} = \lim_{x \to 2^-} \frac{x^2 + x - 6}{x - 2}$$

On factorise : $x^2 + x - 6 = (x - 2)(x + 3)$, donc :

$$f'_g(2) = \lim_{x \to 2^-} \frac{(x - 2)(x + 3)}{x - 2} = \lim_{x \to 2^-} (x + 3) = 5$$

**Derivee a droite** : pour $x > 2$, $f(x) = \sqrt{x^2 - 4} + x - 2$, donc :

$$f'_d(2) = \lim_{x \to 2^+} \frac{f(x) - f(2)}{x - 2} = \lim_{x \to 2^+} \frac{\sqrt{x^2 - 4} + x - 2 - 0}{x - 2} = \lim_{x \to 2^+} \left(\frac{\sqrt{x^2 - 4}}{x - 2} + 1\right)$$

On factorise $x^2 - 4 = (x - 2)(x + 2)$, donc :

$$\frac{\sqrt{x^2 - 4}}{x - 2} = \frac{\sqrt{(x - 2)(x + 2)}}{x - 2} = \frac{\sqrt{x + 2}}{\sqrt{x - 2}}$$

Quand $x \to 2^+$ : $\sqrt{x + 2} \to 2$ et $\sqrt{x - 2} \to 0^+$, donc $\dfrac{\sqrt{x + 2}}{\sqrt{x - 2}} \to +\infty$.

Ainsi $f'_d(2) = +\infty$.

Comme $f'_d(2) = +\infty$, la fonction $f$ **n'est pas derivable en $2$**.

---

**3. Equations des demi-tangentes**

**Demi-tangente a gauche** (pente $f'_g(2) = 5$) :

$$y = 5(x - 2) + 0 = 5x - 10$$

**Demi-tangente a droite** ($f'_d(2) = +\infty$) : c'est une demi-droite verticale d'equation :

$$x = 2$$

dirigee vers le haut (car le taux de variation tend vers $+\infty$).

Le point $A(2, 0)$ est un **point de rebroussement de premiere espece** de la courbe $\mathcal{C}_f$.
:::

:::erreurs
- Oublier de verifier la continuite **avant** d'etudier la derivabilite. Si $f$ n'est pas continue en $2$, elle n'est automatiquement pas derivable.
- Ne pas factoriser $x^2 - 4 = (x - 2)(x + 2)$ sous la racine, ce qui empeche de simplifier le taux de variation.
- Conclure que $f$ est derivable parce que $f'_g(2)$ est fini, sans verifier $f'_d(2)$.
:::
