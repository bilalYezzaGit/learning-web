### Le théorème T3 : dérivable implique continu

:::theorem[Théorème T3]
Si $f$ est dérivable en $a$, alors $f$ est continue en $a$.
:::

**Démonstration complète.**

Pour montrer que $f$ est continue en $a$, il faut montrer que $\lim_{x \to a} f(x) = f(a)$, c'est-a-dire que $\lim_{x \to a} \bigl[f(x) - f(a)\bigr] = 0$.

Pour $x \neq a$, on écrit :

$$f(x) - f(a) = \frac{f(x) - f(a)}{x - a} \times (x - a)$$

Comme $f$ est dérivable en $a$, on a :

$$\lim_{x \to a} \frac{f(x) - f(a)}{x - a} = f'(a)$$

De plus :

$$\lim_{x \to a} (x - a) = 0$$

Par produit des limites :

$$\lim_{x \to a} \bigl[f(x) - f(a)\bigr] = f'(a) \times 0 = 0$$

D'où $\lim_{x \to a} f(x) = f(a)$, ce qui signifie que $f$ est continue en $a$.

### La réciproque est fausse

:::attention[Continu n'implique pas dérivable]
La réciproque de T3 est **fausse** : une fonction peut être continue en un point sans y être dérivable. Il existe deux types de situations.
:::

:::example[Contre-exemple 1 : la valeur absolue (point anguleux)]
Soit $f(x) = |x|$. Cette fonction est continue en $0$ car $\lim_{x \to 0} |x| = 0 = f(0)$.

Étudions la dérivabilité en $0$ :

- **À droite** : pour $h > 0$, $\dfrac{f(0 + h) - f(0)}{h} = \dfrac{|h|}{h} = \dfrac{h}{h} = 1$, donc $f'_d(0) = 1$.

- **À gauche** : pour $h < 0$, $\dfrac{f(0 + h) - f(0)}{h} = \dfrac{|h|}{h} = \dfrac{-h}{h} = -1$, donc $f'_g(0) = -1$.

Comme $f'_d(0) = 1 \neq -1 = f'_g(0)$, la fonction $f$ n'est **pas dérivable** en $0$.

La courbe présente un **point anguleux** en $0$ : deux demi-tangentes de pentes différentes.

```typst {frame}
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-3, 0), (3, 0), stroke: 0.5pt + black, mark: (end: "stealth", size: 0.2))
  line((0, -0.5), (0, 3), stroke: 0.5pt + black, mark: (end: "stealth", size: 0.2))
  content((3, -0.3), $x$)
  content((-0.3, 3), $y$)

  // f(x) = |x| : two segments
  line((-2, 2), (0, 0), stroke: 2pt + blue)
  line((0, 0), (2, 2), stroke: 2pt + blue)

  // Half-tangent left: y = -x (dashed red)
  line((-2.5, 2.5), (0.3, -0.3), stroke: (dash: "dashed", paint: red, thickness: 1.5pt))

  // Half-tangent right: y = x (dashed red)
  line((-0.3, -0.3), (2.5, 2.5), stroke: (dash: "dashed", paint: red, thickness: 1.5pt))

  // Origin dot
  circle((0, 0), radius: 0.08, fill: blue, stroke: none)

  // Label
  content((0.8, -0.5), text(size: 9pt, fill: red)[point anguleux])
})
```

:::

:::example[Contre-exemple 2 : la racine cubique (tangente verticale)]
Soit $g(x) = x^{1/3} = \sqrt[3]{x}$. Cette fonction est continue en $0$ car $\lim_{x \to 0} x^{1/3} = 0 = g(0)$.

Étudions le taux de variation en $0$ :

$$\frac{g(h) - g(0)}{h} = \frac{h^{1/3}}{h} = \frac{1}{h^{2/3}}$$

Or $\lim_{h \to 0} \dfrac{1}{h^{2/3}} = +\infty$, donc $g$ n'est **pas dérivable** en $0$.

La courbe admet une **tangente verticale** au point $O(0, 0)$.

```typst {frame}
#import "@preview/cetz:0.3.4": canvas, draw

#canvas({
  import draw: *

  // Axes
  line((-3, 0), (3, 0), stroke: 0.5pt + black, mark: (end: "stealth", size: 0.2))
  line((0, -2.5), (0, 2.5), stroke: 0.5pt + black, mark: (end: "stealth", size: 0.2))
  content((3, -0.3), $x$)
  content((-0.3, 2.5), $y$)

  // Cube root curve approximation using bezier
  // Right side: from (0,0) to (2, 2^{1/3} ≈ 1.26), steep near origin
  bezier((0, 0), (2, 1.26), (0.1, 0.8), stroke: 2pt + blue)
  bezier((0, 0), (2.5, 1.36), (0.3, 1.1), stroke: 2pt + blue)

  // Left side: from (0,0) to (-2, -1.26), steep near origin
  bezier((0, 0), (-2, -1.26), (-0.1, -0.8), stroke: 2pt + blue)
  bezier((0, 0), (-2.5, -1.36), (-0.3, -1.1), stroke: 2pt + blue)

  // Vertical tangent at origin (dashed red)
  line((0, -2), (0, 2), stroke: (dash: "dashed", paint: red, thickness: 1.5pt))

  // Origin dot
  circle((0, 0), radius: 0.08, fill: blue, stroke: none)

  // Label
  content((1.5, -0.5), text(size: 9pt, fill: red)[tangente verticale])
})
```

:::

:::remark[Curiosité : la fonction de Weierstrass]
Il existe des fonctions **continues partout** mais **dérivables nulle part**. La plus célèbre est la fonction de Weierstrass (1872), définie comme une série de fonctions trigonométriques. Elle montre que la continuité et la dérivabilité sont des propriétés fondamentalement différentes : la dérivabilité est beaucoup plus exigeante.
:::
