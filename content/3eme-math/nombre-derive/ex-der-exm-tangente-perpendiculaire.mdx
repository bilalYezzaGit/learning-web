:::enonce
Soit $f$ la fonction définie par $f(x) = 2x^2 + x + 1$ et soit $D$ la droite d'équation $x - 7y + 7 = 0$.

**1)** Déterminer le coefficient directeur de $D$.

**2)** Déterminer la valeur de $a$ pour laquelle la tangente à $C_f$ au point d'abscisse $a$ est perpendiculaire a $D$.

**3)** Écrire l'équation de cette tangente.
:::

:::solution
**1)** On réécrit l'équation de $D$ sous forme $y = mx + p$ :

$$x - 7y + 7 = 0 \implies 7y = x + 7 \implies y = \frac{1}{7}x + 1$$

Le coefficient directeur de $D$ est $m_D = \dfrac{1}{7}$.

**2)** Deux droites sont perpendiculaires si et seulement si le produit de leurs pentes vaut $-1$ :

$$f'(a) \times m_D = -1$$

$$f'(a) \times \frac{1}{7} = -1$$

$$f'(a) = -7$$

Calculons $f'(a)$. Le taux de variation de $f$ en $a$ est :

$$\frac{f(x) - f(a)}{x - a} = \frac{(2x^2 + x + 1) - (2a^2 + a + 1)}{x - a} = \frac{2(x^2 - a^2) + (x - a)}{x - a}$$

$$= \frac{2(x - a)(x + a) + (x - a)}{x - a} = 2(x + a) + 1$$

En passant à la limite : $f'(a) = 2(a + a) + 1 = 4a + 1$.

On résout $f'(a) = -7$ :

$$4a + 1 = -7$$

$$4a = -8$$

$$\boxed{a = -2}$$

**3)** On calcule $f(-2)$ :

$$f(-2) = 2(4) + (-2) + 1 = 8 - 2 + 1 = 7$$

La pente de la tangente est $f'(-2) = 4(-2) + 1 = -7$.

L'équation de la tangente est :

$$y = -7(x - (-2)) + 7 = -7(x + 2) + 7 = -7x - 14 + 7$$

$$\boxed{y = -7x - 7}$$
:::

:::methode
Pour trouver la tangente perpendiculaire à une droite $D$ de pente $m_D$ :

1. Utiliser la condition de perpendicularité : $f'(a) \times m_D = -1$, soit $f'(a) = -\dfrac{1}{m_D}$
2. Résoudre cette équation pour trouver $a$
3. Écrire l'équation de la tangente
:::
