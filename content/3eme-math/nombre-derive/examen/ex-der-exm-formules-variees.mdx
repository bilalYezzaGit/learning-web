:::enonce
Calculer $f'(a)$ dans chacun des cas suivants :

**a)** $f(x) = x^4 - 2x^3 + 5$ en $a = -1$

**b)** $f(x) = \dfrac{3}{2x + 1}$ en $a = 1$

**c)** $f(x) = \sqrt{3x + 1}$ en $a = 5$

**d)** $f(x) = x^2\sqrt{x + 1}$ en $a = 0$
:::

:::solution
**a)** $f(x) = x^4 - 2x^3 + 5$

On applique les formules $(x^n)' = nx^{n-1}$ :

$$f'(x) = 4x^3 - 6x^2$$

$$f'(-1) = 4(-1)^3 - 6(-1)^2 = -4 - 6 = -10$$

---

**b)** $f(x) = \dfrac{3}{2x + 1}$

On écrit $f(x) = 3 \cdot (2x + 1)^{-1}$. Avec la formule $\left(\dfrac{1}{u}\right)' = \dfrac{-u'}{u^2}$ :

$$f'(x) = 3 \times \frac{-2}{(2x + 1)^2} = \frac{-6}{(2x + 1)^2}$$

$$f'(1) = \frac{-6}{(2 \times 1 + 1)^2} = \frac{-6}{9} = -\frac{2}{3}$$

---

**c)** $f(x) = \sqrt{3x + 1}$

Avec la formule $(\sqrt{u})' = \dfrac{u'}{2\sqrt{u}}$ :

$$f'(x) = \frac{3}{2\sqrt{3x + 1}}$$

$$f'(5) = \frac{3}{2\sqrt{3 \times 5 + 1}} = \frac{3}{2\sqrt{16}} = \frac{3}{2 \times 4} = \frac{3}{8}$$

---

**d)** $f(x) = x^2\sqrt{x + 1}$

On applique la formule du produit $(uv)' = u'v + uv'$ avec $u(x) = x^2$ et $v(x) = \sqrt{x + 1}$ :

$$u'(x) = 2x \quad ; \quad v'(x) = \frac{1}{2\sqrt{x + 1}}$$

$$f'(x) = 2x\sqrt{x + 1} + x^2 \times \frac{1}{2\sqrt{x + 1}}$$

$$f'(0) = 2 \times 0 \times \sqrt{1} + 0^2 \times \frac{1}{2\sqrt{1}} = 0 + 0 = 0$$
:::

:::methode
Pour calculer $f'(a)$ :

1. Identifier le type de fonction (polynôme, rationnelle, racine, produit)
2. Appliquer la formule de dérivation correspondante pour obtenir $f'(x)$
3. Remplacer $x$ par $a$ dans l'expression de $f'(x)$
:::
