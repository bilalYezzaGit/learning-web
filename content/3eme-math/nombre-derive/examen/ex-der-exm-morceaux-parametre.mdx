:::enonce
Soit $m$ un paramètre réel et $f$ la fonction définie sur $\mathbb{R}$ par :

$$f(x) = \begin{cases} 3x^2 - 5x + m & \text{si } x \leq 2 \\ \sqrt{x^2 + 2x - 8} + mx & \text{si } x > 2 \end{cases}$$

1. Déterminer la valeur de $m$ pour que $f$ soit continue en $2$.
2. Pour cette valeur de $m$, étudier la dérivabilité de $f$ en $2$.
3. Écrire les équations des demi-tangentes à la courbe $\mathcal{C}_f$ au point d'abscisse $2$.
:::

:::solution
**1. Continuité en $2$**

**À gauche** :

$$\lim_{x \to 2^-} f(x) = 3(4) - 5(2) + m = 12 - 10 + m = 2 + m$$

$$f(2) = 2 + m$$

**À droite** : on a $2^2 + 2(2) - 8 = 0$, donc :

$$\lim_{x \to 2^+} f(x) = \sqrt{0} + 2m = 2m$$

Pour que $f$ soit continue en $2$, il faut :

$$2 + m = 2m$$

$$m = 2$$

---

**2. Dérivabilité en $2$ avec $m = 2$**

Avec $m = 2$, $f$ s'écrit :

$$f(x) = \begin{cases} 3x^2 - 5x + 2 & \text{si } x \leq 2 \\ \sqrt{x^2 + 2x - 8} + 2x & \text{si } x > 2 \end{cases}$$

et $f(2) = 2 + 2 = 4$.

**Dérivée à gauche** : pour $x < 2$, $f(x) = 3x^2 - 5x + 2$, donc :

$$f'_g(2) = \lim_{x \to 2^-} \frac{f(x) - f(2)}{x - 2} = \lim_{x \to 2^-} \frac{3x^2 - 5x + 2 - 4}{x - 2} = \lim_{x \to 2^-} \frac{3x^2 - 5x - 2}{x - 2}$$

On factorise : $3x^2 - 5x - 2 = (x - 2)(3x + 1)$, donc :

$$f'_g(2) = \lim_{x \to 2^-} (3x + 1) = 7$$

**Dérivée à droite** : pour $x > 2$, $f(x) = \sqrt{x^2 + 2x - 8} + 2x$, donc :

$$f'_d(2) = \lim_{x \to 2^+} \frac{f(x) - f(2)}{x - 2} = \lim_{x \to 2^+} \frac{\sqrt{x^2 + 2x - 8} + 2x - 4}{x - 2} = \lim_{x \to 2^+} \left(\frac{\sqrt{x^2 + 2x - 8}}{x - 2} + 2\right)$$

On factorise $x^2 + 2x - 8 = (x - 2)(x + 4)$, donc :

$$\frac{\sqrt{x^2 + 2x - 8}}{x - 2} = \frac{\sqrt{(x - 2)(x + 4)}}{x - 2} = \frac{\sqrt{x + 4}}{\sqrt{x - 2}}$$

Quand $x \to 2^+$ : $\sqrt{x + 4} \to \sqrt{6}$ et $\sqrt{x - 2} \to 0^+$, donc $\dfrac{\sqrt{x + 4}}{\sqrt{x - 2}} \to +\infty$.

Ainsi $f'_d(2) = +\infty$.

Comme $f'_d(2) = +\infty$, la fonction $f$ **n'est pas dérivable en $2$**.

---

**3. Équations des demi-tangentes**

**Demi-tangente à gauche** (pente $f'_g(2) = 7$) :

$$y = 7(x - 2) + 4 = 7x - 10$$

**Demi-tangente à droite** ($f'_d(2) = +\infty$) : c'est une demi-droite verticale d'équation :

$$x = 2$$

dirigée vers le haut.
:::

:::methode
Pour une fonction par morceaux avec un paramètre :

1. Écrire les limites à gauche et à droite du point de raccordement
2. Égaliser pour trouver $m$ (continuité)
3. Avec la valeur de $m$, calculer les dérivées latérales par le taux de variation
4. Sous une racine, factoriser l'expression pour identifier le comportement du taux
:::
