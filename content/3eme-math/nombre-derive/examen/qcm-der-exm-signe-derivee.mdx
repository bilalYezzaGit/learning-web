:::question
Soit $f(x) = \dfrac{1}{x + 1}$ définie sur $]-1 \,;\, +\infty[$. Que vaut $f'(a)$ pour $a > -1$ ?
:::

:::option
$\dfrac{1}{(a + 1)^2}$
:::

:::option{correct}
$\dfrac{-1}{(a + 1)^2}$
:::

:::option
$\dfrac{-1}{a + 1}$
:::

:::option
$\dfrac{1}{a + 1}$
:::

:::explanation
On applique la formule $\left(\dfrac{1}{u}\right)' = \dfrac{-u'}{u^2}$ avec $u(x) = x + 1$ et $u'(x) = 1$ :

$$f'(x) = \frac{-1}{(x + 1)^2}$$

Le **signe moins** est essentiel : la dérivée de $\dfrac{1}{u}$ est toujours **négative** (quand $u' > 0$). Oublier ce signe est une erreur très fréquente.

De plus, le dénominateur est au **carré**, pas à la puissance $1$.
:::
