:::question
La derivee de $g(x) = \sqrt{4x-3}$ est :
:::

:::option
$\frac{1}{2\sqrt{4x-3}}$
:::
:::option{correct}
$\frac{2}{\sqrt{4x-3}}$
:::
:::option
$\frac{-4}{2\sqrt{4x-3}}$
:::
:::option
$2\sqrt{4x-3}$
:::

:::explanation
On applique la formule $(\sqrt{u})' = \frac{u'}{2\sqrt{u}}$.

Ici $u = 4x-3$ donc $u' = 4$.

$$g'(x) = \frac{4}{2\sqrt{4x-3}} = \frac{2}{\sqrt{4x-3}}$$

**L'erreur classique** est d'oublier le facteur $u' = 4$ et d'ecrire $\frac{1}{2\sqrt{4x-3}}$. Il faut toujours penser a deriver la fonction "interieure" dans une composee.
:::
