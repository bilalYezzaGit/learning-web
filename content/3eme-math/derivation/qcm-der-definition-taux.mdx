:::question
Le nombre derive $f'(a)$ est egal a :
:::

:::option
$f(a+h) - f(a)$
:::
:::option
$\frac{f(a+h) - f(a)}{h}$
:::
:::option{correct}
$\lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$
:::
:::option
$\lim_{x \to a} f(x)$
:::

:::explanation
Le nombre derive est la **limite** du taux d'accroissement quand $h$ tend vers 0, pas le taux lui-meme. La formule $\frac{f(a+h) - f(a)}{h}$ represente le taux d'accroissement, mais ce n'est qu'une fois qu'on prend la limite que l'on obtient le nombre derive $f'(a)$.
:::
