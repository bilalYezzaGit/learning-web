:::enonce
Soit $m$ un reel et $f_m$ la fonction definie sur $\mathbb{R}$ par :

$$f_m(x) = (2m - 3)x^3 + 3x^2 - 3mx + m$$

1. Pour quelles valeurs de $m$, $f_m$ possede-t-elle un seul extremum ?
2. Pour quelles valeurs de $m$, $f_m$ est-elle decroissante sur tout $\mathbb{R}$ ?
3. Pour $m = 1$ : on pose $g(x) = -x^3 + 3|x^2 - x| + 1$.

   a) Montrer que $g(x) = f_1(x)$ pour $x \leq 0$ ou $x \geq 1$.

   b) Exprimer $g(x)$ pour $0 < x < 1$.

   c) Dresser le tableau de variation complet de $g$. Preciser les points anguleux et les vecteurs directeurs des demi-tangentes.
:::

:::solution
**1.** Si $2m - 3 = 0$, c'est-a-dire $m = \dfrac{3}{2}$ : $f_m(x) = 3x^2 - \dfrac{9}{2}x + \dfrac{3}{2}$, polynome de degre $2$. $f'_m(x) = 6x - \dfrac{9}{2} = 0 \Leftrightarrow x = \dfrac{3}{4}$. Un seul extremum.

Si $m \neq \dfrac{3}{2}$ : $f'_m(x) = 3(2m-3)x^2 + 6x - 3m$.

$\Delta' = 9 + 9m(2m-3) = 9(1 + 2m^2 - 3m) = 9(2m^2 - 3m + 1) = 9(2m-1)(m-1)$

- $\Delta' > 0$ ($m < \dfrac{1}{2}$ ou $m > 1$, $m \neq \dfrac{3}{2}$) : $f'_m$ a deux racines, donc $f_m$ a **deux extrema**.
- $\Delta' = 0$ ($m = \dfrac{1}{2}$ ou $m = 1$) : $f'_m$ a une racine double, $f'_m$ ne change pas de signe (ou s'annule ponctuellement). **Pas d'extremum** (car $f'_m$ ne change pas de signe).
- $\Delta' < 0$ ($\dfrac{1}{2} < m < 1$, $m \neq \dfrac{3}{2}$) : $f'_m$ ne s'annule pas, **pas d'extremum**.

$f_m$ possede **un seul extremum** pour $m = \dfrac{3}{2}$.

**2.** $f_m$ est decroissante sur tout $\mathbb{R}$ si $f'_m(x) \leq 0$ pour tout $x$.

Si $2m - 3 > 0$ (i.e. $m > \dfrac{3}{2}$) : $\lim_{x \to +\infty} f'_m(x) = +\infty$, donc $f'_m$ n'est pas $\leq 0$ partout.

Si $2m - 3 = 0$ ($m = \dfrac{3}{2}$) : $f'_m(x) = 6x - \dfrac{9}{2}$, qui prend des valeurs positives. Non.

Si $2m - 3 < 0$ (i.e. $m < \dfrac{3}{2}$) : il faut $\Delta' \leq 0$, soit $\dfrac{1}{2} \leq m \leq 1$.

Verification : pour $\dfrac{1}{2} \leq m \leq 1$ et $m < \dfrac{3}{2}$, le coefficient dominant de $f'_m$ est $3(2m-3) < 0$ et $f'_m$ n'a pas de racine reelle (ou une racine double) donc $f'_m(x) \leq 0$ pour tout $x$.

$f_m$ est decroissante sur $\mathbb{R}$ pour $\dfrac{1}{2} \leq m \leq 1$.

**3.** Pour $m = 1$ : $f_1(x) = -x^3 + 3x^2 - 3x + 1 = (1-x)^3$.

**a)** $g(x) = -x^3 + 3|x^2 - x| + 1$.

Pour $x \leq 0$ : $x^2 - x = x(x-1) \geq 0$ (produit de deux facteurs negatifs ou nul), donc $|x^2-x| = x^2 - x$.

$g(x) = -x^3 + 3(x^2-x) + 1 = -x^3 + 3x^2 - 3x + 1 = f_1(x)$.

Pour $x \geq 1$ : $x^2 - x = x(x-1) \geq 0$, donc $|x^2-x| = x^2 - x$.

$g(x) = f_1(x) = (1-x)^3$.

**b)** Pour $0 < x < 1$ : $x^2 - x = x(x-1) < 0$, donc $|x^2 - x| = -(x^2-x) = x - x^2$.

$$g(x) = -x^3 + 3(x - x^2) + 1 = -x^3 - 3x^2 + 3x + 1$$

**c)** Sur $]-\infty, 0[$ : $g(x) = (1-x)^3$, $g'(x) = -3(1-x)^2 \leq 0$. $g$ est decroissante.

Sur $]0, 1[$ : $g(x) = -x^3 - 3x^2 + 3x + 1$, $g'(x) = -3x^2 - 6x + 3 = -3(x^2 + 2x - 1)$.

$g'(x) = 0 \Leftrightarrow x^2 + 2x - 1 = 0 \Leftrightarrow x = -1 + \sqrt{2} \approx 0{,}41$ (dans $]0, 1[$).

Sur $]1, +\infty[$ : $g(x) = (1-x)^3$, $g'(x) = -3(1-x)^2 \leq 0$. $g$ est decroissante.

**Points anguleux** :

En $x = 0$ : $g'_g(0) = -3(1-0)^2 = -3$ et $g'_d(0) = -3(0+0-1) = 3$. Point anguleux, demi-tangentes de pentes $-3$ et $3$.

En $x = 1$ : $g'_g(1) = -3(1+2-1) = -6$ et $g'_d(1) = -3(1-1)^2 = 0$. Point anguleux, demi-tangentes de pentes $-6$ et $0$.

$= -(3\sqrt{2}-5) - 3(3-2\sqrt{2}) + 3\sqrt{2} - 3 + 1 = 5-3\sqrt{2} - 9 + 6\sqrt{2} + 3\sqrt{2} - 2 = -6 + 6\sqrt{2}$
