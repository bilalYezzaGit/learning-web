:::question
On tire **simultanément** $4$ boules d'une urne contenant $12$ boules distinctes. Le nombre de tirages possibles est :
:::

:::option
$A_{12}^4 = 11\,880$
:::

:::option
$12^4 = 20\,736$
:::

:::option{correct}
$C_{12}^4 = 495$
:::

:::option
$4^{12}$
:::

:::explanation
Un tirage **simultané** ne tient pas compte de l'ordre et se fait sans remise. On utilise donc les **combinaisons** : $C_{12}^4 = \frac{12!}{4! \times 8!} = 495$.

- $A_{12}^4$ correspondrait à un tirage successif **sans remise** (ordre compte).
- $12^4$ correspondrait à un tirage successif **avec remise** (ordre compte, répétition possible).
:::
