:::enonce
Une grenouille se trouve au bas d'un escalier de $13$ marches. À chaque saut, elle monte d'**une** ou de **deux** marches.

1. Lister tous les couples $(N, P)$ où $N$ est le nombre de sauts de $2$ marches et $P$ le nombre de sauts de $1$ marche, vérifiant $2N + P = 13$.
2. Pour chaque couple $(N, P)$, calculer le nombre de façons de monter l'escalier.
3. En déduire le nombre total de façons de monter les $13$ marches.
4. Montrer que ce résultat est lié à la suite de **Fibonacci**.
:::

:::solution
**1.** On cherche les couples $(N, P) \in \mathbb{N}^2$ tels que $2N + P = 13$.

```typst
#table(
  columns: 3,
  align: center,
  stroke: 0.5pt,
  [*$N$*], [*$P = 13 - 2N$*], [*Nombre de sauts $N + P$*],
  [$0$], [$13$], [$13$],
  [$1$], [$11$], [$12$],
  [$2$], [$9$], [$11$],
  [$3$], [$7$], [$10$],
  [$4$], [$5$], [$9$],
  [$5$], [$3$], [$8$],
  [$6$], [$1$], [$7$],
)
```

**2.** Pour un couple $(N, P)$, le nombre total de sauts est $N + P$. La grenouille effectue $N + P$ sauts dont $N$ sont des sauts de $2$ et $P$ des sauts de $1$. On choisit les positions des sauts de $2$ parmi $N + P$ sauts :

$$C_{N+P}^{N}$$

```typst
#table(
  columns: 3,
  align: center,
  stroke: 0.5pt,
  [*$(N, P)$*], [*$N + P$*], [*$C_{N+P}^N$*],
  [$(0, 13)$], [$13$], [$C_{13}^0 = 1$],
  [$(1, 11)$], [$12$], [$C_{12}^1 = 12$],
  [$(2, 9)$], [$11$], [$C_{11}^2 = 55$],
  [$(3, 7)$], [$10$], [$C_{10}^3 = 120$],
  [$(4, 5)$], [$9$], [$C_9^4 = 126$],
  [$(5, 3)$], [$8$], [$C_8^5 = 56$],
  [$(6, 1)$], [$7$], [$C_7^6 = 7$],
)
```

**3.** Le nombre total de façons :

$$1 + 12 + 55 + 120 + 126 + 56 + 7 = 377$$

**4.** Notons $u_n$ le nombre de façons de monter un escalier de $n$ marches. Pour atteindre la marche $n$, la grenouille vient :
- de la marche $n - 1$ (dernier saut de $1$) : $u_{n-1}$ façons,
- de la marche $n - 2$ (dernier saut de $2$) : $u_{n-2}$ façons.

Donc :

$$u_n = u_{n-1} + u_{n-2}$$

Avec $u_1 = 1$ (un seul saut de $1$) et $u_2 = 2$ (deux sauts de $1$, ou un saut de $2$).

C'est la **suite de Fibonacci** (décalée). On vérifie :

$u_1 = 1$, $u_2 = 2$, $u_3 = 3$, $u_4 = 5$, $u_5 = 8$, $u_6 = 13$, $u_7 = 21$, $u_8 = 34$, $u_9 = 55$, $u_{10} = 89$, $u_{11} = 144$, $u_{12} = 233$, $u_{13} = 377$.
:::
