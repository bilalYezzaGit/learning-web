:::question
Le coefficient de $x^3$ dans le développement de $(1+x)^5$ est :
:::

:::option
$5^3 = 125$
:::

:::option
$3! = 6$
:::

:::option
$C_5^2 = 10$
:::

:::option{correct}
$C_5^3 = 10$
:::

:::explanation
D'après le binôme de Newton, $(1+x)^5 = \sum_{k=0}^{5} C_5^k \, x^k$.

Le coefficient de $x^3$ correspond au terme $k = 3$, soit $C_5^3$.

$C_5^3 = \frac{5!}{3! \times 2!} = \frac{120}{6 \times 2} = 10$.

L'erreur $125 = 5^3$ confond avec la formule $n^p$. L'erreur $6 = 3!$ confond avec les permutations. L'option $C_5^2$ donne aussi $10$ par symétrie, mais le raisonnement correct identifie $k = 3$ (l'exposant de $x$), donc c'est $C_5^3$.
:::
